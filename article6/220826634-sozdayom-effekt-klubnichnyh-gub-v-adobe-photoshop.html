Создаём Эффект Клубничных Губ в Adobe Photoshop...

Интро

В этом уроке я покажу вам, как создать вкусный эффект клубничных губ. Вы узнаете, как объединить два исходных изображения с помощью базовых техник ретуширования, совмещения, подбора цвета и коррекции деталей....

В этом уроке я покажу вам, как создать вкусный эффект клубничных губ. Вы узнаете, как объединить два исходных изображения с помощью базовых техник ретуширования, совмещения, подбора цвета и коррекции деталей.

Каким бы ни был ваш уровень мастерства относительно создания фотоманипуляций и ретуширования, мы рекомендуем вам использовать следующие важные исходные материалы в ваших будущих проектах.

Исходные материалы

Следующие исходные материалыбыли использованы для создания данного урока:

  • Модель
  • Клубника

1. Кадрируем Изображение и Меняем Цвет Губ

Шаг 1

Откройте исходное изображение с моделью. Мы будем работать только с губами, поэтому, с помощью инструмента Рамка (Crop Tool (C), выделите только область вокруг губ. Запомните, что задний фон не должен входить в область выделения.

Нажмите клавишу (Enter), чтобы применить изменения.

Шаг 2

После кадрирования, мы немного изменим цвет губ. Основная причина- это совместить цвет губ с цветом клубники. Выберите инструмент Быстрое выделение (Quick Selection Tool (W), а затем выделите область губ. Не переживайте, если результат выглядит не очень хорошо, мы исправим это буквально сейчас.

Чтобы создать более точное выделение, нажмите клавишу "Q", чтобы активировать режим Быстрой маски (Quick Mask).

Убедитесь, чтобы цвет переднего план был установлен на чёрный цвет, а затем нажмите клавишу "B", чтобы активировать инструмент Кисть (Brush Tool). С помощью жёсткой круглой кисти, аккуратно прокрасьте лишние участки, расположенные за внешним контуром губ, а также зубы внутри рта, таким образом, на них будет воздействовать коррекция, которую мы проведём далее. Вы можете переключить цвет переднего плана на белый цвет, чтобы заполнить пустоты на губах или что-то подкорректировать.

Шаг 3

Ещё раз нажмите клавишу "Q", чтобы выйти из режима Быстрой маски (Quick Mask) и увидеть результат выделения.

Шаг 4

Находясь на слое с губами, идём Слой- Новый корректирующий слой - Цветовой тон / Насыщенность (Layer > New Adjustment Layer > Hue/Saturation). Назовите этот первый корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation). Таким образом, данная коррекция будет воздействовать только на губы. Поменяйте значение Цветового тона (Hue) на +6.

2. Ретушируем Рот

Шаг 1

Вокруг рта есть участки, на которых видны мелкие волоски. Вы можете оставить всё, как есть, но я хочу отретушировать их. Нажмите клавиши (Ctrl+Shift+N), чтобы создать новый слой.

Выберите инструмент Штамп (Clone Tool (S), в панели управления данного инструмента, установите режим Образец (Sample) на Все слои (All Layers). Размер кисти инструмента Штамп (Clone Tool) будет зависеть от размеров вашего рабочего документа. Моя размеры документа составляют 1200 x 800 px, и я использовала размер кисти 50-55 px. Удерживая клавишу (Alt), щёлкните по области ниже волосков. После этого, проведите кистью поверх участков, которые вы хотите отредактировать. Повторите этот шаг, пока не исправите все участки. Я не хочу, чтобы результат выглядел слишком неестественно, поэтому, уменьшите Непрозрачность (Opacity) кисти до 60-65%.

Шаг 2

Результат должен быть немного темнее, поскольку это область тени. С помощью инструмента Лассо (Lasso Tool (L), выделите область ниже губ, а затем идём Слой- Новый корректирующий слой- Кривые (Layer > New Adjustment Layer > Curves). В появившемся окне, не забудьте поставить галочку в окошке Использовать предыдущий слой для создания обтравочной маски (Use Previous Layer To Create Clipping Mask), а затем уменьшите яркость, пока область коррекции не совпадёт с областью тени.

Шаг 3

Правый уголок нижней губа выглядит не очень хорошо. Чтобы исправить его, создайте новый слой и с помощью инструмента Штамп (Clone Tool), исправьте указанную область, сделав правый уголок губы более пропорциональным с левым.

3. Добавляем Текстуру Клубники

Шаг 1

Откройте исходное изображение с клубникой. С помощью инструмента Лассо (Lasso Tool), выделите основную часть клубники, а затем с помощью инструмента Перемещение (Move Tool), переместите выделенную часть клубники на наш рабочий документ, расположив поверх области губ.

Далее, мы создадим нижнюю губу, поэтому, выберите инструмент Свободная Трансформация (Free Transform Tool (Ctrl+T), активируйте режим Деформации (Warp), а затем потяните обе стороны выделенного фрагмента клубники вверх, а среднюю часть вниз, повторяя форму нижней губы.

Шаг 2

Далее, удерживая клавишу (Ctrl), щёлкните по маске первого корректирующего слоя Цветовой тон / Насыщенность (Hue/Saturation) (который был создан в параграфе 1 Шаг 4), чтобы загрузить активное выделение.

В нижней части панели слоёв, нажмите значок Добавить слой-маску (Add Layer-mask), чтобы добавить слой-маску к слою с фрагментом клубники.

Теперь с помощью круглой кисти средней мягкости, цвет кисти чёрный, аккуратно скройте текстуру клубники поверх верхней губы и правого края уголка нижней губы. Причина удаления текстуры клубники поверх верхней губы заключается в том, что детали текстуры выглядят не очень хорошо, и свет на текстуре не соответствует лицу. Уменьшите непрозрачность кисти при прокрашивании контура, чтобы создать более плавное и естественное совмещение текстуры клубники с существующей губой. Примечание переводчика: чтобы удалить текстуру поверх верхней губы, не забудьте перейти на маску слоя.

Шаг 3

Далее, выделите нижнюю часть клубники, а затем добавьте эту часть на верхнюю губу. С помощью свободной трансформации (Ctrl+T) в режиме Деформации (Warp), немного изогните фрагмент текстуры клубники, повторяя форму верхней губы. Сместите светлые участки текстуры немного влево, чтобы совместить с освещением изображения модели (где свет поступает слева направо).

Шаг 4

Проведите коррекцию текстуры поверх верхней губы, скрыв лишние участки за внешним контуром рта, используя ту же самую технику, которая описана в Шаге 2. Уменьшите непрозрачность кисти, как указано выше, чтобы показать детали контура. Примечание переводчика: не забудьте добавить слой-маску к слою с текстурой клубники.

Шаг 5

Результат всё ещё выглядит плоским и тёмным. Чтобы исправить это, выделите оба слоя с текстурой клубники, а затем нажмите клавиши (Ctrl+G), чтобы сгруппировать их в один слой. Поменяйте режим наложения для слоя с группой на Нормальный (Normal), непрозрачность слоя 100%, а затем удерживая клавишу (Ctrl), щёлкните по маске первого корректирующего слоя Цветовой тон / Насыщенность (Hue/Saturation), чтобы загрузить активное выделение вокруг губ. Значительно усильте яркость выделенной области с помощью корректирующего слоя Кривые (Curves). Примечание переводчика: добавьте корректирующий слой Кривых (Curves) внутри созданной группы.

Далее, перейдите на маску корректирующего слоя Кривые (Curves) и с помощью мягкой чёрной кисти, прокрасьте вдоль области тени под губами, вдоль контура, а также область между губами, чтобы воссоздать светотени. Постарайтесь сделать правую сторону немного темнее, чем левую в соответствии с источником света.

Шаг 6

Добавьте ещё один корректирующий слой Кривые (Curves), потяните кривую вниз, чтобы уменьшить яркость, а затем с помощью мягкой чёрной кисти, скройте коррекцию затемнения, восстановив светлые участки слева, на отдельных семенах справа и в средней части. Сохраните коррекцию кривых видимой на краях и на участках тени для усиления контраста. Варьируйте размер кисти во время прокрашивания, используя клавиши [ и ]. Вы также можете уменьшить непрозрачность кисти, чтобы добиться реалистичного результата.

4. Проводим Заключительные Штрихи

Результат выглядит очень хорошо, но всё ещё незавершённым. Создайте новый слой поверх всех остальных слоёв, поменяйте режим наложения на Перекрытие (Overlay), непрозрачность слоя 100%, атакже выполните заливку 50% нейтральнымсерым (fill it with 50% gray).

Нажмите клавишу "О", чтобы активировать инструмент Осветлитель / Затемнитель (Dodge and Burn Tool (O). Вначале выберите инструмент Осветлитель (Dodge Tool) в панели управления данногоинструмента, установите Диапазон (Range) на Света (Highlights), а значение Экспозиции (Exposure) примерно на 20-25%. Теперь добавьте светлые блики на среднюю часть губ, особенно на левую, а также на верхний контур губ.

Далее, выберите инструмент Затемнитель (Burn Tool) в панели управления данного инструмента, установитеДиапазон (Range) на Средниетона (Midtones), значение Экспозиции (Exposure) примерно 20-25%. Усильте тени, выделяя края между губами, а также добавьте детали на отдельных мелких участках. Удерживайте клавишу (Alt) во время коррекции, чтобы переключаться между инструментами Осветлитель / Затемнитель (Dodgeand Burn Tool (O).

Далее, мы проведём небольшой трюк, поменяйте цвет переднего плана на серый (лучше выбрать #808080, тот же самый цвет, что цвет слоя с заливкой), а затем с помощью инструмента Кисть (Brush Tool), скройте то, что выглядит неверно, а затем вы можете вновь воссоздать детали с помощью инструмента Осветлитель / Затемнитель (Dodgeand Burn Tool (O).
