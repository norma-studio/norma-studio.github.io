Вознаменование этого события, а этот фильм обязательно станет захватывающим,давайте создадим эпический портрет, вдохновлённый Черной Пантерой. В этомуроке вы узнаете, как воссоздать потрясающие портреты африканских воинов изфильма.

Поиск Вдохновения и Стоковых Ресурсов

Для даннойманипуляции, я искала вдохновение, просматривая различные кинопостеры к данномуфильму через Google, в частности стилизацию персонажа, M"Baku, иего впечатляющие доспехи с мехом. Зная это,я решила использовать мех и в голове собрала образ для этого воина из будущего.

Мы создадимдоспехи воина с нуля, поэтому будет неплохо записать некоторые ключевые моменты,когда вы будете продумывать стиль своего персонажа. Ниже представлены мои заметки:

  • Выбратьпортрет с серьёзным и ярким выражением.
  • Создатьэкипировку, используя комбинацию деталей одежды аборигенов и реальные доспехи.
  • Убедиться,чтобы добавляемая одежда подходила в перспективе к оригинальному объекту.
  • Создатьобщую сцену в прохладных тонах с синеватыми оттенками для усиления драматизма.

Почащепросматривайте свои заметки, чтобы сохранить согласованность ключевых моментов. Вы будете ещё благодарить себя за это!

Теперьдавайте приступим!

1. Выставляем Портрет Чёрной Пантеры

Шаг 1

Давайтеначнём с создания нового документа 1396 x 2016 px. Эти размерыне представляют размеры официального постера, не стесняйтесь использовать их,если хотите.

С помощьюинструмента Магнитное лассо (Magnetic Lasso Tool (L), выделите изображение мужчины на исходномизображении. Скопируйте/ вклейте выделенное изображение мужчины на наш рабочий документ, откорректируйтеразмер изображения для стандартного портрета с помощью инструмента СвободнаяТрансформация (Free Transform Tool (Ctrl+T).

Шаг 2

Теперьдобавьте небо на задний фон.

Скопируйте/ вклейте исходное изображение с небом и облаками на наш рабочий документ,расположив слой с небом ниже слоя с мужчиной. Далее,проведите коррекцию масштаба с помощью инструмента Свободная Трансформация (Free Transform Tool (Ctrl+T), сместив изображение с небом немного выше.

Шаг 3

Размойтезадний фон, чтобы показать глубину резкости.

Для этого,перейдите на слой с небом и облаками, а затем идём Фильтр – Размытие- Размытиепо Гауссу (Filter > Blur > Gaussian Blur).

ПрименитеРадиус (Radius) размытия 10 рx, азатем нажмите кнопку ‘OK’.

Результатна данный момент урока.

2. Стилизуем Портрет

Мыподошли к самой интересной части урока! Гардеробу! Для даннойстилизации воина, мы используем комбинацию из предметов одежды аборигенов иподлинных металлических доспехов.

Однакоимейте в виду, что это вымышленный персонаж, поэтому результат будет в действительностизависеть от вас!

Шаг 1

Давайтепостепенно воссоздадим экипировку, начиная с шеи.

Скопируйте/ вклейте исходное изображение с ацтекским украшением на наш рабочий документ,расположив поверх слоя с мужчиной. К слоюс украшением, добавьте слой-маску и с помощью чёрной кисти, удалите верхнюючасть амулета в области шеи. Проделайте то же самое с доспехами.

Продолжайтекоррекцию на слой-маске, чтобы воссоздать экипировку на нашем воине. Исходныематериалы отлично подходят благодаря серому цвету! Ура!

Шаг 2

Теперьдавайте добавим мех!

  1. Создайтевыделение меха на исходном изображении с доспехами и мехом с помощьюинструмента Прямолинейное лассо (Polygonal Lasso Tool (L).
  2. Скопируйте/ Вклейте выделенный мех на наш рабочий документ. Примените масштабирование с помощьюинструмента Свободная Трансформация (Free Transform Tool (Ctrl+T).
  3. Получаем Результат!

Мы можемрастушевать края, чтобы создать ощущение настоящего меха с помощью инструмента Пластика(Liquify).

Находясьна слое с мехом, идём Фильтр- Пластика (Filter > Liquify). Далее,с помощью инструмента Деформация (Forward Warp Tool (W), размажьте острые края изображения, чтобы создатьощущение меха. Продолжайтеформировать мех вокруг мужчины, создавая более аутентичный вид.

Если краявсё ещё выглядят жёсткими, тогда просто воспользуйтесь инструментом Ластик (Eraser Tool (E), чтобы дополнительно растушевать края.

Результатна данный момент урока.

Шаг 3

Давайтедобавим больше аксессуаров! Используяте же самые техники, которые мы использовали ранее, выделите изображенияожерелья из дерева и металлическое украшение на их стоковых изображениях.

Скопируйте/ вклейте выделенные изображения на наш рабочий документ, расположив поверхслоя с мужчиной. Примените масштабирование с помощью инструмента Свободная Трансформация(Free Transform Tool (Ctrl+T). Не бойтесьдобавить слой-маску к каждому слою для коррекцию краёв и удаления любых лишнихучастков.

Шаг 4

Мы подошли к руке. Чтобы воссоздатьметаллическую руку, выделите металлическую кольчугу на исходном изображении с доспехамии мехом. Скопируйте/ вклейте выделенную кольчугу на наш рабочий документ, а затем применитемасштабирование, чтобы кольчуга подошла к руке.

Продублируйтеслой при необходимости, чтобы убедиться, что рука полностью покрыта металлическойкольчугой. Объедините фрагменты кольчуги, если имеются, в один слой вместе с оригинальнымслоем с кольчуги.

Шаг 5

Мыпочти завершили работу с экипировкой!

Последниедетали, которые мы добавим к нашей экипировке – это блестящие шипы, вставленныев ожерелье. С помощьюинструмента Прямолинейное лассо (Polygonal Lasso Tool (L), выделите металлический шип на исходном изображении.

Скопируйте/ вклейте шип на наш рабочий документ, расположив поверх всех остальных слоёв. Расположитешип в нижней части между деревянными бусинами, а затем откорректируйте масштабс помощью инструмента Свободная Трансформация (Free Transform Tool (Ctrl+T).

Растушуйтекрая с помощью простой слой-маски.

Добавьтеещё шипов, создав ещё несколько дубликатов слоя с шипом (Ctrl+J). Расположитекаждый новый шип под соответствующей деревянной бусиной. Отклонитепод углом шипы, чтобы создать красивый эффект!

Этипоследние детали завершают работу над стилистической часть манипуляции. Пришловремя для коррекции цветов и освещения!

Результатна данный момент урока.

3. Проводим Коррекцию Освещения

Продолжаемурок!

Итак, унас есть множество элементов композиции, которые необходимо совместить поосвещению. Давайте рассмотрим этот вопрос, включая коррекцию цветов.

Шаг 1

Начнёмс заднего фона. Создайтеновый корректирующий слой Кривые (Curves),преобразуйте данный корректирующий слой в обтравочную маску к слою с небом иоблаками.

Потянитекривую для RGB иСинего Канала вниз, чтобы затемнить небо.

Шаг 2

Давайтеперейдём к нашему персонажу. Мы постараемсясовместить освещение с фоновым небом, для этого нам понадобится добавитьнесколько корректирующих слоёв.

Добавьтеновый корректирующий слой Цветовой тон / Насыщенность (Hue and Saturation) в качестве обтравочной маски к слою с мужчиной.

Установитеследующие настройки, которые указаны на скриншоте ниже:

  • Цветовойтон (Hue):0
  • Насыщенность (Saturation): -18
  • Яркость (Lightness): -22

Теперьдобавьте новый корректирующий слой Кривые (Curves), преобразуйтеданный корректирующий слой в обтравочную маску к слою с мужчиной.

Установитекривую для RGB Канала,как показано на скриншоте ниже, чтобы затемнить освещение на нашем персонаже.

Шаг 3

Теперьдавайте перейдём к коррекции освещения для экипировки.

Начнёмс ожерелья на шее.

  1. Создайтеновый слой в качестве обтравоной маски к слою с ожерельем на шее. С помощьюмягкой круглой кисти, цвет кисти тёмно-синий #0d1d22,прокрасьте под ожерельем, чтобы создать тени.
  2. Далее,добавьте новый корректирующий слой Цветовой тон / Насыщенность (Hue and Saturation) в качестве обтравочной маски. Установитеследующие настройки: 0, -100 и -70. Далее,поменяйте режим наложения на Жёсткий свет (Hard Light).

Шаг 4

Пришловремя для коррекции освещения на мехе!

Добавьтеновый корректирующий слой Цветовой тон / Насыщенность (Hue and Saturation) в качестве обтравочной маски к слою с мехом.

Установитеследующие настройки, которые указаны на скриншоте ниже:

  • Цветовойтон (Hue): -18
  • Насыщенность(Saturation): -55
  • Яркость (Lightness): -36

Шаг 5

Продолжитепроцесс световой коррекции для металлического украшения на плече.

Добавьтеновый корректирующий слой Цветовой тон / Насыщенность (Hue and Saturation) в качестве обтравочной маски к слою с металлическимукрашением. Уменьшитезначение Яркости (Lightness) до -38, это поможет совместить ожерелье со остальнойэкипировкой.

Шаг 6

Посколькуметаллическая кольчуга на руке похоже отлично совмещена по свету, мы не будем еёкорректировать. Такимобразом, всё, что у нас останется- это деревянное ожерелье.

  1. Создайте новый слой в качестве обтравоной маски к слою с ожерельем. С помощьюмягкой круглой кисти чёрного цвета, добавьте тени поверх ожерелья. Тень будет падать от подбородка мужчины.
  2. Далее,добавьте новый корректирующий слой Цветовой тон / Насыщенность (Hue and Saturation) в качестве обтравочной маски. Установитеследующие настройки: -40, -68 и -67.

4. Проводим Цветовую Коррекцию

Как толькомы завершили коррекцию освещения, давайте проведём цветовую коррекцию!

Соблюдаятональность оригинального изображения, я использую красивый синий тон дляоптимального воздействия.

Шаг 1

Вначале,создайте новый слой в качестве обтравочной маски к слою с металлическим украшениемна плече. С помощьюмягкой круглой кисти, цвет кисти тёмно-сине-зелёного цвета #122b2d,прокрасьте поверх металла. Далее,поменяйте режим наложения для данного слоя на Цветовой тон (Hue) дляокрашивания. Растушуйтекрая с помощью инструмента Ластик (Eraser Tool (E).

Шаг 2

Давайтесоздадим более унифицированную цветовую схему.

Выделитевсе слои, связанные с мужчиной и его экипировкой. Далее,поместите выделенные слои в новую группу.

Добавьтеновый корректирующий слой Кривые (Curves) в качествеобтравочной маски к слою с новой группой. Установитекривые для RGB, Синего и Красного Канала, как показано на скриншотениже для усиления более синих тонов.

Шаг 3

Теперьсоздайте новый слой поверх всех остальных слоёв.

  1. Залейтеэтот слой светло-голубым оттенком #a5c7ff с помощью инструмента Заливка (Paint Bucket Tool (G), а затем с помощью ластика, удалите любые участки,где вы не хотите применять данную коррекцию, например, на лице и груди.
  2. Уменьшитезначение Непрозрачности (Opacity) слоя до 25%, а затем поменяйте режим наложения дляданного слоя на Вычитание (Subtract).

Шаг 4

Добавьте ещё корректирующие слои.

Добавьтеновый корректирующий слой Поиск цвета (Color Lookup) поверх всех остальных слоёв. Установитефильтр filmstock_50.3dl из выпадающегоменю 3D LUT File, а затем поменяйте режим наложения на Яркость (Luminosity).

Шаг 5

Посколькумы используем всё больше корректирующих слоёв, нам необходимо ещё раз сбалансироватьосвещение.

Создайтеновый слой, поменяйте режим наложения для данного слоя на Умножение (Multiply).

С помощьюмягкой круглой кисти, цвет кисти коричневый #2e2523,прокрасьте участки тени по всей сцене. Постарайтесьзаполнить те участки, которые требуют немного больше совмещения.

Шаг 6

Давайтеосветлим лицо мужчины, добавив больше света!

Создайтеновый слой и с помощью мягкой круглой кисти, цвет кисти светло-кремовый #ffebdd,прокрасьте полностью лицо и тело мужчины.

Создайтеновый слой, поменяйте режим наложения для данного слоя на Перекрытие (Overlay), азатем уменьшите значение Непрозрачности (Opacity) до 50%.

Создайтеещё один новый слой поверх предыдущего, установите светло-коричневый цвет кисти#9e7e57, а затем прокрасьте лицо и руку с помощью жёсткойкруглой кисти. Далее,поменяйте режим наложения для данного слоя на Мягкий свет (Soft Light), а также уменьшите Непрозрачность (Opacity) до 50%.

5. Завершаем Фотоманипуляцию

Мыпочти завершили урок!

Чтобызавершить данную манипуляцию, нам необходимо добавить ещё несколько деталей ипровести коррекцию.

Давайтеначнём со шрамов на лице.

Шаг 1

Шрамы налице – это типично для западно-африканских культур. Они частоиспользуются как средство, символизирующее о важной вехе в жизни любогочеловека или женщины.

Чтобы создатьреалистичный эффект рубцов, нарисуйте один кружок коричневого цвет #704d4b нахолсте. Затем,добавьте светло-коричневый блик #c7836c вцентре нарисованного кружка. Продублируйтесозданный кружок (Ctrl+J) несколько раз, создав ряды рубцов, которые вы можетенанести прямо на кожу.

Шаг 2

Объединитеслои с нарисованными кружками, расположив их на одной щеке. Деформируйтерубцы с помощью инструмента Свободная трансформация (Free Transform Tool (Ctrl+T), чтобы они выглядели реалистичными.

Продублируйтеобъединённый слой с рубцами, а затем отразите по горизонтали дубликат слоя, дляэтого идём Редактирование – Трансформирование - Отразить по горизонтали (Edit > Transform > Flip Horizontal). Смягчитеэффект с помощью инструмента Ластик (Eraser Tool (E).

Шаг 3

Давайтедополнительно проведём цветовую коррекцию.

Добавьтеновый корректирующий слой Цветовой Баланс (Color Balance).

Применитеследующие настройки для Теней (Shadows), Средних тонов (Midtones) иСвета (Highlights). Далее,поменяйте режим наложения для данного корректирующего слоя на Светлее (Lighter Color), а также установите Непрозрачность (Opacity) слояна 54%.

Далее,добавьте новый корректирующий слой Уровни (Levels).

Установитеследующие настройки для RGB, Красного и Синего Канала. Далее,поменяйте режим наложения для данного корректирующего слоя на Перекрытие (Overlay), атакже установите Непрозрачность (Opacity) слоя на 20%.

Выможете увидеть результат на скриншоте ниже. Итак,мы создали более голубовато-зелёную цветовую гамму, схожую с оригинальными кинопостерами.

Шаг 4

Давайтеусилим насыщенность! Теперьдобавьте новый корректирующий слой Поиск цвета (Color Lookup).

Установитефильтр Beach Bypass.look из выпадающегоменю 3DLUT File, а затем поменяйте режим наложения для данногокорректирующего слоя на Яркость (Luminosity). Уменьшитезначение Непрозрачности (Opacity) до 30%.

Шаг 5

Не торопясь,исправьте любые дефекты с помощью инструмента Кисть (Brush Tool (B). Простопрокрасьте поверх этих участков, чтобы сделать линии более чёткими и ровными. Большевсего требуют коррекции участки вокруг ожерелья.

Далее,создайте новый слой, поменяйте режим наложения для данного слоя на Перекрытие (Overlay), азатем с помощью кисти ярко-жёлтого цвета #d2d4d0,прокрасьте небо и мужчину, усиливая блики. Добавьтеещё один слой для усиления бликов, чтобы показать, что наш воин освещаетсясзади.

Шаг 6

Давайтезавершим нашу фотоманипуляцию!

Выделитефрагмент облаков на исходном изображение с облаками и дымом, а затем добавьтевыделенный фрагмент на наш рабочий документ, расположив в нижнем углу нашейсцены. Растушуйтекрая с помощью слой-маски или инструмента Ластик (Eraser Tool (E).

Добавьтепоследний корректирующий слой Кривые (Curves), а такжедобавьте белые блики, где они необходимы на лице и доспехах.

Я такжерешила добавить несколько белых точек в небе для усиления атмосферы.
