Создаём Фантазийную Фотоманипуляцию с Портретом в Adobe Photoshop...

Интро

В этом уроке я покажу вам, как создать научно-фантастический портрет. Вы узнаете, как совместить портрет модели с космическим фоном, как создавать световые эффекты, используя текстуры со световыми эффектами и туманностью, применять режимы наложения, усиливать цвета, добавлять свет и многое другое....

В этом уроке я покажу вам, как создать научно-фантастический портрет. Вы узнаете, как совместить портрет модели с космическим фоном, как создавать световые эффекты, используя текстуры со световыми эффектами и туманностью, применять режимы наложения, усиливать цвета, добавлять свет и многое другое.

Исходные материалы

Следующие исходные материалыбыли использованы для создания данного урока:

  • Модель
  • Задний фон
  • Световая текстура
  • Туманность 1
  • Туманность 2

1. Создаём Задний Фон

Шаг 1

Создайте новый документ 1500 x 900 px в программе Photoshop, установите следующие настройки, которые указаны ниже:

Установите цвет заднего фона #010e27.

Шаг 2

Идём Слой- Новый корректирующий слой - Кривые (Layer > New Adjustment Layer > Curves), чтобы уменьшить яркость. Далее, перейдите на маску данного корректирующего слоя, а затем выберите инструмент Кисть (Brush Tool (B). С помощью мягкой чёрной кисти, скройте эффект коррекции с обеих сторон холста. Это делается для того, чтобы эти стороны не были слишком тёмными, а также чтобы были видны детали в дальнейшем.

Шаг 3

Откройте исходную текстуру с туманностью. Переместите её на наш рабочий документ, расположив поверх остальных слоёв. Поменяйте режим наложения для слоя с текстурой туманности на Осветление (Screen), Непрозрачность слоя 100%:

Нажмите на второй значок в нижней части панели слоёв, чтобы добавить слой-маску к слою текстурой туманности. Далее, с помощью мягкой чёрной кисти, прокрасьте отдельные участки текстуры в центре, а также по обеим сторонам.

Шаг 4

К слою с текстурой туманности, добавьте корректирующий слой Кривые (Curves) в качестве обтравочной маски. Уменьшите яркость, чтобы сделать эффект более нежным.

2. Добавляем Модель

Шаг 1

Откройте исходное изображение с моделью в программе Photoshop. Отделите изображение модели от заднего фона с помощью инструмента Прямолинейного лассо (Polygonal Lasso Tool (L), а затем переместите выделенное изображение модели на наш рабочий документ с помощью инструмента Перемещение (Move Tool (V), расположив в центре холста.

К слою с моделью, добавьте слой-маску и с помощью мягкой чёрной кисти, прокрасьте область шеи, плеч, волос, включая лобную часть.

Шаг 2

Создайте новый слой поверх слоя с моделью в качестве обтравочной маски. Далее, с помощью инструмента Штамп (Clone Tool (S), удалите серёжку.

Шаг 3

Далее, добавьте корректирующий слой Кривые (Curves), чтобы уменьшить яркость нижнего края подбородка и щеки. Далее, перейдите на маску данного корректирующего слояи с помощью мягкой чёрной кисти, прокрасьте другие участки, чтобы скрыть коррекцию корректирующего слоя.

Шаг 4

Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски к слою с моделью. Уменьшите значение Насыщенности (Saturation) до -16:

Шаг 5

Далее, добавьте корректирующий слой Цветовой Баланс (Color Balance) в качестве обтравочной маски, чтобы совместить цвет лица модели с задним фоном. Поменяйте настройки для Средних тонов (Midtones):

Шаг 6

С помощью инструмента Лассо (Lasso Tool), выделите только область губ, а затем идём Слой- Новый корректирующий слой - Кривые (Layer > New Adjustment Layer > Curves). Потяните кривую вверх, чтобы сделать выделенную область ярче.

3. Создаём Макияж для Глаз

Шаг 1

Откройте исходную Световую текстуру в программе Photoshop. Переместите данную текстуру на наш рабочий документ с помощью инструмента Перемещение (Move Tool). Поменяйте режим наложения для данного слоя текстурой на Осветление (Screen), непрозрачность слоя 100%.

Шаг 2

Далее, идём Фильтр - Пластика (Filter > Liquify), выберите инструмент Скручивание по часовой стрелке (Twirl Clockwise Tool (C). Скрутите световое свечение, пока у вас не получится результат, как на скриншоте ниже или по крайне мере, до тех пор, пока не появится область свечения в виде глаза. Закройте свечением левый глаз.

С помощью слой-маски, скройте область свечение вокруг глаза.

Шаг 3

Продублируйте слой со световым эффектом, а затем отразите дубликат слоя по горизонтали, для этого идём Редактирование - Трансформация - Отразить по горизонтали (Edit > Transform > Flip Horizontal). Сместите второе свечение на правый глаз.

Шаг 4

Выделите оба слоя со свечением поверх глаз, а затем нажмите клавиши (Ctrl+G), чтобы сгруппировать их в одну группу. Поменяйте режим наложения для слоя с группой на Осветление (Screen), непрозрачность слоя 100%, а затем добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation), чтобы изменить цвет свечения на глазах.

Шаг 5

Далее, добавьте корректирующий слой Цветовой Баланс (Color Balance). Поменяйте настройки для Средних тонов (Midtones):

4. Добавляем Эффект Туманности

Шаг 1

Откройте исходное изображение с туманностью в программе Photoshop. Переместите данную текстуру на наш рабочий документ, расположив поверх слоя с головой модели. Поменяйте режим наложения для данного слоя с текстурой на Осветление (Screen), непрозрачность слоя 100%. С помощью инструмента Свободная трансформация (Free Transform Tool (Ctrl+T) в режиме Деформация (Warp), деформируйте текстуру туманности в соответствии с перспективой головы и глазами.

С помощью слой-маски и мягкой чёрной кисти, скройте жёсткие края.

Шаг 2

Продублируйте слой с текстурой туманности несколько раз, расположив дубликаты слоёв вокруг головы и лица модели, особенно постарайтесь, чтобы световые эффекты были видны на щеках. При необходимости используйте слой-маску, чтобы скрыть лишние детали и получить желаемый вид.

Шаг 3

Сгруппируйте слои с туманностью в одну группу, а затем поменяйте режим наложения для данного слоя с группой на Осветление (Screen), непрозрачность слоя 100%. Далее, добавьте корректирующий слой Цветовой Баланс (Color Balance). Поменяйте настройки для Средних тонов (Midtones) и Света (Highlights):

Шаг 4

Добавьте корректирующий слой Уровни (Levels), чтобы скрыть все оставшиеся края и сделать эффект более нежным.

Шаг 5

Добавьте второе исходное изображение с текстурой туманности на наш рабочий документ, расположив в нижней части холста. С помощью свободной трансформации (Ctrl+T), поверните текстуру, как показано на скриншоте ниже. Поменяйте режим наложения для данного слоя с текстурой туманности на Осветление (Screen), непрозрачность слоя 100%.

Шаг 6

Добавьте корректирующий слой Уровни (Levels), чтобы скрыть края и сделать эффект более нежным.

Шаг 7

Далее, добавьте корректирующий слой Цветовой Баланс (Color Balance). Поменяйте настройки для Средних тонов (Midtones).

5. Добавляем Волшебный Свет

Шаг 1

Откройте исходное изображение со свечением туманности, переместите данное изображение на наш рабочий документ, расположив поверх слоя с головой модели. Поменяйте режим наложения для данного слоя с туманностью на Осветление (Screen), непрозрачность слоя 100%.

С помощью инструмента Пластика (Liquify Tool (инструмент Деформация (Forward Warp Tool), деформируйте свечение, чтобы получить результат, как на скриншоте ниже.

Шаг 2

Дважды продублируйте слой со свечением туманности, поместите одни дубликат с туманности поверх лица, а второй в правой части сцены. С помощью слой-маски, удалите любые детали, которые вы не хотите видеть.

Шаг 3

С помощью инструмента Лассо (Lasso Tool), выделите левую часть туманности на оригинальном изображении со свечением туманности, переместите выделенную часть туманности на наш рабочий документ, поместив поверх нижней левой части головы модели. Превратите её в кривую с помощью инструмента Деформация (Forward Warp Tool).

С помощью слой-маски, скройте жёсткие края туманности.

Шаг 4

Выделите ещё один фрагмент туманности на изображении со свечением туманности, чтобы проделать ту же самую операцию. Добавьте этот фрагмент туманности с другой стороны головы модели.

Шаг 5

Добавьте мелких деталей, используя ту же самую технику.

Шаг 6

Сгруппируйте все слои со световыми эффектами в одну группу. Далее, добавьте корректирующий слой Цветовой Баланс (Color Balance) внутри созданной группы. Поменяйте настройки для Средних тонов (Midtones).

Шаг 7

Далее, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation ). Уменьшите значение Насыщенности (saturation) для Синих (Blue) тонов до -81:

Шаг 8

Добавьте корректирующий слой Уровни (Levels), чтобы оставшиеся края полностью исчезли.

6. Создаём Макияж для Лица

Шаг 1

Создайте новый слой, а затем с помощью инструмента Овальная область (Elliptical Marquee Tool (M), создайте круглое выделение. Далее, выберите инструмент Градиент (Gradient Tool (G), установите Радиальный (Radial) градиент, цвета градиента: #f8b2f2 и #834115. Залейте градиентом созданное выделение.

С помощью инструмента Деформация (Forward Warp Tool), деформируйте круг, расположив его вдоль контура лица модели.

Шаг 2

Поменяйте режим наложения для слоя с деформированным кругом на Осветление основы (Color Dodge), непрозрачность слоя 100%. Добавьте слой-маску к слою с кругом и с помощью мягкой чёрной кисти, создайте нежный переход вдоль нижней части щеки.

Шаг 3

Дважды продублируйте слой с деформированным кругом. Один дубликат слоя расположите поверх щеки справа, а второй отразите по горизонтали, расположив на другой щеке.

7. Добавляем Световые Эффекты

Шаг 1

Создайте новый слой поверх всех остальных слоёв. С помощью мягкой кисти, цвет кисти #746482, прокрасьте вокруг лица модели. Поменяйте режим наложения для слоя с световым эффектом на Осветление основы (Color Dodge), непрозрачность слоя 100%.

Шаг 2

Создайте ещё один слой поверх всех остальных слоёв. С помощью мягкой кисти, цвет кисти #81a3ef, прокрасьте над головой модели. а также поверх щеки и подбородка. Поменяйте режим наложения для данного слоя на Перекрытие (Overlay), непрозрачность 100%.

Шаг 3

Создайте ещё один слой. С помощью мягкой кисти, цвет кисти #8c8dcb, прокрасьте нижнюю область сцены. Поменяйте режим наложения для данного слоя на Жёсткий свет (Hard Light), непрозрачность 30%.

Шаг 4

Создайте ещё один слой. С помощью мягкой кисти, цвет кисти #a797c1, прокрасьте верхнюю левую область головы модели. Поменяйте режим наложения для данного слоя на Жёсткий свет (Hard Light), непрозрачность 30%.

8. Проводим Заключительную Коррекцию

Шаг 1

Добавьте корректирующий слой Карта градиента (Gradient Map) поверх всех остальных слоёв. Установитеследующие цвета градиента #912d1d и #004460. Уменьшите Непрозрачность (opacity) данногокорректирующего слоя до 20%.

Шаг 2

Добавьте корректирующий слой Цветовой Баланс (Color Balance). Поменяйте настройки для Среднихтонов (Midtones) и Света (Highlights):

Шаг 3

Добавьте корректирующий слой Фото фильтр (Photo Filter). Установите следующий цветовой оттенок для данного фильтра #ecde00.

Шаг 4

Добавьте корректирующий слой Кривые (Curves),чтобы добавить тёплых тонов и усилить контрастность изображения.

Шаг 5

Добавьте корректирующий слой Фото фильтр (Photo Filter). Установите следующий цветовой оттенок для данного фильтра #dab9f4.

Шаг 6

Добавьте корректирующий слой Вибрация (Vibrance), чтобы усилить заключительный эффект.
