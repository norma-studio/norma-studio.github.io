В этом уроке я покажу вам, как создать сцену с мистическим лесом и волшебным оленем. Вначале, мы создадим основу для сцены, используя несколько исходных изображений с лесом и деревьями. Далее, мы добавим изображение оленя и украсим его голову ветвями и листьями. После этого, мы проведём небольшую коррекцию цвета, а также добавим эффекты освещения к сцене. В заключение, мы применим несколько корректирующих слоёв для усиления заключительного эффекта.

Исходные материалы

Следующие исходные материалы были использованы длясоздания данного урока:

  • Лес
  • Олень
  • Сухие ветки
  • Дерево 1
  • Дерево 2
  • Дерево 3
  • Листья

1. Создаём Основу для Нашей Сцены

Шаг 1

Создайте новый документ 2000 х 1130 рх в программе Photoshop. Установите настройки,которые указаны на скриншоте ниже.

Шаг 2

Откройте исходное изображение с лесом. Переместите данное изображение на наш рабочий документ с помощью инструмента Перемещение (Move Tool (V). С помощью инструмента Свободная трансформация (Free Transform Tool (Ctrl+T) в режиме Деформация (Warp), измените перспективу изображения, сделав верхнюю часть леса уже, а нижнюю часть шире. Примечание переводчика: не забудьте преобразовать слой с лесом в смарт-объект, иначе у вас не получиться следующий шаг.

Шаг 3

Далее, идём Фильтр - Размытие- Размытие по Гауссу (Filter > Blur > Gaussian Blur). Установите Радиус (Radius) размытия на 8 px:

Перейдите на маску фильтра и с помощью мягкой круглой кисти чёрного цвета, скройте эффект размытия на заднем плане, сохранив размытие только на переднем плане. Цель этого шага- добавить глубину к сцене.

Шаг 4

Нам нужно удалить следы на земле, а также дерево, которое стоит у левого края сцены (я планирую добавить другое дерево на этот край, поэтому, эту область необходимо расчистить, а затем затемнить, чтобы дерево выделялось). Создайте новый слой поверх слоя с лесом в качестве обтравочной маски. Выберите инструмент Штамп (Clone Tool (S), чтобы скрыть указанные детали.

Шаг 5

Далее, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски к слою с лесом. Идём Слой- Новый корректирующий слой- Цветовой тон / Насыщенность (Layer > New Adjustment Layer > Hue/Saturation). Уменьшите значение Насыщенности (Saturation) до -62:

Шаг 6

Добавьте корректирующий слой Кривые (Curves), чтобы уменьшить яркость изображения леса. Далее, перейдите на маску данного корректирующего слояи с помощью мягкой чёрной кисти, скройте эффект затемнения на участках, которые выделены на скриншоте ниже, в основном это средняя часть сцены.

2. Добавляем Деревья

Шаг 1

Откройте исходное изображение с Деревом 1. С помощью инструмента Прямолинейное лассо (Polygonal Lasso Tool (L), выделите дерево, а также область вокруг корней. Переместите выделенное изображение дерева на наш рабочий документ, расположив его у левого края сцены. Отразите это дерево по горизонтали, для этого идём Редактирование - Трансформация - Отразить по горизонтали (Edit > Transform > Flip Horizontal).

Нажмите второй значок в нижней части панели слоёв, чтобы добавить слой-маску к данному слою с деревом. Далее, с помощью мягкой чёрной кисти, удалите жёсткие края на изображении, чтобы совместить корни дерева с существующей землёй. Примечание переводчика: не забудьте перейти на маску слоя.

Шаг 2

Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation).Уменьшите значение Насыщенности (Saturation) до -95: Примечание переводчика: незабудьте преобразовать корректирующий слой в этом и в следующих шагах вобтравочные маски к слою с деревом.

Шаг 3

Добавьте корректирующий слой Кривые (Curves), чтобы уменьшить яркость дерева. Далее, перейдите на маску данного корректирующего слояи с помощью мягкой чёрной кисти, скройте эффект затемнения во фронтальной части дерева, сохранив затемнение с другой стороны дерева. Я планирую добавить основной источник света в средней части сцены, поэтому, фронтальная часть дерева будет светлее, чем остальные части дерева.

Шаг 4

Добавьте ещё один корректирующий слой Кривые (Curves), чтобы усилить яркость фронтальной части дерева. Далее, перейдите на маску данного корректирующего слояи с помощью мягкой чёрной кисти, скройте осветление поверх остальных участков дерева, сохраняя их тёмными.

Шаг 5

Мне не нравится текстура дерева 1, поэтому, я решила использовать другое дерево, чтобы заменить эту текстуру. Откройте второе исходное изображение с деревом 2. Выделите только нижнюю часть ствола. Переместите выделенную часть ствола на наш рабочий документ, расположив поверх дерева 1. С помощью свободной трансформации (Ctrl+T) в режиме Деформация (Warp), деформируйте ствол дерева 2 в соответствии с формой дерева 1.

С помощью слой-маски, совместите дерево 2 с существующим деревом, а также восстановите детали корней дерева 1. Примечание переводчика: 1. к слою со стволом дерева 2, добавьте слой-маску 2. с помощью кисти чёрного цвета, обработайте жёсткие края, чтобы создать плавный переход.

Шаг 6

Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation). Уменьшите значение Насыщенности(Saturation) до -67:

Шаг 7

Добавьте корректирующий слой Кривые (Curves), чтобы усилить яркость фронтальной части дерева 2. Далее, перейдите на маску данного корректирующего слоя и с помощью мягкой чёрной кисти, скройте осветление поверх остальных участков дерева, сохраняя их тёмными.

Шаг 8

Добавьте корректирующий слой Уровни (Levels) с той же самой целью. На скриншоте ниже выделены участки, на которых вам необходимоскрыть коррекцию осветления на маске уровней.

Шаг 9

Откройте третье исходное изображение с деревом 3. Отделите дерево от заднего фона с помощью инструмента Прямолинейное лассо (Polygonal Lasso Tool), а затем переместите выделенное изображение дерева на наш рабочий документ, расположив его в правой части сцены. Отразите данное изображение с деревом по горизонтали.

К данному слою с деревом, добавьте слой-маску и с помощью мягкой чёрной кисти, совместите нижнюю часть дерева с землей.

Шаг 10

Создайте новый слой и с помощью инструмента Штамп (Clone Tool), удалите веточку с листьями у основания корней.

Шаг 11

Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation). Уменьшите значение Насыщенности(Saturation) до -69:

Шаг 12

Добавьте корректирующий слой Кривые (Curves) для усиления яркости дерева. Далее, перейдите на маску данного корректирующего слояи с помощью мягкой чёрной кисти, скройте эффекты осветления поверх правой части дерева, т.к. эта часть скрыта от света.

3. Проводим Базовую Коррекцию

Добавьте корректирующий слой Кривые (Curves) поверх всех остальных слоёв, чтобы затемнить всю сцену. Далее, перейдите на маску данного корректирующего слояи с помощью мягкой чёрной кисти с низкой непрозрачностью (20-25%), слегка скройте эффект затемнения в средней части сцены, а также на отдельных участках деревьев.

4. Добавляем Оленя

Шаг 1

Откройте исходное изображение с оленем. Отделите оленя от заднего фона, а затем переместите выделенное изображение на наш рабочий документ, расположив его в центре нашей сцены. Примечание переводчика: преобразуйте слой с оленем в смарт-объект.

Шаг 2

Изображение оленя выглядит слишком отчётливым на фоне туманного леса. Чтобы исправить это, добавьте слой-заливку Цвет (Color Fill), установите цвет заливки #857771. Уменьшите Непрозрачность (Opacity) слоя с заливкой до 30%. Примечание переводчика: не забудьте преобразовать данный корректирующий слой, а также следующие корректирующие слои в обтравочные маски к слою с оленем.

Шаг 3

Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation). Уменьшите значение Насыщенности(Saturation) до -40.

Шаг 4

Добавьте корректирующий слой Кривые (Curves), чтобы затемнить оленя. Далее, перейдите на маску данного корректирующего слоя и с помощью мягкой чёрной кисти, прокрасьте внешние края изображения оленя, т.к. на эти участки в основном попадает свет.

Шаг 5

Добавьте ещё один корректирующий слой Кривые (Curves), чтобы изменить контраст изображения оленя и сделать его немного темнее и затуманенным.

Шаг 6

Создайте новый слой и с помощью мягкой кисти, цвет кисти #d1d9dd, добавьте светлые блики на внешний контур изображения оленя. Поменяйте режим наложения для данного слоя на Перекрытие (Overlay), непрозрачность слоя 100%.

Шаг 7

Края изображения оленя выглядят всё ещё слишком жёсткими. Чтобы исправить это, к слою с оленем, примените фильтр Размытие по Гауссу (Gaussian Blur), радиус размытия 8 px. Перейдите на маску фильтра и с помощью мягкой чёрной кисти, скройте эффект размытия, сохранив размытие только на краях изображения оленя.

Шаг 8

Я планирую добавить основной источник света на заднем плане в центре сцены, поэтому, тень оленя должна падать на землю. Создайте новый слой ниже слоя с оленем. Удерживая клавишу (Ctrl), щёлкните по миниатюре слоя с оленем, чтобы загрузить активное выделение. Залейте выделенную область чёрным цветом, а затем отразите тень по вертикали. С помощью свободной трансформации (Ctrl+T), расположите тень на земле.

Уменьшите Непрозрачность (Opacity) слоя с тенью до 60%, а затем примените к этому же слою с тенью фильтр Размытие по Гауссу (Gaussian Blur). Установите радиус размытия 4 px.

5. Добавляем Ветви и Листья

Шаг 1

Откройте исходное изображение с сухими ветвями. Отделите ветви от белого заднего фона, а затем переместите выделенные ветви на наш рабочий документ, расположив поверх рогов оленя. С помощью свободной трансформации (Ctrl+T) в режиме Деформация (Warp), деформируйте ветви в соответствии с формой рогов.

С помощью слой-маски, совместите ветви с рогами.

Шаг 2

Сгруппируйте слои с ветвями в одну группу (Ctrl+G). Добавьте корректирующий слой Кривые (Curves) внутри созданной группы, чтобы затемнить ветви. Примечание переводчика: поменяйте режим наложения для слоя с группой на Нормальный (Normal), непрозрачностьслоя 100%.

Шаг 3

Откройте исходное изображение с листьями. Используйте различные фрагменты листьев, чтобы расположить их на ветвях. Создайте дубликаты слоёв с листьями, примените к ним масштабирование и поворот, чтобы они не выглядели однообразно.

Шаг 4

Сгруппируйте слои с листьями в одну группу (Ctrl+G). Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) внутри созданной группы.Уменьшите значение Насыщенности (Saturation) до -38. Примечание переводчика: поменяйте режим наложения для слоя с группой на Нормальный (Normal), непрозрачность слоя 100%.

Шаг 5

Добавьте корректирующий слой Кривые (Curves), чтобы затемнить листья.

Шаг 6

Добавьте тени от ветвей и листьев таким же образом, как мы создали тень от оленя. Расположите эти тени поверх тени от рогов оленя, а также уменьшите Непрозрачность (Opacity) этого слоя с тенями до 30%, т.к. эти тени более удалены от источника света, чем тень от рогов.

6. Заключительная Коррекция

Шаг 1

Добавьте слой-заливку Цвет (Color Fill) поверх всех слоёв. Установите цвет заливки #909090. Уменьшите Непрозрачность (Opacity) данного слоя с заливкой до 20%.

Далее, перейдите на маску данного корректирующего слояи с помощью мягкой чёрной кисти Непрозрачностью (Opacity) примерно 40-45%, скройте эффект дымки на участках, которые выделены на скриншоте ниже, особенно на краях.

Шаг 2

Добавьте корректирующий слой Кривые (Curves), чтобы уменьшить яркость. Далее, перейдите на маску данного корректирующего слояи с помощью мягкой чёрной кисти, скройте эффект затемнения в средней части сцены.

Шаг 3

Создайте новый слой и с помощью мягкой кисти, цвет кисти #f2f2f2, добавьте свет в самую дальнюю часть леса на заднем плане. Далее, поменяйте режим наложения для данного слоя наМягкий свет (Soft Light), непрозрачность слоя 100% и с помощью слой-маски, ослабьте свет над оленем, а также вокруг ног оленя.

Шаг 4

Создайте новый слой, а затем поменяйте режим наложения для данного слоя на Перекрытие (Overlay ), непрозрачность слоя 100%. С помощью всё той же самой кисти, прокрасьте фронтальные стороны деревьев, которые выделены на скриншоте ниже.

Шаг 5

Добавьте корректирующий слой Карта градиента (Gradient Map). Установите следующие цвета градиента #6f5740 и #0082fb. Далее, поменяйте режим наложения для данного корректирующего слоя на Мягкий свет (Soft Light), непрозрачность слоя 90%.

Шаг 6

Добавьте ещё один корректирующий слой Карта градиента (Gradient Map). Установитеследующие цвета градиента #996a65 и #c6e6d3. Далее, поменяйте режим наложения для данногокорректирующего слоя на Мягкий свет (Soft Light), непрозрачность слоя 100%.

Шаг 7

Добавьте корректирующий слой Цветовой Баланс (Color Balance). Поменяйте настройки для Среднихтонов (Midtones).

Шаг 8

Добавьте слой-заливку Цвет (Color Fill) поверх всех слоёв. Установите цвет заливки #1f0107. Поменяйте режим наложения для данного слоя с заливкой на Исключение (Exclusion), непрозрачность слоя 100%.

Шаг 9

Добавьте ещё один корректирующий слой Карта градиента (Gradient Map). Установитеследующие цвета градиента: #847c74 и #62a0b5. Далее, поменяйте режим наложения для данного корректирующего слоя на Мягкий свет (Soft Light), непрозрачность слоя 100%.

Шаг 10

Добавьте корректирующий слой Фото фильтр (Photo Filter). Установите следующий цветовой оттенок для данного фильтра #0149ec.

Шаг 11
