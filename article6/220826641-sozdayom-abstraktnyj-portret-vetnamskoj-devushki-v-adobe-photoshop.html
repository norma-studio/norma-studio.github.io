Создаём Абстрактный Портрет Вьетнамской Девушки в Adobe Photoshop...

Design & IllustrationPhoto Manipulation

Интро

В этом уроке я покажу вам, как создать абстрактную сцену с молодой вьетнамкой в традиционном длинном платье. Длинные платья, лотос и бамбук известны, как культурные символы Вьетнама. ...

В этом уроке я покажу вам, как создать абстрактную сцену с молодой вьетнамкой в традиционном длинном платье. Длинные платья, лотос и бамбук известны, как культурные символы Вьетнама.

Вначале мы создадим задний фон, используя старую текстуру и несколько текстур акварели, после этого, мы добавим изображения модели, лотоса и веточек бамбука. Позже мы добавим световые эффекты, немного брызг и эффект боке. Мы применим несколько корректирующих слоёв для завершения итогового результата.

Черпайте вдохновение! Создавайте мгновенные красивые абстрактные портреты с помощью Экшенов Photoshop от GraphicRiver.

1. Создаём Задний Фон

Шаг 1

Создайте новый документ 2000 x 1500 px в программе Photoshop. Содержимое заднего фона белый цвет. Установите следующие настройки, которые указаны на скриншоте ниже:

Шаг 2

Откройте исходную текстуру Старой Бумаги. Переместите данную текстуру на наш рабочий документ с помощью инструмента Перемещение (Move Tool (V). Уменьшите значение Непрозрачности (Opacity) данного слоя с текстурой до 40%:

Шаг 3

Далее, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation), для этого идём Слой- Новый корректирующий слой - Цветовой тон / Насыщенность (Layer > New Adjustment Layer > Hue/Saturation). Уменьшите значение Насыщенности (Saturation) до -54.

Шаг 4

Нажмите второй значок в нижней части панели слоёв, чтобы добавить слой-маску к слою с текстурой. Перейдите на данную слой-маску, а затем выберите инструмент Кисть (Brush Tool (B). Установите мягкую круглую кисть, цвет кисти чёрный. Теперь прокрасьте кистью левую область ближе к середине холста. Это позволит добавить яркость в соответствии с освещением на изображении модели (мы увидим это в следующей части урока).

Шаг 5

Добавьте корректирующий слой Цветовой Баланс (Color Balance). Поменяйте настройки для Средних тонов (Midtones).

Шаг 6

Откройте исходный набор текстур Акварели. Выберите текстуру с названием 1030 или ту текстуру, которая вам нравится, а затем поместите её на наш рабочий документ. С помощью инструмента Свободная трансформация (Free Transform Tool (Ctrl+T), примените масштабирование к текстуре Акварели в соответствии с размерами холста. Поменяйте режим наложения на Умножение (Multiply), непрозрачность слоя 100%.

Добавьте слой-маску к слою с текстурой Акварели, а затем с помощью мягкой чёрной кисти с низкой Непрозрачностью (Opacity) примерно 30-35%, ослабьте видимость эффекта акварели. Примечание переводчика: не забудьте перейти на слой-маску.

Шаг 7

Выберите текстуру Акварели с названием 1029, поместите данную текстуру в правой части холста. Добавьте слой-маску, а затем с помощью мягкой чёрной кисти, скройте часть текстуры, сохранив только голубые фрагменты в середине холста.

Шаг 8

Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) поверх данного слоя с текстурой в качестве обтравочной маски. Поменяйте значения для Всех тонов (Master) и Голубых (Cyans), чтобы изменить цвет видимых голубых фрагментов. Примечание переводчика: добавьте корректирующий слой ко второй текстуре акварели, которую мы только что добавили.

Шаг 9

Добавьте исходную текстуру Акварели 2 в среднюю часть холста. Поменяйте режим наложения на Умножение (Multiply), а также уменьшите значение Непрозрачности (Opacity) данного слоя с текстурой до 50%. С помощью слой-маски, проведите совмещение с существующим задним фоном.

2. Добавляем Изображение Модели

Шаг 1

Выделите изображение модели, а затем с помощью инструмента Перемещение (Move Tool (V), переместите выделенное изображение модели на наш рабочий документ, расположив в центре холста.

Шаг 2

Добавьте слой-маску к слою с моделью. Активируйте кисти Брызги Акварели. Выберите ту кисть, которая вам понравилась, а затем нажмите клавишу (F5), чтобы поменять настройки кисти, как показано на скриншоте ниже:

С помощью настроенной кисти, смягчите нижнюю часть изображения модели, чтобы совместить её с задним фоном.

Шаг 3

Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски к слою с моделью. Поменяйте настройки для Всех тонов (Master), Красных (Reds) и Пурпурных (Magentas). Основная цель- это поменять цвет платья.

Шаг 4

Добавьте корректирующий слой Цветовой Баланс (Color Balance) в качестве обтравочной маски к слою с моделью, чтобы добавить к модели немного яркости.

Шаг 5

Далее, добавьте корректирующий слой Кривые (Curves) в качестве обтравочной маски к слою с моделью, чтобы немного затемнить переднюю часть модели. Установите кривую, как показано на скриншоте ниже. Перейдите на маску корректирующего слоя Кривые (Curves) и с помощью мягкой чёрной кисти, скройте коррекцию кривых поверх области спины модели и участков тени.

3. Добавляем Цветки Лотоса

Шаг 1

Вырежьте цветок лотоса, поместите его на левом плече модели. Продублируйте слой со цветком лотоса четыре раза, поместив дубликаты цветков на спине, волосах и лице модели. С помощью Свободной трансформации (Ctrl+T), поменяйте размер и направление цветков.

Шаг 2

Добавьте слой-маски к слоям с лотосом на плече и лице. С помощью мягкой чёрной кисти средних размеров, совместите цветок лотоса с плечом модели. Далее, перейдите на маску слоя со цветком на лице модели, чтобы удалить весь цветок, оставив лишь два лепестка возле глаза.

Шаг 3

Выделите все слои со цветками лотоса, а затем нажмите клавиши (Ctrl+G), чтобы сгруппировать их в одну группу. Поменяйте режим наложения для данного слоя с группой на Нормальный (Normal), непрозрачность слоя 100%, а также добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) внутри созданной группы. Увеличьте значение Цветового тона (Hue) до +15:

Шаг 4

Далее, добавьте корректирующий слой Цветовой Баланс (Color Balance), чтобы усилить красные тона цветков лотоса. Примечание переводчика: поменяйте значение красных для Средних тонов (Midtones).

Шаг 5

В конце, добавьте корректирующий слой Кривые (Curves) для осветления цветков лотоса. Установите кривую, как показано на скриншоте ниже. Перейдите на маску корректирующего слой Кривые (Curves) и с помощью мягкой чёрной кисти, скройте коррекцию осветления поверх участков тени.

4. Украшаем Сцену Веточками Бамбука

Шаг 1

Выделите веточку бамбука, поместив её в любом месте на холсте. Расположите слой с веточкой бамбука ниже слоя с группой со цветками лотоса.

Далее, идём Фильтр- Пластика (Filter > Liquify), выберите инструмент Деформация (Forward Warp Tool). С помощью данного инструмента, вытяните веточку бамбука, придав ей изогнутую форму, как у меня.

Поместите деформированную веточку бамбука за большим цветком лотоса.

Шаг 2

С помощью слой-маски, скройте верхнюю часть веточки бамбука.

Шаг 3

Используйте данную технику, чтобы добавить ещё различных деформированных веточек бамбука. Расположите их вокруг модели, над головой, вдоль волос и сбоку шляпы. При необходимости, используйте слой-маску, чтобы совместить веточки с головой модели или показать их за изображением модели или впереди модели.

Шаг 4

Сгруппируйте все слои с веточками бамбука. Добавьте корректирующий слой Цветовой Баланс (Color Balance) внутри созданной группы, чтобы совместить веточки бамбука с существующим задним фоном, добавив к ним немного жёлтых тонов. Примечание переводчика: 1. Не забудьте поменять режим наложения для созданной группы на Нормальный (Normal), непрозрачность слоя 100%. 2. В настройках корректирующего слоя, поменяйте значения для Средних тонов (Midtones).

Шаг 5

Возвращаемся к слою с моделью. Перейдите на маску слоя с моделью и с помощью мягкой чёрной кисти очень маленького размера 2-3рх, скройте пространство между веточками на голове.

5. Добавляем Немного Света

Шаг 1

Добавьте новый слой поверх всех остальных слоёв и с помощью мягкой круглой кисти, цвет кисти #251601, прокрасьте область модели. Поменяйте режим наложения для данного слоя на Линейный осветлитель (Linear Dodge), непрозрачность слоя 100%.

Шаг 2

Дважды продублируйте этот слой, чтобы усилить световой эффект. С помощью слой-маски, скройте световой эффект поверх лица модели, чтобы сфокусировать световой эффект только в области шляпы.

Шаг 3

Добавьте новый слой поверх всех остальных слоёв и с помощью мягкой круглой кисти, цвет кисти #d9ab53, прокрасьте цветок лотоса на плече модели. Поменяйте режим наложения для данного слоя на Перекрытие (Overlay ), непрозрачность слоя 100%.

6. Добавляем Эффект Брызг

Шаг 1

Откройте исходное изображение с брызгами 1. Данное изображение имеет прозрачный задний фон, поэтому, просто переместите данное изображение на наш рабочий документ с помощью инструмента Перемещение (Move Tool), расположив его ниже лотоса за моделью.

Шаг 2

Дважды щёлкните по слою с брызгами, чтобы применить стиль слоя Наложение цвета (Color Overlay), установите цвет наложения на зелёный оттенок, схожий с оттенком веточек бамбука (#8aa013).

Шаг 3

Дважды продублируйте слой с брызгами, уменьшите масштаб дубликатов слоёв. Сместите немного вправо один из дубликатов слоёв, а второй дубликат слоя с брызгами, расположите на плече. Добавьте слой-маску к каждому дубликату слоя с брызгами, чтобы смягчить линии брызг.

Шаг 4

Добавьте ещё одно исходное изображение с брызгами, расположив их ниже большого цветка лотоса за моделью. Дважды щёлкните по данному слою с брызгами, чтобы применить стиль слоя Наложение цвета (Color Overlay), установите цвет наложения #df7877 для соответствия цвета с лотосом.

Шаг 5

Ещё раз откройте исходное изображение с Акварелью 2. Расположите эти акварельные брызги поверх лотоса на спине модели. Уменьшите масштаб брызг, а затем поменяйте режим наложения на Затемнение (Darken), непрозрачность слоя 100%.

К данному слою с брызгами, добавьте слой-маску и с помощью мягкой чёрной кисти, скройте эффект брызг поверх большого цветка лотоса.

Шаг 6

Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски к слою с брызгами, чтобы поменять цвет брызг на зелёный.

Шаг 7

Добавьте корректирующий слой Кривые (Curves) в качестве обтравочной маски, чтобы ослабить видимость эффекта.

Шаг 8

Откройте исходное изображение с брызгами 2. Отделите брызги от белого заднего фона с помощью инструмента Волшебная палочка (Magic Wand Tool (W). После этого, с помощью инструмента Лассо (Lasso Tool (L), выделите правую часть брызг (или любую часть изображения брызг, которая вам нравится - на своё усмотрение), а затем добавьте выделенную часть брызг на наш рабочий документ. Продублируйте слой с брызгами, сместив дубликат слоя в область головы модели, а оригинальный слой с брызгами в области спины модели.

Теперь выделите верхнюю часть брызг на исходном изображении с брызгами, поместив эти брызги в области шляпы.

Шаг 9

Добавьте слой-маску к каждому слою с брызгами, а затем проведите коррекцию, скрыв лишние участки брызг.

Шаг 10

Сгруппируйте слои с брызгами 2, поменяйте цвет брызг на зелёный с помощью корректирующего слоя Цветовой тон / Насыщенность (Hue/Saturation). Примечание переводчика: 1. не забудьте поменять режим наложения для созданной группы на Нормальный (Normal), непрозрачность слоя 100% 2. добавьте данный корректирующий слой внутри созданной группы.

Шаг 11

Создайте новый слой поверх всех остальных слоёв. Поменяйте цвет переднего плана на #809100. Далее, с помощью кисти Акварель, прокрасьте верхнюю часть головы модели.

Добавьте слой-маску к данному слою с акварельным пятном, а затем с помощью всё той же самой кисти Акварель небольшого размера, создайте определённый эффект. Примечание переводчика: в этом шаге просто поэкспериментируйте с эффектом.

7. Добавляем Текстуру Боке

Шаг 1

Поместите текстуру Боке поверх цветков лотоса на спине модели. Поменяйте режим наложения для слоя с текстурой Боке на Перекрытие (Overlay) непрозрачность слоя 100% и с помощью слой-маски, скройте жёсткие края текстуры, создав видимый эффект Боке поверх цветков лотоса.

Шаг 2

Выделите различные участки текстуры Боке, чтобы добавить их поверх области шляпы, головы и спины модели. Добавьте слой-маску к каждому слою с фрагментом текстуры Боке, а затем обработайте их, чтобы показать эффект Боке и добавить световые эффекты.

Шаг 3

Сгруппируйте слои с фрагментами текстуры Боке в одну группу. Поменяйте режим наложения для созданного слоя с группой на Перекрытие (Overlay) непрозрачность слоя 100%. Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation). Уменьшите значение Цветового тона (Hue) до -13:

Шаг 4

Добавьте корректирующий слой Кривые (Curves) для усиления эффекта и яркости.

8. Проводим Заключительную Цветовую Коррекцию

Шаг 1

Добавьте корректирующий слой Карта градиента (Gradient Map) поверх всех остальных слоёв. Установите цвета градиента #e10019 и #ae730c. Уменьшите Непрозрачность (Opacity) данногокорректирующего слоя до 20%.

Шаг 2

Далее, добавьте корректирующий слой Цветовой Баланс (Color Balance). Поменяйте настройки для Средних тонов (Midtones).

Шаг 3

Добавьте корректирующий слой Кривые (Curves), чтобы усилить контраст всей сцены.
