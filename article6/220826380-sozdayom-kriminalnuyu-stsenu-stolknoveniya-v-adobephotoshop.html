1. Создаём Основу Сцены

Шаг 1

Создайтеновый документ 2000 x 1231 px в Photoshop со следующиминастройками, которые указаны ниже:

Шаг 2

Откройтеисходное изображение Улица 1. Переместите данное изображение на наш рабочийдокумент с помощью инструмента Перемещение (Move Tool (V). Расположите данное изображение в левой части холста.

Шаг 3

ИдёмФильтр – Размытие – Размытие по Гауссу (Filter > Blur > Gaussian Blur). Установите Радиус (Radius)размытия на 7 px. Данный шаг направлен на усиление глубины резкостивсей сцены.

Шаг 4

Идём Слой– Новый корректирующий слой – Кривые (Layer > New Adjustment Layer > Curves), а затем преобразуйте данный корректирующий слой вобтравочную маску. Потяните кривую вниз, чтобы уменьшить яркость. Далее,перейдите на маску данного корректирующего слоя, выберите инструмент Кисть (Brush Tool (B), установите мягкую круглую кисть, цвет кисти чёрный. С помощьюданной кисти, прокрасьте центральную часть улицы, в частности землю, а такжедетали, куда падает свет, чтобы скрыть коррекцию кривых.

Шаг 5

Нам нужноослабить свет фонарей, для этого, создайте новый слой и с помощью мягкой кисти,цвет кисти #625448, Непрозрачность (Opacity)кисти примерно 30-35%, прокрасьте поверх фонарей.

Шаг 6

Создайтеновый слой, поменяйте режим наложения для данного слоя на Перекрытие (Overlay),непрозрачность 100%, а затем выполните заливку 50% серым (fill with 50% gray).

Выберитеинструмент Осветлитель (Dodge Tool (O), установите Диапазон (Range) на Средниетона (Midtones), а значение Экспозиции (Exposure)примерно на 10-15%, а затем добавьте световые блики на отдельные домов нижефонарей. Вы можете увидеть результат моей обработки на скриншоте ниже при режимеНормальный (Normal) и результат при режиме Перекрытие (Overlay):

Шаг 7

Откройтеисходное изображение Улица 2. Переместите данное изображение на наш рабочийдокумент с помощью инструмента Перемещение (Move Tool (V). Расположите данное изображение ниже слоя с Улицей1. Расположите изображение таким образом, чтобы часть с освещением быларасположена в верхней правой части холста, как показано на скриншоте ниже.

Шаг 8

Применитефильтр Размытие по Гауссу (Gaussian Bur) к данному слою с улицей, чтобы слегка смягчить изображение.Установите радиус размытия на 4 px.

2. Ретушируем Мужчину на Переднем Плане

Шаг 1

Переместитеизображение Мужчины 1 на наш рабочий документ, а затем нажмите второй значок внижней части панели слоёв, чтобы добавить слой-маску к данному слою. С помощью кистисредней мягкости, цвет кисти чёрный, удалите тёмный задний фон, чтобы быливидны улицы.

Шаг 2

У слоюс мужчиной 1, добавьте корректирующий слой Кривые (Curves) в качествеобтравочной маски, чтобы уменьшить световые блики на изображении мужчины, вчастности на щеках и рубашке. Далее, перейдите на маску данного корректирующегослоя и с помощью мягкой чёрной кисти, удалите эффект затемнения на участкахтени или на любых деталях, которые на ваш взгляд, слишком затемнены.

Шаг 3

Добавьтеещё один корректирующий слой Кривые (Curves) в качествеобтравочной маски к слою с мужчиной, чтобы добавить больше света на участкитени. Далее, перейдите на маску данного корректирующего слоя и с помощью мягкойчёрной кисти, прокрасьте участки светлых бликов, чтобы скрыть воздействиеданного корректирующего слоя.

Шаг 4

Создайте новый слой, поменяйте режим наложения для данногослоя на Перекрытие (Overlay),непрозрачность 100%, а затем выполните заливку 50% серым (fill with 50% gray).Выберите инструмент Осветлитель / Затемнитель (Dodge and Burn Tool (O) для коррекции светотеней на изображении мужчины, вчастности для усиления экспрессии в его глазах. Примечание переводчика:используйте те же самые настройки для инструмент Осветлитель / Затемнитель (Dodge and Burn Tool (O), что и в Шаге 6 или поиграйте с настройкамиинструмента.

Шаг 5

Создайтеновый слой и с помощью кисти средней мягкости маленького диаметра, цвет кистибелый, добавьте блик на нижнюю часть глаза (который расположен в области тени). Уменьшите Непрозрачность (Opacity) данногослоя до 50%, а затем с помощью слой-маски, создайте затенение блика с глазом.

Шаг 6

Создайтеновый слой и с помощью жёсткий кисти маленького диаметра, цвет кисти #9e9e9e, дорисуйтебородку мужчине.

3. Добавляем Полицейского

Шаг 1

Откройтеисходное изображение с Мужчиной 2, отделите изображение полицейского от заднегофона, а затем добавьте его на наш рабочий документ, расположив на улице назаднем фоне. Расположите слой с изображением полицейского ниже слоя с мужчиной1, а затем примените фильтр Размытие по Гауссу (Gaussian Blur) с размытием 7 px,чтобы совместить его с размытием и мягкостью изображения улицы.

Шаг 2

Чтобы добавитьтень к полицейскому, создайте новый слой ниже слоя с полицейским. Удерживая клавишу(Ctrl), щёлкните по миниатюре слоя с полицейским, чтобызагрузить активное выделение.

Залейтевыделенную область цветовым оттенком #0f0f0f, азатем отразите по вертикали, для этого идём Редактирование – Трансформация –Отразить по вертикали (Edit > Transform > Flip Vertical). С помощью инструмента Свободная Трансформация (Free Transform Tool (Ctrl+T), увеличьте верхнюю часть тени, а также слегка поменяйтенаправление тени в соответствии с направлением света от фонарей.

Шаг 3

К слоюс тенью, добавьте фильтр Размытие по Гауссу (Gaussian Blur) с радиусом размытия 7 px, а затемдобавьте слой-маску, чтобы ослабить видимость верхней части тени.

Шаг 4

Создайте новый слой поверх слоя с полицейским, поменяйте режимналожения для данного слоя на Перекрытие (Overlay), непрозрачность 100%, а затем выполните заливку 50%серым (fill with50% gray).С помощью инструмента Осветлитель (Dodge Tool (O), добавьте световые блики на отдельные детали вдольконтура изображения мужчины, а затем с помощью инструмента Затемнитель (Burn Tool), усильте тени на оружии, а также на брючине однойноги.

4. Рисуем Дождь и Капли Дождя

Шаг 1

Создайтеновый слой поверх всех остальных слоёв. Поменяйте цвет переднего плана на белыйцвет, а затем с помощью кисти Дождь, нарисуйте эффект дождя. Выберите кисть Дождьсо средней интенсивностью, чтобы добавить эффект дождя на отдельные участкисцены. После этого, преобразуйте данный слой в смарт-объект. Вы можете увидетьмой результат на скриншоте ниже:

Шаг 2

К слоюс эффектом дождя, примените фильтр Размытие по Гауссу (Gaussian Blur) с радиусом размытия 5 px. Далее,перейдите на маску фильтра и с помощью мягкой чёрной кисти, восстановите эффектдождя на мужчине, а также на некоторых участках улицы, сохраняя размытый дождя назаднем плане для соответствия мягкости этой области сцены.

Шаг 3

Создайтеновый слой и помощью кисти средней мягкости и маленького диаметра, цвет кисти белый,добавьте капли дождя на лицо мужчины. Блики на каплях должны повторятьнаправление света на заднем плане.

5. Проводим Заключительную Цветовую Коррекцию

Шаг 1

Добавьтекорректирующий слой Карта градиента (Gradient Map) поверх всех остальных слоёв. Установите следующие цвета градиента: #1f1203 и #2c85e9.

Шаг 2

Добавьтекорректирующий слой Цветовой Баланс (Color Balance). Установите настройкидля Средних тонов (Midtones) и Света (Highlights),которые указаны ниже:

Шаг 3

Добавьтекорректирующий слой Кривые (Curves) для усиления контрастности всей сцены.

Шаг 4

Чтобы изменитьосвещение и цвет улицы на заднем плане, используйте инструмент Лассо (Lasso Tool (L), чтобы выделить эту область, установите радиус растушёвки(feather) для данного выделения на 50:

Далее,идём Слой – Новый корректирующий слой – Микширование каналов (Layer > New Adjustment Layer > Channel Mixer). Поменяйте значения для Красных (Red), Зелёных(Green) и Синих (Blue)тонов:

Шаг 5

Загрузитемаску слоя Микширование каналов (Channel Mixer), для этого, удерживая клавишу (Ctrl), щёлкнитепо миниатюре маски. Создайте корректирующий слой Цветовой Баланс (Color Balance) к созданному выделению, чтобы усилить золотисто-жёлтыйсвет.

Шаг 6

Добавьтекорректирующий слой Микширование каналов (Channel Mixer). Поменяйте значения для Синих (Blue) тонов,чтобы изменить синие тона справа на зелёные.

Шаг 7

Создайтеновый слой и с помощью мягкой кисти, цвет кисти #140908, затемните левый крайсцены. Уменьшите Непрозрачность (Opacity) данного слоя до 50%, чтобы эффект выглядел болеенежным.

Шаг 8

Добавьтеещё один корректирующий слой Кривые (Curves), чтобыслегка затемнить и изменить цвета всей сцены. Далее, перейдите на маску данногокорректирующего слоя и с помощью мягкой чёрной кисти, прокрасьте поверхизображение мужчины 1, чтобы скрыть воздействие данного корректирующего слоя.
