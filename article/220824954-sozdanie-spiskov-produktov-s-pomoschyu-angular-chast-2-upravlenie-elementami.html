В первой части этой серии уроков по Angular вы видели, как начать работу с созданием менеджера списка продуктов, используя Angular. Вы научились создавать представление для компонента Grocery, добавлять элементы в список продуктов и отображать добавленные элементы.

В этой части вы будете реализовывать функционал для того, чтобы отметить элемент, как выполненный; редактирования существующих элементов и удаления добавленных элементов.

Поехали

Давайте начнём с клонирования исходного кода из первой части наших уроков. В вашем терминале напишите следующий код для клонирования исходного кода:

git clone https://github.com/royagasthyan/AngularGrocery

Как только исходный код скопирован, перейдите в папку проекта и установите необходимые зависимости.

cd AngularGrocerynpm install

После установки зависимостей, у вас должно получиться запустить сервер. Из папки проекта, наберите следующие команды:

ng serve

Откройте в браузере адрес http://localhost:4200/ и вы приложение должно запуститься

Обновление элементов продуктов

Как только вы добавили продуктовые элементы в список, у вас должна быть возможность их редактировать и обновлять. Давайте предоставим кнопку для редактирования в перечне, которая, при клике, будет включать редактирование существующего элемента.

Измените код app.grocery.html, для добавления кнопки редактирования.

<span class="glyphicon glyphicon-pencil floatright"></span>

Сохраните изменения выше и перезапустите сервер. Загрузите страницу, введите несколько значений и вы увидите кнопку редактирования для каждого элемента.

Когда пользователь нажимает кнопку редактирования, вам нужно добавить на неё метод для обработки клика. Измените app.grocery.html, чтобы добавить событие по клику, для редактирования элемента.

<span (click)="onEdit(task)" class="glyphicon glyphicon-pencil floatright"></span>

Передайте task в метод onEdit, как показано в приведенном выше коде, чтобы идентифицировать элемент для редактирования.

Внутри класса GroceryComponent инициализируйте переменную task, как показано:

task = {    name: "",  id: 0  };

В методе onClick, вы проверяете id, чтобы понять, существующий ли это элемент или новый. Измените метод onClick, как показано:

if(this.task.id == 0){    this.tasks.push({id: (new Date()).getTime(),name: this.task.name});}

Как видно, вы назначили уникальную временную метку в качестве id для каждой задачи. Теперь давайте определим метод onEdit для редактирования существующего элемента. Внутри метода onEdit назначьте переданный item (элемент) в переменную task.

onEdit(item){    this.task = item;}

Сохраните изменения и перезапустите сервер. Введите новый элемент в список продуктов и нажмите соответствующую кнопку редактирования. Вы сможете отредактировать и обновить запись, нажав кнопку ОК.

Удаление элементов

Давайте добавим значок удаления, для удаления существующих элементов. Обновите файл app.grocery.html, изменив HTML код, как показано ниже:

<span class="glyphicon glyphicon-remove floatright paddingLeft"></span>

Ну вот как выглядит готовый файл app.grocery.html:

<div>    <h2> Grocery list manager </h2>    <div class="row">        <div class="col-xs-6 col-md-4">            <div class="input-group">                <input type="text" class="form-control" placeholder="enter grocery items" [(ngModel)]="task.name" />                <div class="input-group-btn">                    <button class="btn btn-primary" type="button" (click)="onClick()">    <span class="glyphicon glyphicon-ok"></span></button                    >                </div>            </div>            <ul class="list-group">                <li *ngFor="let task of tasks" class="list-group-item">                    <span> {{ task.name }} </span>                    <span class="glyphicon glyphicon-remove floatright paddingLeft"></span>                    <span (click)="onEdit(task)" class="glyphicon glyphicon-pencil floatright"></span>                </li>            </ul>        </div>    </div></div>

Добавьте события по клику значка удаления, чтобы удалить элемент продукта.

<span (click)="onDelete(task)" class="glyphicon glyphicon-remove floatright paddingLeft"></span>

Сохраните изменения и перезапустите сервер. Попробуйте добавить новый элемент в приложение менеджера продуктов и вы увидите список элементов со значками удалить и изменить.

Чтобы реализовать функционал удаления, вам нужно добавить метод onDelete в файл app.grocery.ts, как показано:

onDelete(item){    // Delete functionality will be here}

Как только пользователь нажимаю на значок удаления, вам нужно проверить id элемента в списке продуктов и удалить запись из списка tasks. Вот как выглядит метод onDelete:

onDelete(item){    for(var i = 0;i < this.tasks.length; i++){    if(item.id == this.tasks[i].id){    this.tasks.splice(i,1);    break;    }    }}

Как видно из кода выше, вы должны пройтись по списку tasks и проверить если в нём id нужного элемента. Если находится соответствие элемента в списке tasks, то он удаляется с использованием метода splice.

Сохраните изменения, сделанные выше и перезапустите сервер. Добавьте несколько элементов в менеджер списка покупок. Они добавятся со значками удаления и редактирования. Попробуйте нажать на значок удаления и элемент будет удалён из списка.

Отмечаем элемент товара выполненным

Давайте добавим функциональность для зачеркивания элементов, добавленых в список. Как только пользователь выполнил задачу в списке менеджера продуктов, должна быть возможность зачеркнуть выполненный пункт. Чтобы отслеживать новые и завершенные задачи, добавьте новую переменную strike в данные задач.

Измените метод onClick так, чтобы добавить новую переменную strike, как показано ниже:

  onClick(){    if(this.task.id == 0){this.tasks.push({id: (new Date()).getTime(),name: this.task.name, strike: false});}  this.task = {name: "",id: 0};  }

Добавьте новый класс, названный strike в файл src/style.css, с помощью которого будут перечёркиваться элементы.

.strike{    text-decoration:line-through;}

Добавьте события по клику на элемент, для переключения переменной strike. Вы будете применять класс strike, на основе булевого значения переменной strike. По умолчанию, значение будет ложным. Вот метод onStrike для переключения значения переменной strike:

onStrike(item){  for(var i = 0;i < this.tasks.length; i++){    if(item.id == this.tasks[i].id){      if(this.tasks[i].strike){        this.tasks[i].strike = false;      }      else{        this.tasks[i].strike = true;      }      break;    }  }}

Как видно из метода выше, мы проходимся по списку элементов. Как только элемент будет найден, переключаем значение strike.

Основываясь на переменной strike вам нужно применить класс strike к span"у с названием задачи. Вот как это выглядит:

<span (click)="onStrike(task)" [ngClass]="{"strike": task.strike}"> {{ task.name }} </span>

Как видно, мы использовали директиву ngClass, чтобы применить класс strike к элементу span, если значение task.strike равно true.

Сохраните изменения и перезапустите сервер. Добавьте элементы в список и нажмите на добавленный элемент. После нажатия, элемент будет перечёркнут, как и ожидалось.
