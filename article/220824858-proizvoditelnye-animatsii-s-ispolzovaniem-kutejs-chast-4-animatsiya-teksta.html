KUTE.js также имеет текстовый плагин, который позволяет анимировать текст внутри разных элементов, либо путем увеличения, либо уменьшения числа, как в обратном отсчете, или путем записи строки символ за символом.

В этом уроке вы узнаете, как анимировать текст внутри разных элементов на веб-странице, используя CSS и плагины Text в KUTE.js.

Анимация свойств текста CSS

Как я упоминал ранее, вы можете использовать плагин KUTE.js CSS для анимации четырех различных свойств CSS, связанных с текстом. Этими свойствами являются font-size, line-height, letter-spacing, и word-spacing. Мы также будем использовать некоторые свойства основного ядра, обсуждаемые в первом учебнике, для анимации отдельных букв. Давайте посмотрим, как мы можем использовать все эти концепции вместе, чтобы создать вибрирующий текст HELLO в следующей демонстрации.

Вот код, который использовался для создания вышеупомянутой анимации:

var theLetters = document.querySelectorAll("span");var h = document.querySelector(".h");var e = document.querySelector(".e");var la = document.querySelector(".la");var lb = document.querySelector(".lb");var o = document.querySelector(".o");var startButton = document.querySelector(".start");var animateColor = KUTE.allFromTo(  theLetters,  { color: "white" },  { color: "red" },  { offset: 200, duration: 50});var animateFontSize = KUTE.allFromTo(  theLetters,  { fontSize: "2em" },  { fontSize: "4em" },  { offset: 100, duration: 200, repeat: 10, yoyo: true});var animateSkewing = KUTE.allTo(  theLetters,  { skewX: -15},  { offset: 200, duration: 200 });var animateH = KUTE.to(  h,  { color: "#009688" });var animateE = KUTE.to(  e,  { translateY: -40, color: "#E91E63" });var animateLA = KUTE.to(  la,  { color: "#8BC34A" });var animateLB = KUTE.to(  lb,  { translateY: 20, color: "#FFC107" });var animateO = KUTE.to(  o,  { color: "#FF5722" });var lettersSqueezed = KUTE.allTo(  theLetters,  { letterSpacing: "-15px" },  { offset: 0, duration: 200 });animateColor.chain(animateFontSize);animateFontSize.chain(animateSkewing);animateSkewing.chain(animateH, animateE, animateLA, animateLB, animateO);animateE.chain(lettersSqueezed);startButton.addEventListener(  "click",  function() {    animateColor.start();  },  false);

Каждая буква слова обернута внутри тега span и имеет свой собственный уникальный класс. Первая tween анимирует цвета всех букв от белого до красного со смещением 200 мс. Это первая анимация, которая воспроизводится после нажатия кнопки Start Animation. Анимация animateFontSize была связана с animateColor. Таким образом, анимация font-size начинается, как только заканчивается цветовая анимация.

Возможно, вы заметили, что я использовал два атрибута repeat и yoyo для управления поведением анимации. Атрибут yoyo используется для изменения анимации, которая в данный момент воспроизводится повторно. Это позволяет избежать внезапных скачков значений различных свойств во время анимации и сделать их гладкими.

Анимация font-size была скопирована с помощью animateSkewing, которая искажает все буквы на -15 градусов. Свойства skewX и skewY доступны в самом ядре движка.

Все tweens для анимации цвета разных букв были привязаны к animateSkewing. Таким образом, вы можете убедиться, что все связанные цветовые анимации начнут воспроизводиться, как только анимация наклона закончится. Наконец, lettersShqued tween уменьшают расстояние между разными буквами на 15 пикселей.

Вы можете создавать более интересные эффекты, используя различные комбинации свойств.

Анимация цифр

Вы также можете анимировать числа в KUTE.js. Тем не менее, вам нужно будет добавить дополнительный текстовый плагин для создания анимации.

Процесс анимации цифр на самом деле очень прост. Вам просто нужно указать селектор, где должны отображаться анимированные цифры, а также последняя цифра, на которой должна заканчиваться анимация.

Вот пример, показывающий общее количество аэропортов в США в 2016 году с использованием анимации.

var usa = document.querySelector(".usa");var startButton = document.querySelector(".start");var animateUSA = KUTE.to(  usa,  { number: 19536 });startButton.addEventListener(  "click",  function() {    animateUSA.start();  },  false);

Написание текста символ за символом

Это очень популярный эффект, который можно найти на нескольких веб-сайтах. Текстовый плагин KUTE.js позволяет вам указать новое предложение, которое должно заменить исходное предложение по одному символу за раз.

Перед заменой исходных символов на их окончательное значение, случайные символы анимируются, как в примере с цифрами, который вы только что видели. Встроенная демо-версия CodePen должна сделать ее более понятной:

Вот код, который вам нужно написать, чтобы создать вышеупомянутую анимацию:

var animateHeading = KUTE.to(  heading,  { text: "70% Surface of Earth is Covered with Water." },  { duration: 5000});startButton.addEventListener(  "click",  function() {    animateHeading.start();  },  false);

Анимация символов для всего предложения заканчивается в течение 5 секунд. Как вы могли заметить, начальное и последнее предложения не обязательно должны содержать одинаковое количество символов. Это дает нам большую свободу при установке значения параметра text.

Вы также можете включить HTML-теги внутри значения параметра text, а затем использовать CSS для изменения внешнего вида текста, который вы только что анимировали.

var animateHeading = KUTE.to(  heading,  { text: "70% SURFACE OF <span class="earth">EARTH</span> IS COVERED WITH <span class="water">WATER</span>." },  { duration: 10000, textChars: "upper" });

Будет задержка в появлении Earth после того как of уже появилось. Это происходит потому, что плагин также пишет <span class="earth"> , используя ту же анимацию символов, но ни один из этих символов фактически не виден пользователю. Задержка может быть желательной или нежелательной в зависимости от ваших предпочтений.

Промежуточные символы, отображаемые во время анимации, по умолчанию являются строчными буквенными значениями. Это может быть проблемой, когда символы, которые вы хотите анимировать, все прописные буквы или цифры. Какие промежуточные символы используются для анимации, определяется значением параметра textChars. Он принимает шесть различных значений:

  • alpha: в этом случае промежуточные символы будут строчными буквами.
  • upper: в этом случае промежуточными символами будут заглавные буквы.
  • numeric: в этом случае для анимации используются числовые символы. Это отличается от анимации числа, так как значения не будут увеличиваться последовательно.
  • symbols: в этом случае плагин будет использовать для анимации такие символы, как #, % и $.
  • all: это значение можно использовать, если промежуточные символы должны состоять из букв, цифр и символов.
  • Если для вас ничего не подходит, KUTE.js дает вам возможность указать ваш собственный список символов, который должен использоваться во время анимации.
