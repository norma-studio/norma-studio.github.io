Приложения на Angular делается из компонентов. В приложении на Angular, компоненты состоят из HTML шаблона и класса компонента. Из официальный документов:

Компоненты — это самый базовый строительный блок пользовательского интерфейса в приложении Angular. Angular приложение представляет собой дерево компонентов Angular. Компоненты Angular являются подмножеством директив. В отличие от директив, у компонентов всегда есть шаблон и только один компонент может быть создан для элемента в шаблоне.

На этом уроке вы узнаете, как начать работу по созданию менеджера списка покупок с помощью Angular.

Приступим

Вам понадобится Node версии > 6.9.x и Node Package Manager (npm) > 3.x.x. После их установки, попробуйте установить Angular CLI используя npm.

npm install -g @angular/cli

После установки Angular CLI, попробуйте так же создать проект.

ng new AngularGrocery --skip-npm

Команда выше создаст структуру проекта. Перейдите к папке проекта и установите необходимые зависимости.

cd AngularGrocerynpm install

Чтобы запустить веб-сервер приложения, введите следующую команду:

ng serve

Как только сервер приложения запустится, откройте в браузере http://localhost:4200/ и вы увидите приложение по умолчанию.

Структура приложения

Изображение выше показывает структура приложения для нашего менеджера списка покупок. Папка src содержит исходный код для базового приложения. Папка app внутри папки src содержит код TypeScript. index.html внутри папки src это главный HTML файл, в котором создаются компоненты Angular.

Создание представления для добавления элементов

По умолчанию приложение имеет компонент app-root. Давайте создадим новый компонент под названием app-grocery для создания нашего приложения для менеджера продуктов. В папке приложения создайте файл app.grocery.ts.

Импортируйте модуль Component из angular-core.

import { Component } from "@angular/core";

Вы будете использовать Bootstrap, для оформления компонента. Загрузите и добавьте пакет Bootstrap с официального сайта. Положите файлы в папку src/assets.

Давайте определим компоненты в файле app.grocery.ts. У компонента будет три свойства: selector, template и styleUrls. selector символизирует о способе, которым используется компонент. template определяет HTML, который будет выдаваться, при использовани определённого селектора. styleUrls определяет URL стилей, используемых в компоненте.

В app.grocery.ts, после импорта модуля компонента, определяем компонент app-grocery, как показано:

@Component({    selector: "app-grocery",  templateUrl: "./app.grocery.html",styleUrls: [        "../assets/css/bootstrap.min.css"    ]})

Как видим в примере кода выше, компонент использует шаблон, названный app.grocery.html. Создайте файл app.grocery.html в папаке src/app. Добавьте следующий код в файл app.grocery.html:

<div>    <h2> Grocery list manager </h2>    <div class="row">        <div class="col-xs-6 col-md-4">            <div class="input-group">                <input type="text" class="form-control" placeholder="enter grocery items" id="txtSearch" />                <div class="input-group-btn">                    <button class="btn btn-primary" type="button">        <span class="glyphicon glyphicon-ok"></span>        </button>                </div>            </div>        </div>    </div></div>

В файле app.grocery.ts, добавьте класс GroceryComponentclass для экспорта.

export class GroceryComponent {}

Вам нужно импортировать, только что созданный компонент в app.module.ts, прежде чем вы сможете его использовать. Испортируйте GroceryComponent в app.module.ts.

import { GroceryComponent } from "./app.grocery";

Объявите (declare) и загрузите (bootstrap) вновь созданный компонент Grocery в файле app.module.ts.

@NgModule({  declarations: [    GroceryComponent  ],  imports: [    BrowserModule,    FormsModule,    HttpModule  ],  providers: [],  bootstrap: [GroceryComponent]})

Теперь вы готовы использовать компонент Grocery на странице index.html. Удалите компонент по умолчанию со страницы index.html и добавьте следующий код HTML:

<app-grocery>Loading...</app-grocery>

Сохраните указанные выше изменения и перезапустите сервер. Перейдите в браузере по адресу http://localhost:4200/ и вы увидите компонент Grocery.

Добавление элементов

После ввода элемента в текстовое поле ввода, вам необходимо прочитать элемент и сохранить его в список. Чтобы прочитать ввод, вы будете использовать ngModel, который привяжет его к переменной. Добавьте ngModel в текстовое поле ввода.

<input type="text" class="form-control" placeholder="enter grocery items" id="txtSearch" [(ngModel)]="task" />

Каждый раз, когда какой-то текст вводится в текстовое поле, переменная task обновляется соответственно. Определите переменную task в GroceryComponent в файле app.grocery.ts. Добавьте переменную, названную tasks, для сбора введенных задач.

  task: string;  tasks = [];

При нажатии на кнопку OK, задача будет добавлена в список колекции tasks, который вы определили в GroceryComponent. Определите метод onClick в GroceryComponent для обработки клика по кнопке OK. Вот как это выглядит: 

  onClick(){      this.tasks.push({name: this.task});  this.task = "";  }

И событие клика по кнопке OK, как показано:

<button class="btn btn-primary" type="button" (click)="onClick()">   <span class="glyphicon glyphicon-ok"></span></button>

После нажатия кнопки OK, новая задача добавляется в список коллекции tasks. Переменная task сбрасывается для очистки поля ввода.

Сохраните изменения сделанные выше, а при вводе задачи и нажатии кнопки ОК, задача добавляется в коллекцию tasks. Чтобы отобразить коллекцию задач, добавьте span app.grocery.html.

<span>    {{tasks | json}}</span>

Введите задачу в поле ввода и нажмите кнопку OK. На вашей странице отобразится переменная tasks в формате JSON.

Вот весь файл app.grocery.ts:

import { Component } from "@angular/core";@Component({    selector: "app-grocery",templateUrl: "./app.grocery.html",styleUrls: [        "../assets/css/bootstrap.min.css"    ]})export class GroceryComponent {  task: string;  tasks = [];  onClick(){  this.tasks.push({name: this.task});  this.task = "";  }}

Отображение добавленных элементов

Теперь, так как у нас есть добавленные элементы в переменную tasks, вы можете использовать её для отображения задач. Вы будете использовать директиву NgFor для проходу по коллекции tasks и динамического создания HTML для отображения задач. Вы будете отображать задачу внутри элемента UL и повторяющихся LI, используя директиву NgFor. Вот как это выглядит:

<ul class="list-group">    <li *ngFor="let task of tasks" class="list-group-item">        <span> {{ task.name }} </span>    </li></ul>

Как видно их кода выше, мы проходим по переменной tasks и динамически создаём элементы LI и span. Вот как выглядит готовый файл шаблона app.grocery.html:

<div>    <h2> Grocery list manager </h2>    <div class="row">        <div class="col-xs-6 col-md-4">            <div class="input-group">                <input type="text" class="form-control" placeholder="enter grocery items" [(ngModel)]="task" />                <div class="input-group-btn">                    <button class="btn btn-primary" type="button" (click)="onClick()">        <span class="glyphicon glyphicon-ok"></span>        </button>                </div>            </div>        </div>    </div>    <br />    <ul class="list-group">        <li *ngFor="let task of tasks" class="list-group-item">            <span> {{ task.name }} </span>        </li>    </ul></div>

Сохраните изменения сделанные выше и перезапустите сервер. Откройте в вашем браузере http://localhost:4200/, введите задачу и нажмите кнопку OK. Каждая добавленная задача отобразится ниже списком.
