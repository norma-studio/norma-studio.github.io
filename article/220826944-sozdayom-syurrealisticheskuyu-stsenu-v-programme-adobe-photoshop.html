Создаём Сюрреалистическую Сцену в программе Adobe Photoshop...

Design & Illustration Photo Manipulation
Tutorial

В этом уроке я покажу вам, как создать сюрреалистическую сцену с мужчиной с головой деревом.

Вначале мы создадим задний фон, используя исходные изображения с полем и небом, а затем добавим изображение модели и проведём небольшую ретушь данного изображения. Далее, мы добавим изображение дерева, облака и молнии и совместим эти элементы композиции со всей сценой с помощью корректирующих слоёв, работы с маской и кистями. После этого, мы нарисуем эффект дождя, включая капли дождя, а также добавим осенние листья к нашей сцене. В завершение, мы используем несколько корректирующих слоёв, чтобы усилить итоговый результат.

Исходные материалы

Следующие исходные материалы были использованы для создания данного урока:

  • Поле
  • Небо
  • Мужчина
  • Дерево
  • Молния
  • Облако
  • Листья

1. Добавляем Поле и Небо

Шаг 1

Создайте новый документ 2000рхх 1581рх в программе Photoshop. Установите настройки,которые указаны на скриншоте ниже.

Шаг 2

Откройте исходное изображение с полем в программе Photoshop. С помощью инструмента Прямоугольная область (Rectangular Marquee Tool (M), выделите только изображение поля. С помощью инструмента Перемещение (Move Tool (V), переместите выделенное изображение поля на наш рабочий документ:

Щёлкните по второму значку в нижней части панели слоёв, чтобы добавить слой-маску к слою с полем. С помощью чёрной кисти средней мягкости, прокрасьте линию горизонта изображения поля, чтобы смягчить край.

Шаг 3

Далее, к слою с полем, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски. Идём Слой- Новый корректирующий слой- Цветовой тон / Насыщенность (Layer > New Adjustment Layer > Hue/Saturation). Уменьшите значение Насыщенности (Saturation) до -12:

Шаг 4

Добавьте корректирующий слой Кривые (Curves), чтобы затемнить поле. Далее, перейдите на маску данного корректирующегослоя и с помощью мягкой чёрной кисти, скройте коррекцию кривых в указанной области, восстановив светлый участок на краях поля. Примечание переводчика: не забывайте преобразовывать корректирующие слои вобтравочные маски.

Шаг 5

Откройте исходное изображение с небом. Переместите данное изображение с небом на наш рабочий документ, расположив ниже слоя с полем.

2. Добавляем Модель

Шаг 1

Откройте исходное изображение с моделью. Отделите изображение тела мужчины от заднего фона.Переместите выделенное изображение тела на наш рабочий документ, расположив его в центре поля.

Шаг 2

К слою с телом мужчины, добавьте слой-маску и с помощью чёрной кисти средней жёсткости очень маленького диаметра, обработайте нижний край изображения ног, чтобы создать ощущение, как будто мужчина стоит в поле посреди травы. Примечание переводчика: в качестве кисти вы можете использовать стандартную кисть Трава, сделайте буквально пару кликов мышкой в области нижнего края изображения ног. Не забудьте перейти на маску слоя.

Шаг 3

Далее, идём Редактирование - Марионеточная деформация (Edit > Puppet Warp) для небольшой коррекции позы ног мужчины. Основная цель- это увеличить пространство между ног мужчины.

Шаг 4

Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation), чтобы изменить цвет изображения модели. Примечание переводчика: не забудьте преобразовать данный корректирующий слой в обтравочную маску к слою с телом мужчины.

Шаг 5

Создайте новый слой, поменяйте режим наложения на Перекрытие (Overlay), непрозрачность слоя 100%, а также выполните заливку 50% нейтральным серым (fill it with 50% gray):

Выберите инструмент Осветлитель и Затемнитель (Dodge and Burn Tool (O) в панели управления данногоинструмента, установите Диапазон (Range) на Средние тона (Midtones), значение Экспозиции (Exposure) примерно 20-30%. Теперь затемните плечи, а также добавьте детали на складки брюк.

3. Добавляем Дерево

Шаг 1

Откройте исходное изображение с деревом. С помощью инструмента Перемещение (Move Tool), переместите изображение дерева на наш рабочий документ. Расположите слой с деревом ниже слоя с телом мужчины.

Шаг 2

Теперь добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation), поменяйте настройки для опции Все (Master). Примечание переводчика: не забывайте преобразовывать корректирующие слои в обтравочные маски к слою с деревом.

Шаг 3

Добавьте корректирующий слой Цветовой Баланс (Color Balance). Поменяйте значения для тонового режима Средние тона (Midtones).

Шаг 4

Добавьте корректирующий слой Кривые (Curves), чтобы уменьшить контраст дерева. Далее, перейдите на маску данного корректирующего слояи с помощью мягкой чёрной кисти, скройте эффект кривых в верхней части изображения дерева - эта часть дерева должная быть светлее, чем остальная часть (т.к. на верхнюю часть дерева будет падать свет от молнии).

4. Добавляем Облако

Шаг 1

Откройте исходное изображение с облаком. С помощью инструмента Лассо (Lasso Tool), выделите нижнее правое облако. Установите радиус Растушёвки (feather) на 15. Переместите выделенное изображение облака на наш рабочий документ, расположив его над деревом.

К слою с облаком, добавьте слой-маску и аккуратно скройте вкрапления синего заднего фона с помощью чёрной кисти средней мягкости. Вы можете использовать планшетное перо, чтобы получить более точный результат.

Шаг 2

Светотени на изображении облака не соответствуют заднему фону. Чтобы исправить это, добавьте корректирующий слой Кривые (Curves), чтобы уменьшить яркость. Далее, перейдите на маску данного корректирующего слояи с помощью мягкой чёрной кисти, прокрасьте нижнюю правую часть облака, чтобы эффект коррекции был виден только в верхней левой части облака.

Шаг 3

Добавьте ещё один корректирующий слой Кривые (Curves), чтобы немного осветлить верхнюю правую часть облака. Далее, перейдите на маску данного корректирующего слояи с помощью мягкой чёрной кисти, прокрасьте левую часть облака, чтобы скрыть коррекцию кривых.

Шаг 4

Теперь добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation), чтобы обесцветить облако.

5. Добавляем Молнию

Шаг 1

Откройте исходное изображение с молнией. Переместите данное изображение на наш рабочий документ. Расположите молнию между облаком и деревом. Поменяйте режим наложения для слоя с молнией на Осветление (Screen), непрозрачность 100%:

С помощью слой-маски, удалите лишние участки изображения молнии, ограничив область молнии в пределах изображения облака.

Шаг 2

Добавьте корректирующий слой Уровни (Levels). Увеличьте значение тени. Данный шаг направлен на то, чтобы скрыть основные участки заднего фона на изображении молнии.

Шаг 3

Теперь добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation). Уменьшите значение Насыщенности (Saturation) до -100:

6. Создаём Эффект Дождя

Шаг 1

На данном этапе урока, мы создадим эффект дождя. Создайте новый слой поверх всех остальных слоёв. Установите цвет переднего плана на #ededed. Выберите жёсткую кисть, а затем нажмите клавишу (F5), чтобы установить настройки для данной кисти.

С помощью настроенной кисти, нарисуйте произвольные вертикальные линии в пределах пространства между облаком и деревом.

Отразите капли дождя по вертикали, для этого идём Редактирование - Трансформация - Отразить по вертикали (Edit > Transform > Flip Vertical):

Шаг 2

Далее, идём Фильтр - Размытие - Размытие в движении (Filter > Blur > Motion Blur).

С помощью слой-маски, скройте лишние фрагменты дождя за пределами области облака и дерева.

Шаг 3

Продублируйте слой с эффектом дождя несколько раз, а затем уменьшите значение Смещения (distance) фильтра Размытие в движении (Motion Blur filter) для дубликатов слоёв с 20, например, до -15, -12, -10, а также уменьшите масштаб. Таким образом, мы создадим небольшую глубину резкости и придадим дождю естественный вид.

Шаг 4

Создайте новый слой и с помощью всё той же самой кисти, нарисуйте капли дождя и брызги от капель дождя на дереве, а также на плечах мужчины. Варьируйте размер кисти, чтобы добиться наилучшего результата.

Шаг 5

Создайте новый слой, создайте несколько выделенных участков на рубашке мужчины с помощью инструмента Лассо (Lasso Tool), установите значение растушёвки (feather) на 2 px. Залейте выделенные участки цветовым оттенком #d2d1cb (отберите образец цветового оттенка с рубашки мужчины).

Поменяйте режим наложения для данного слоя с заливкой на Умножение (Multiply), непрозрачность слоя 100%, чтобы создать эффект промокшей рубашки.

Шаг 6

Создайте новый слой и с помощью мягкой кисти, цвет кисти #ededed, прокрасьте под облаком поверх области молнии, чтобы добавить свечение на эту часть изображения. Поменяйте режим наложения для данного слоя на Мягкий свет (Soft Light), непрозрачность 100%.

7. Добавляем Листья.

Шаг 1

Откройте исходное изображение с листьями. Выделите изображение листочка, а затем добавьте выделенное изображение на наш рабочий документ. Я расположил листочек в левой части сцены, а затем с помощью инструмента свободной трансформации (Ctrl+T) в режиме Деформации (Warp), деформировал листочек. как показано на скриншоте ниже.

Далее, идём Фильтр - Размытие- Размытие по Гауссу (Filter > Blur > Gaussian Blur). Установите Радиус (Radius) размытия на 8 pх.

Шаг 2

Добавьте ещё листьев к нашей сцене, используя ту же самую технику. Варьируйте размер листьев, а также радиус размытия, чтобы усилить глубину резкости.

Шаг 3

Сгруппируйте все слои с листьями в одну группу, а затем поменяйте режим наложения для слоя с группой на Нормальный (Normal), непрозрачность 100%. Теперь добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation), поменяйте настройки для опции Все (Master).

Шаг 4

Добавьте корректирующий слой Кривые (Curves), чтобы усилить яркость.

Вы можете увидеть результат после коррекции на маске корректирующего слоя кривые. Основная цель - это создать основной свет в центре, поэтому, вы знаете, как обработать листья кистью.

8. Заключительная Коррекция

Шаг 1

Добавьте корректирующий слой Цветовой Баланс (Color Balance). Поменяйте значения для тонового режима Средние тона (Midtones), а также Света (Highlights).

Шаг 2

Добавьте слой-заливку Цвет (Color Fill). Установите цвет заливки #150231 Поменяйте режим наложения для слоя с заливкой на Исключение (Exclusion), непрозрачность слоя 100%:

Шаг 3

Добавьте корректирующий слой Кривые (Curves), чтобы усилить контраст.

Шаг 4

Добавьте корректирующий слой Фото фильтр (Photo Filter). Установите следующий цветовой оттенок для данного фильтра #ec8a00:

Шаг 5

Добавьте ещё один корректирующий слой Кривые (Curves), чтобы дополнительно затемнить сцену. Далее, перейдите на маску данного корректирующегослоя и с помощью мягкой чёрной кисти, прокрасьте среднюю часть сцены, а также мужчину, чтобы сохранить эффекты освещения.

Шаг 6

Добавьте слой-заливку Цвет (Color Fill). Установите цвет заливки #01080d. Поменяйте режим наложения для слоя с заливкой на Вычитание (Subtract), непрозрачность слоя 100%:

Шаг 7

Продублируйте данный слой с заливкой, а затем поменяйте режим наложения для дубликата слоя на Нормальный (Normal), а также уменьшите непрочность слоя до 40%. С помощью слой-маски, прокрасьте мягкой чёрной кистью среднюю часть изображения, чтобы создать нежный эффект виньетки.
