В этом уроке вы узнаете, как создавать линейные диаграммы в Plotly. Я также написал еще одну серию в прошлом о легкой библиотеке под названием Chart.js, которая может быть использована для создания диаграмм на основе холста в JavaScript. В одном уроке, озаглавленном «Начало работы с диаграммами Chart.js: Line и Bar», я рассмотрел процесс создания линейных диаграмм с использованием Chart.js. Люди, которые прочитали этот учебник, вероятно, помнят, что вам нужно установить атрибут type в line, чтобы создать линейную диаграмму в Chart.js.

С Plotly.js вы не можете установить атрибут type в line для создания линейной диаграммы. Вам нужно будет установить атрибут type для scatter, а затем установить атрибут mode в «lines», «lines + markers» или «lines + markers + text». Помня об этом, давайте начнем с учебника и создадим несколько удивительных линейных диаграмм.

Создание базовой линейной диаграммы

В этом разделе мы построим два разных графика scatter в виде линейных диаграмм с использованием Plotly. Код очень похож на тот, который мы использовали в последнем уроке для создания линейной диаграммы с использованием одной трассы. Вот фактический код:

var lineDiv = document.getElementById("line-chart");var traceA = {  x: [1, 5, 13, 24, 35, 46, 60],  y: [80, 40, 70, 65, 15, 75, 49],  type: "scatter"};var traceB = {  x: [4, 9, 17, 21, 31, 42, 56],  y: [64, 81, 3, 49, 25, 17, 26],  type: "scatter"};var data = [traceA, traceB];var layout = {  title:"A Line Chart in Plotly"};Plotly.plot( lineDiv, data, layout );

Стилирование линий диаграммы с использованием разных атрибутов

На этом этапе все в нашем графике было создано с использованием параметров по умолчанию. Остальные разделы в учебнике покажут, как использовать все атрибуты, доступные для трассы типа scatter, для создания пользовательских линейных графиков. Варианты изменения внешнего вида построенной линии доступны по ключу line.

Цвет строки можно указать с помощью ключа color. Вы также можете управлять шириной линии, используя атрибут width. Ширина указана в пикселях, а по умолчанию - 2.

Форма линии между различными точками, которые строятся, может быть указана с использованием атрибута shape. Форма по умолчанию является linear, но вы также можете установить ее spline, vh, hv, hvh или vhv. Когда shape установлена в linear, в строке нет изгибов, соединяющих две последовательные точки. В случае vh, hv, hvh и vhv линии никогда не рисуются под углом. Они либо горизонтальные, либо вертикальные, с углом 90 градусов, которые могут возникать в первой точке, второй точке, средней точке или обеих точках в зависимости от заданного значения формы.

Последний вариант - установить shape в spline. В этом случае линия фактически становится кривой, без резких изгибов. Гладкость этой кривой можно задать с помощью другого атрибута, называемого smoothing. Этот атрибут может принимать любое значение от 0 до 1.3 включительно. Установка на нуль приведет к прямым линиям, таким как значение linear. Вот какой код, который использует все эти атрибуты для построения пяти разных строк на диаграмме:

var lineDiv = document.getElementById("line-chart");var traceLinear = {  x: [1, 5, 13, 24, 35, 46, 60],  y: [180, 160, 170, 165, 175, 175, 149],  type: "scatter",  name: "Linear Shape",  line: {    shape: "linear",    color: "rgb(255, 157, 98)"  }};var traceSpline = {  x: [1, 5, 13, 24, 35, 46, 60],  y: [150, 130, 140, 135, 145, 145, 119],  type: "scatter",  name: "Spline Shape",  line: {    shape: "spline",    color: "rgb(157, 255, 98)"  }};var traceVH = {  x: [1, 5, 13, 24, 35, 46, 60],  y: [120, 100, 110, 105, 115, 115, 89],  type: "scatter",  name: "VH Shape",  line: {    shape: "vh",    color: "rgb(157, 98, 255)"  }};var traceHVH = {  x: [1, 5, 13, 24, 35, 46, 60],  y: [90, 70, 80, 75, 85, 85, 59],  type: "scatter",  name: "HVH Shape",  line: {    shape: "hvh",    color: "rgb(98, 157, 255)"  }};var traceSplineB = {  x: [1, 5, 13, 24, 35, 46, 60],  y: [60, 40, 50, 45, 55, 55, 29],  type: "scatter",  name: "Spline Shape (1.3)",  line: {    shape: "spline",    smoothing: 1.3,    color: "rgb(255, 98, 157)"  }};var data = [traceLinear, traceSpline, traceVH, traceHVH, traceSplineB];var layout = {  title:"Different Shapes of a Line Chart",  yaxis: {    rangemode: "tozero"  }};Plotly.plot( lineDiv, data, layout );

Следующая линейная диаграмма показывает разницу между различными значениями атрибута shape. Я использовал параметр name для назначения разных имен каждой строке, чтобы вы могли четко видеть форму, созданную каждым значением.

Еще одним параметром, который можно использовать при рисовании сюжетных линий, является параметр dash. Вы можете установить этот параметр в строковое значение, чтобы установить стиль тире для ваших линий. Допустимыми значениями для этого атрибута являются: solid, dot, dash, longdash, dashdot и longdashdot.

Другим вариантом является предоставление самой длины штриха, используя список значений пикселей, таких как «4px, 4px, 10px». В следующей демонстрации используются разные значения свойства dash для создания уникальных стилей линий.

Настройка заливки цветов и маркеров

Площадь под построенной графикой по графику остается прозрачной по умолчанию, но вы можете заполнить ее цветом по вашему выбору, если хотите. Этого можно достичь, используя параметр fill. По умолчанию установлено значение none, но другие допустимые значения включают tozeroy, tozerox, tonexty, tonextx, toself и tonext.

Значение tozeroy заполнит всю область, начиная с трассы линии до y = 0. Аналогично, значение tozerox будет заполнять всю область, начиная с трассы линии до x = 0. Значения tonexty и tonextx заполняют всю область между конечными точками текущей трассировки и трассировкой перед ней. Когда других следов не осталось, эти значения будут действовать как tozeroy и tozerox соответственно. Эти два значения можно использовать для создания уложенных графов.

Как следует из названия, значение toitself соединяет конечные точки трассы друг с другом, образуя замкнутую форму. Последнее значение, tonext, заполняет все доступное пространство между двумя трассами только тогда, когда одна трасса полностью закрывает другую.

По умолчанию область, заданная значением параметра fill, заполняется с использованием полупрозрачного варианта цвета линии, цвета маркера или цвета линии маркера в зависимости от доступности. Однако вы можете указать свой собственный цвет, чтобы заполнить эту область, используя параметр fillcolor.

Точки, которые представляют нарисованные точки на графике, по умолчанию отмечены кругами. Plotly.js также предоставляет множество других символов маркера для вас. Существует около 44 различных символов маркера, и почти все они доступны в четырех разных версиях. Вы можете найти список всех этих символов в документации. Маркеры также имеют параметры opacity, size, и color, которые позволяют вам контролировать прозрачность, размер и цвет этих символов. Значение opacity по умолчанию равно 1, а значение по умолчанию для size - 6.

Вот небольшой код, который использует все параметры из этого раздела для создания линейных диаграмм с заполненными областями и некруглыми маркерами:

var lineDiv = document.getElementById("line-chart");var traceA = {  x: [1, 8, 13, 24, 35, 46, 60],  y: [180, 150, 210, 165, 175, 185, 220],  type: "scatter",  name: "marker: circle",  fill: "tonexty",  marker: {    symbol: "circle",    size: 10  }};var traceB = {  x: [1, 8, 13, 24, 35, 46, 60],  y: [130, 120, 160, 135, 145, 145, 119],  type: "scatter",  name: "marker: diamond",  fill: "tonexty",  marker: {    symbol: "diamond",    size: 10  }};var traceC = {  x: [1, 8, 13, 24, 35, 46, 60],  y: [110, 100, 105, 100, 115, 85, 95],  type: "scatter",  name: "marker: pentagon",  fill: "tonexty",  marker: {    symbol: "pentagon",    size: 12  }};var traceD = {  x: [1, 8, 13, 24, 35, 46, 60],  y: [70, 80, 60, 85, 75, 50, 59],  type: "scatter",  name: "marker: star",  fill: "tonexty",  marker: {    symbol: "star",    size: 12  }};var traceE = {  x: [1, 8, 13, 24, 35, 46, 60],  y: [30, 20, 30, 15, 55, 15, 30],  type: "scatter",  name: "dash: 4px, 4px, 10px",  fill: "tonexty",  marker: {    symbol: "bowtie",    size: 12  }};var data = [traceE, traceD, traceC, traceB, traceA];var layout = {  title:"Dash values for a Line Chart",  xaxis: {    rangemode: "tozero"  },  yaxis: {    rangemode: "tozero"  }};Plotly.plot( lineDiv, data, layout );

При заполнении области между разными трассами с заданным цветом необходимо иметь в виду, что все эти параметры заполняют область трассировки на основе порядка, в котором трассы передаются функции plot(), а не порядок, в котором были объявлены следы.

В нашем случае traceE является первой страссой, и перед ней ничего нет. Это означает, что значение tonexty эффективно становится tozeroy на этот раз. Мы прошли traceD после traceE, поэтому все точки между этими двумя трассами будут заполнены цветом traceD. Если бы мы сначала перенесли traceA, тогда область заполнения была бы полностью расширена сверху до y = 0. В некоторых случаях это может быть нежелательным результатом. Поэтому вы всегда должны соблюдать порядок, в котором учитываются следы.
