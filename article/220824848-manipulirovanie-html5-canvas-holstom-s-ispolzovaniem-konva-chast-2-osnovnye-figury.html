Рисование прямоугольников, кругов и эллипсов

Вы можете создавать прямоугольники в Konva, используя объект Konva.rect(). Позиция верхнего левого угла прямоугольника может быть задана с использованием свойств x и y. Аналогично, вы можете указать ширину и высоту прямоугольника, используя свойства width и height. По умолчанию все прямоугольники, которые вы рисуете, будут иметь острые углы. Однако, вы можете сделать их круглыми, выбрав подходящее значение для свойства cornerRadius.

Можно показать или скрыть прямоугольник, используя свойство visible. Если вы не хотите полностью скрывать прямоугольник, но все же делаете его полупрозрачным, вы можете использовать свойство opacity. Вы можете установить этому свойству любое число от 0 до 1 включительно. Фигура не будет видна, если opacity равно нулю.

Вы также можете поворачивать или масштабировать фигуры прямоугольника, используя свойства rotation и scale соответственно. Вращение задается как простое число, но применяется в градусах. У вас есть возможность масштабировать любой прямоугольник по оси x или y независимо, используя свойства scaleX и scaleY.

Вот пример, который рисует разные прямоугольники на холсте, используя все свойства, которые мы только что обсуждали.

var canvasWidth = 600;var canvasHeight = 400;var stage = new Konva.Stage({  container: "example",  width: canvasWidth,  height: canvasHeight});var layerA = new Konva.Layer();var rectA = new Konva.Rect({  x: 10,  y: 10,  width: 200,  height: 50,  fill: "yellow",  stroke: "black"});var rectB = new Konva.Rect({  x: 160,  y: 30,  width: 80,  height: 250,  fill: "orange",  stroke: "black"});var rectC = new Konva.Rect({  x: 200,  y: 160,  width: 180,  height: 180,  cornerRadius: 10,  strokeWidth: 10,  opacity: 0.8,  fill: "red",  stroke: "black"});var rectD = new Konva.Rect({  x: 400,  y: 20,  width: 180,  height: 180,  scaleX: 1.8,  scaleY: 0.75,  rotation: 45,  fill: "lightgreen",  stroke: "black"});layerA.add(rectA, rectB, rectC, rectD);stage.add(layerA);

Следует отметить, что прямоугольники не рисуются в том порядке, в котором они созданы. Вместо этого они рисуются в том порядке, в котором они были добавлены в слой. Свойства fill и stroke используются для установки цвета заливки и обводки соответственно.

Вы можете создавать круги в Konva с помощью объекта Konva.circle(). На этот раз свойства x и y определяют центральную точку для рисования окружности. По-прежнему можно указать значение для свойств width и height. Эти значения используются для вычисления диаметра рисуемой окружности. Однако, круг имеет одинаковую ширину и высоту. Это означает, что в случае конфликта значение, указанное ниже, имеет приоритет над указанным ранее. Другими словами, если вы установите width окружности 100, а затем установите height 180, окружность будет иметь диаметр 180 и ширина будет игнорироваться.

Чтобы избежать путаницы, можно опустить свойства width и height и указать значение radius окружности. Имейте в виду, что вы должны установить радиус 50 для того, чтобы нарисовать круг, ширина и высота которого 100.

Аналогичным образом можно создать эллипс с помощью объекта Konva.ellipse(). Единственное отличие состоит в том, что свойство radius теперь принимает объект со свойствами x и y в качестве значения. Свойства width и height, если они заданы, теперь будут применяться независимо для определения окончательной формы эллипса.

var canvasWidth = 600;var canvasHeight = 400;var stage = new Konva.Stage({  container: "example",  width: canvasWidth,  height: canvasHeight});var layerA = new Konva.Layer();var circA = new Konva.Circle({  x: 100,  y: 100,  width: 180,  fill: "yellow",  stroke: "black"});var circB = new Konva.Circle({  x: 180,  y: 150,  height: 100,  fill: "orange",  stroke: "black"});var circC = new Konva.Circle({  x: 200,  y: 275,  radius: 100,  opacity: 0.5,  fill: "red",  stroke: "black"});var ellipA = new Konva.Ellipse({  x: 400,  y: 75,  width: 70,  height: 100,  rotation: -15,  fill: "lightgreen",  stroke: "black"});var ellipB = new Konva.Ellipse({  x: 400,  y: 75,  width: 80,  height: 120,  rotation: -15,  strokeWidth: 5,  fill: "white",  stroke: "black"});var ellipC = new Konva.Ellipse({  x: 450,  y: 275,  radius: {    x: 100,    y: 50  },  scaleY: 2,  fill: "violet",  stroke: "black"});layerA.add(circA, circB, circC, ellipB, ellipA, ellipC);stage.add(layerA);

Следует отметить, что ellipB имеет большую высоту и ширину по сравнению с ellipA. Так как они оба имеют одинаковые значения x и y, мне пришлось добавить ellipB в первый слой для того, чтобы сохранить ellipA видимым. Если бы ellipB был добавлен после ellipA, он был бы нарисован над ellipA, скрывая его от зрителей.

Если вы внимательно посмотрите, вы увидите, что фиолетовый круг на самом деле является эллипсом с радиусом y, равным 50, и радиусом x, равным 100. Однако, он был масштабирован в 2 раза в направлении y. Масштабирование также увеличивает ширину штриха, делая его в два раза шире в верхней и нижней части эллипса по сравнению с его левым и правым краями.

Рисование линий с помощью Konva

Вы можете использовать объект Konva.Line() для создания различных видов линий и кривых. Все линии требуют указания точек, через которые должна проходить линия, с помощью свойства points. Точки задаются в виде массива.

Можно объединить любой набор точек, предоставленный с помощью массива points, чтобы сформировать многоугольник, задав для атрибута closed значение true. Аналогичным образом можно преобразовать набор прямых линий в сплайны, задав значение атрибута tension. Нулевое значение приведет к прямым линиям. Более высокие значения создают кривые линии.

Можно создать замкнутую и изогнутую фигуру (большой двоичный объект), задав значение для свойства tension, а также закрыв кривую, установив для параметра closed значение true.

Как и остальным фигурам, которые мы обсуждали, здесь можно задать ширину штриха для рисования линий с помощью атрибута strokeWidth. Можно также указать цвет fill для замкнутых фигур.

В следующем примере я использовал тот же набор точек для построения всех фигур. Тем не менее, я использовал метод move() для сдвига каждой фигуры на определенное расстояние, чтобы избежать перекрытий.

var canvasWidth = 600;var canvasHeight = 400;var stage = new Konva.Stage({  container: "example",  width: canvasWidth,  height: canvasHeight});var layerA = new Konva.Layer();var lineA = new Konva.Line({  points: [50, 20, 20, 100, 80, 140, 60, 80, 200, 20],  stroke: "black"});var lineB = new Konva.Line({  points: [50, 20, 20, 100, 80, 140, 60, 80, 200, 20],  closed: true,  fill: "yellow",  stroke: "black"});var lineC = new Konva.Line({  points: [50, 20, 20, 100, 80, 140, 60, 80, 200, 20],  tension: 0.8,  stroke: "blue"});var lineD = new Konva.Line({  points: [50, 20, 20, 100, 80, 140, 60, 80, 200, 20],  tension: 1.8,  stroke: "red"});var lineE = new Konva.Line({  points: [50, 20, 20, 100, 80, 140, 60, 80, 200, 20],  closed: true,  tension: 2.2,  fill: "lightblue",  stroke: "black"});lineB.move({  x: 180,  y: 40});lineC.move({  x: 380,  y: 0});lineD.move({  x: 0,  y: 200});lineE.move({  x: 180,  y: 220});layerA.add(lineA, lineB, lineC, lineD, lineE);stage.add(layerA);

Следует также отметить, что красная и синяя линии построены с использованием одного и того же набора точек, но другое значение tension значительно изменяет окончательную форму кривой.

Рисование правильных многоугольников

Вы можете тщательно выбрать значение различных точек в массиве points, чтобы нарисовать правильные многоугольники, такие как пятиугольники и шестиугольники. При рисовании более сложных правильных многоугольников, таких как восьмиугольники, используя этот метод, может быть громоздким и подверженным ошибкам. Чтобы избежать ошибок, для создания правильных многоугольников, вы должны использовать объект Konva.RegularPolygon().

Свойства x и y используются для определения центра многоугольника. Свойство radius используется для указания расстояния между центральной точкой многоугольника и всеми его вершинами. С помощью свойства sides можно указать количество сторон, которое должен иметь многоугольник. Имейте в виду, что все стороны многоугольника, созданные с помощью этого метода, будут иметь одинаковую длину. Вы можете изменить длину некоторых сторон, используя свойства scaleX и scaleY, но это также изменит ширину обводки масштабируемой стороны.

Как и все другие формы, которые мы обсуждали, вы можете изменить ширину обводки, непрозрачность и видимость правильных полигонов, используя strokeWidth, opacity и visibility.

var canvasWidth = 600;var canvasHeight = 400;var stage = new Konva.Stage({  container: "example",  width: canvasWidth,  height: canvasHeight});var layerA = new Konva.Layer();var triangle = new Konva.RegularPolygon({  x: 150,  y: 275,  sides: 3,  radius: 100,  scaleY: 1.6,  stroke: "black",  fill: "rgba(200,0,200, 1)",});var square = new Konva.RegularPolygon({  x: 60,  y: 60,  sides: 4,  radius: 50,  fill: "rgba(200,0,0, 0.5)",  stroke: "black"});var pentagon = new Konva.RegularPolygon({  x: 160,  y: 160,  sides: 5,  radius: 80,  fill: "rgba(0,200,0, 0.5)",  stroke: "black"});var hexagon = new Konva.RegularPolygon({  x: 350,  y: 120,  sides: 6,  radius: 80,  fill: "rgba(0,0,200, 0.5)",  stroke: "black"});var octagon = new Konva.RegularPolygon({  x: 450,  y: 275,  sides: 8,  radius: 100,  fill: "rgba(200,200,0, 0.5)",  stroke: "black"});layerA.add(triangle, square, pentagon, hexagon, octagon);stage.add(layerA);
