Facebook имеет более 1,71 миллиарда активных пользователей по всему миру, которые тратят в среднем 20 минут за посещение. Это объясняет, почему «в среднем кнопки Like и Share просматриваются почти на 10 миллионах веб-сайтов ежедневно» (через Zephoria).

По правде говоря, Facebook нельзя игнорировать.

Но добавляет ли иконка Facebook все, что вам нужно для вашего сайта? Есть несколько отличных способов интеграции Facebook на ваш веб-сайт, основанных на WordPress. От календаря событий Facebook до красивого Facebook Like Box, вот 10 лучших виджетов Facebook для WordPress, заслуживающих вашего внимания:

1. Facebook Events Calendar WordPress Plugin

Использующий официальный Graph API от Facebook, Events Calendar WordPress Plugin - отличный виджет календаря событий.

Одной из самых мощных функций является возможность отображения событий из нескольких страниц, профилей и групп Facebook.

Дополнительные функции:

  • поддержка виджетов и шорткодов
  • использование часового пояса браузера пользователя
  • готовые WPML и qTranslate
  • поддержка нескольких экземпляров
  • и многое другое

Дизайн чист и проработан, и CSS полностью настраивается, чтобы вписаться в вашу текущую тему еще больше.

Facebook Events Calendar WordPress Plugin - отличный способ интегрировать календарь событий Facebook в ваш сайт на WordPress.

2. Facebook Events Calendar For WordPress

Facebook Events Calendar For WordPress plugin - действительно хороший виджет календаря событий Facebook. Фактически, вы, вероятно, обнаружите, что встраиваете его в полноэкранное представление страницы больше, чем как виджет боковой панели.

Функционал включает:

  • включает два макета: полный и компактный
  • встраивание с помощью виджетов или шорткодов
  • подробные сведения о всплывающем окне быстрого просмотра
  • стиль календаря или списка
  • и многое другое

Мне очень нравится, как этот виджет Facebook для WordPress дает вам возможность отображать события в полном формате календаря или простым представлением списка.

Facebook Events Calendar For WordPress также можно найти для Joomla, а также PHP версию.

3. Facebook Group Widget

Этот виджет Facebook для WordPress - отличный способ связать активность между вашим сайтом на WordPress и группой Facebook.

Facebook Group Widget отображает последнюю дискуссию из любой публичной группы Facebook.

Этот виджет:

  • Имеет настройку CSS
  • хорошо документирован
  • легок в использовании
  • отзывчивый

Настройте количество постов в Facebook, установите ограничение по лимиту и несколько других параметров, и у вас будет Facebook Group Widget, который легко будет добавлен в область виджетов WordPress.

4. Facebook Plugins, Comments & Dialogs for WordPress

Предыдущие плагины WordPress Facebook были довольно нишевыми. Facebook Plugins, Comments & Dialogs for WordPress, являются универсальным решением для виджетов Facebook.

Этот плагин Facebook для WordPress включает в себя:

  • 11 плагинов Facebook, интегрируемых одной строкой кода
  • 8 настраиваемых виджетов Facebook
  • интеграция с комментариями Facebook
  • поддержка шорткода
  • и многое другое

Если вы хотите наиболее полнофункциональное решение для интеграции Facebook на свой сайт, вам необходимо рассмотреть плагин Facebook Plugins, Comments & Dialogs for WordPress.

5. Facebook Recent Comments Widget for WordPress

Facebook Recent Comments Widget for WordPress приносит активность комментариев со страницы вашего Facebook и переносит их в область виджетов WordPress.

Он SEO дружелюбный и супер простой в использовании.

Возможности:

  • получение уведомлений по электронной почте для комментариев в Facebook
  • ручное или автоматическое одобрение комментариев для показа
  • настройка виджета

Facebook Recent Comments Widget for WordPress - это простой способ показать последние комментарии Facebook на вашем веб-сайте, основанном на WordPress.

6. Facebook Subscribe for WordPress

Плагин Facebook Subscribe for WordPress - это самый простой способ добавить кнопку Facebook Subscribe в WordPress.

Он также включает полную поддержку тегов шаблонов и простые в использовании шорткоды.

Дополнительный функционал:

  • встроенный плагин TinyMCE для создания кнопок подписок
  • поддерживает параметры кнопок, URL профиля, макеты и т. д.
  • Facebook API Locale для других языков
  • и многое другое

Facebook Subscribe for WordPress дает вам все, что вам нужно, чтобы легко создавать кнопки подписок Facebook без необходимости знать какой-либо код.

7. Social Stream for WordPress with Carousel

Social Stream for WordPress with Carousel - это жизнеспособный вариант, независимо от того, хотите ли вы одновременно показывать только Facebook или несколько социальных сетей.

Погружайте свои потоки социальных медиа на свой веб-сайт на WordPress стильно.

Есть несколько вариантов отображения, включенных в эту карусель социальных сетей:

  • объединить несколько каналов социальных сетей в один
  • 3 разных стиля и настраиваемые цвета
  • более 60 анимационных эффектов
  • отзывчивый макет фидов
  • и многое другое

Если вы заинтересованы в смешивании в другими фидами социальных сетей вместе с фидом Facebook, Social Stream for WordPress with Carousel может быть именно тем, что вы ищете.

8. Social SEO Responsive Landing Page Facebook

В то время как Социальная SEO-ориентированная целевая страница Facebook не является виджетами (и не темой WordPress), это любопытный ресурс для тех, кто заинтересован в размещении контента Facebook на своем веб-сайте.

Эта отдельная целевая страница генерирует контент исключительно на основе данных Facebook - с возможностью входить в плейлисты Twitter и YouTube.

Все отображаемое приходит из Facebook Graph API. Изображение обложки, содержимое и даже цветовая схема могут быть автоматически установлены на основе обложки. Если вы заинтересованы во внесении изменений, вы можете легко добавить немного контента, отредактировать основной файл HTML или даже настроить PHP, если у вас есть навыки кодинга.

Social SEO Responsive Landing Page Facebook является уникальным решением и идеально подходит для организаций, которые исторически имели только страницу на Facebook и заинтересованы в том, чтобы начать продвигать себя в Интернете.

9. Jetpack by WordPress.com

Если вы уже используете Jetpack by WordPress.com, и вам нужен только Facebook-Box, использование Jetpack является явным победителем.

Откройте параметры Jetpack, убедитесь, что он включен, и просто зайдите в свои виджеты, чтобы установить его на место и настроить соответствующим образом.

Задайте свои параметры:

  • отображение фото, ленты или обложки
  • URL страницы в Facebook
  • заголовок и высота

Вот и все.

Jetpack by WordPress.com прост, свободен, и возможно умеет все, что вам нужно.

Я предлагаю вам избегать его.

Тем не менее, Facebook предоставляет все необходимое для создания своего собственного виджета.

Вы можете создать:

  • встраиваемые комментарии
  • кнопки шаринга
  • кнопки подписок
  • Кнопки лайков
  • и многое другое

Следуйте онлайн справочникам и пошаговым инструкциям, чтобы настроить код. Вставка вашего фрагмента в WordPress Text Widget может быть всем, что вам необходимо для успешного создания вашего социального плагина Facebook.
