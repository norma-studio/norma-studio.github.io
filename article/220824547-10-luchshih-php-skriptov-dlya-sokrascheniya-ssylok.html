10 лучших PHP-скриптов для сокращения ссылок...

Выбор PHP-скриптов для сокращения ссылок приносит столько же разнообразия и различной степени сложности, как и любое другое решение для веб-разработки. Self-hosted скрипты могут быть настроены для внутреннего использования, но они также могут быть использованы публично....

Выбор PHP-скриптов для сокращения ссылок приносит столько же разнообразия и различной степени сложности, как и любое другое решение для веб-разработки. Self-hosted скрипты могут быть настроены для внутреннего использования, но они также могут быть использованы публично.

Существует большое количество PHP-скриптов для сокращения ссылок, которые могут быть переделаны в полностью монетизированные сервисы сокращения ссылок, такие как AdFly. В то же время эти решения могут использоваться маркетологами для анализа и максимизации их маркетинговых усилий, не говоря уже о тех, кто может отслеживать свои личные ссылки, не используя сторонние решения, такие как Bitly.

Независимо от того, какой тип решения для вы ищете, вот PHP-скриптов для сокращения ссылок, которые, несомненно, помогут вам начать работу:

1. AdLinkFly - Monetized URL Shortener

AdLinkFly - Монетизируемый сокращатель ссылок, не являющиеся типичным решением. Это надежная система, предназначенная для монетизации сокращателей - во многом как монетизированные сервисы сокращения ссылок, такие как AdFly и ouo.io.

«Позволить пользователям сокращать ссылки и зарабатывать деньги, а также получать долю прибыли».

Эта готовая система издателей и рекламодателей поддерживает несколько платежных шлюзов, включая:

  • PayPal
  • Payza
  • Bitcoin
  • Банковские переводы
  • и много других

С помощью четырех различных инструментов API, админ-панели, входа в систему в социальных сетях и т. д. Вы мгновенно запустите свою собственную настроенную систему сокращалок.

AdLinkFly - Monetized URL Shortener идеально подходит для рекламодателей, реферальных систем и т. д.

2. Mighty URL Shortener | Short URL Script

Mighty URL Shortener | Short URL Script предлагает несколько полезных функций сокращения URL-адресов и упаковывает их в сервис на основе членства.

Пользователи могут выбирать из разных планов, которые могут быть настроены и оплачивать услугу сокращения ссылок через PayPal, биткойн, банковский перевод и многие другие.

Некоторые из функций, включенных для пользователей:

  • отображение статей веб-сайта на странице перенаправления
  • система комментариев на страницах короткой ссылки
  • расширенная статистика и аналитика
  • логины пользователей социальных сетей
  • и многое другое

Вы также можете разрабатывать приложения, которые работают с сервисом, используя расширенный API. Mighty URL Shortener | Short URL Script действительно мощьный.

3. Premium URL Shortener

Этот PHP URL сокращатель предлагает некоторые приятные функции и включает в себя некоторые инструменты CMS, которые помогут вам создать что-то потрясающее. Независимо от того, хотите ли вы публичную или self-hosted систему, Premium URL Shortener может это совместить.

Функционал включает:

  • полностью отзывчивый шаблон со встроенным редактором шаблонов
  • полнофункциональная панель администратора и мощная панель инструментов
  • Вход для пользователей Facebook и Twitter
  • геотаргетинг и расширенный API
  • jQuery с fallback

Он также включает систему закладок URL с букмарклетом:

«URL-адреса автоматически связаны с зарегистрированными пользователями и сохраняются в их учетной записи для будущего доступа».

Это позволяет пользователям перетаскивать инструмент буклета в свою панель закладок браузера, что упрощает сокращение URL-адреса любого веб-сайта, который они просматривают.

Документация и поддержка включены, Premium URL Shortener не следует упускать из виду.

4. LinkPress - URL Shortener, Links Directory & Link Management Platform

Получите строгое представление о ваших маркетинговых каналов, используя LinkPress - URL Shortener, Links Directory и Link Management Platform. Построенная на PHP, MySQL, Laravel и Bootstrap, эта платформа управления также выглядит хорошо с ее отзывчивым дизайном и поддержнкой retina.

Вы можете легко добавлять ссылки по одному, но также поддерживается массовый импорт. Имеется возможность быстро и легко добавлять и сокращать тысячи ссылок за один раз.

Другие функции:

  • Система комментариев Disqus
  • защита от спама и ботов
  • Google AdSense
  • отслеживание ссылок со статистикой
  • мощный API
  • и многое другое

С помощью этой системы у вас также есть возможность собирать детали ссылок, такие как заголовок, описание, ключевые слова и т. д. Затем эта информация может появляться на карточках деталей ссылок с предварительными просмотрами и ключевыми словами.

Создание, анализ, сокращение и многое другое с помощью LinkPress - URL Shortener, Links Directory и Link Management Platform.

5. Shortny - The URL Shortener

Shortny - Shortcut сокращает большую часть раздувания, найденного в других скриптах, и предлагает простое решение, которое выполняет свою работу.


В интерфейсе пользователи могут:

  • создавать сокращенные URL-адреса
  • защищать ссылоки паролем
  • копировать и делиться ссылками
  • смотреть статистику ссылок

На back-end стороне, администратор веб-сайта может:

  • смотреть статистику
  • удалять любые URL-ссылки
  • изменить настройки сайта и рекламы

Shortny - The URL Shortener - это надежное решение, которое предоставляет только необходимое количество функций, которые ищут большинство людей.

6. URL Shortener with Ads and Powerful Admin Panel

The URL Shortener with Ads and Powerful Admin Panel - это еще одна система типа AdFly. Когда пользователь нажимает на ссылку, созданную этим сокращателем, перед отображением исходного URL отображается до двух страниц рекламы.

С технической стороны функционал включает в себя:

  • Работает на PHP / MySql
  • настраиваемые шаблоны
  • включает API, файлы LESS и скомпилированный CSS

Некоторый функционал пользователя и администратора включает:

  • назначение различных права администратора
  • Поддержка AdSense и аналитики
  • Социальные ссылки
  • поисковые URL-адреса
  • и многое другое

URL Shortener with Ads and Powerful Admin Panel включает в себя достаточно возможностей для создания собственного рекламного сокращателя ссылок.

7. Shortny B - Shorten in Style

Если вы ищете простой и стильный сокращатель, вам нужно взглянуть на Shortny B - Shorten in Style. С его отзывчивым дизайном и чистым внешним видом, реклама с перенаправлением URL редко выглядит так хорошо.

Когда URL-адреса сокращаются, у пользователей есть несколько вариантов настроек:

  • вобор тегов, позиция сверху или снизу для размещения объявлений
  • добавление кнопки с настраиваемым URL-адресом
  • установка текста заголовка
  • Пользовательский URL

А затем можно конировать и делиться своей недавно созданной ссылкой.

После того, как ссылка будет доступна, можно посмотреть статистику ссылок:

  • топ визиты
  • топ ссылки
  • и многое другое

Back-end панели администратора есть возможности:

  • изменять настройки сайта
  • редактировать и удалять ссылки
  • добавлять скрипты трекинга
  • веб-статистика
  • и многое другое

Shortny B - Shorten in Style - элегантный сокращатель, который стоит попробовать.

8. Shortix – URL Shortener

Shortix - URL Shortener упрощает разработку анонимной и / или системы с зарегистрированными пользователями. Зарегистрированные пользователи могут просматривать статистику ссылок и могут получить доступ к API-интерфейсу сокращений, в то время как анонимные пользователи могут просто поделиться своим недавно созданным сокращенным URL-адресом.

«Shortix» - это простой и красивый скрипт, который позволяет запустить собственный сервис сокращения URL-адресов, веб-сервис, который предоставляет короткие псевдонимы для перенаправления длинных URL-адресов».

Дополнительные функции:

  • Поддержка PHP и SMTP
  • опциональный API-интерфейс сокращения
  • защита от спама
  • Интерфейс AJAX
  • и многое другое

Shortix - URL Shortener прост в установке и готов к использованию для вашего следующего проекта.

9. URL Shortener

Одним из самых простых сокращений URL-адресов является URL Shortener.

Но не позволяйте простоте обмануть вас. У него есть много мощного.

Функционал включает:

  • генерация QR-кода для сокращенных URL-адресов
  • Проверка Google, Bing и Яндекс.
  • поддержка AdSense и аналитики
  • кнопки социальных ссылок
  • кнопка автоматической копии
  • Поддержка AJAX
  • API разработчика
  • и многое другое

Благодаря мощной панели администрирования и простому установщику URL Shortener может быть именно тем, что вы ищете.

10. URL Shortener Script with Statistics

Shortener Script with Statistics вмещает невероятное для его ценовой категории. Используя Google Shortener API, этот скрипт для URL-адресов является хорошим и мощным.

Поскольку он использует Shortener API от Google, «он предлагает расширенную статистику ссылок из Google, такую ​​как рефералы, браузеры или операционную систему, используемые посетителями».

Дополнительные функции:

  • Карта посетителей из Google Shortener API
  • ссылки с настраиваемым псевдонимом
  • чистый и полностью отзывчивый дизайн
  • QR-код сгенерированной короткой ссылки
  • 3 Готовые для AdSense области
  • и многое другое

URL Shortener Script with Statistics - это впечатляющее решение, которое обеспечивает как форму, так и функционал.
