Домашняя страница Laravel описывает фреймворк так:

PHP Framework для веб-мастеров

Коротко, ясно, и по существу. То есть, если вы уже знакомы с PHP и другими фреймворками, то конечно. Если нет, то у нас есть много материала для вас. Например:

Если это не удовлетворит ваше желание, у нас также есть несколько доступных курсов.

После того, как вы сделали решительный шаг и познакомились с фреймворком, вы можете заинтересоваться инструментами и пакетами, которые доступны для вас, чтобы включить их в свою работу.

Чтобы помочь вам в этом, рассмотрим двадцатку самых популярных пакетов Laravel, доступных в настоящее время на Envato Marketplace.

1. Генератор Sximo CMS (для Laravel 4.1)

Sximo стремится добавить базовые функции CRUD (create-read-update-delete) к вашему приложению Laravel, поддерживаемому базой данных MySQL.

Он предлагает следующие функции:

  • PHP Framework, основанный на Laravel. Он быстр и лёгок.
  • Дизайн основан на фреймворке Bootstrap CSS, чтобы быть уверенным, что ваш сайт выглядит великолепно и выглядит хорошо на всех устройствах.
  • Code Builder позволяет вам создавать модули, которые будут расширять приложение дальше.
  • Sximo предоставляет полную библиотеку, чтобы приступить к созданию приложения как на фронтенде, так и на бекнде.

Он предлагает дополнительные функции, такие как встроенные функции входа в систему, функции пользователя и группы, управление меню и многое другое.

2. Шаблон администрирования Josh

Josh - еще один шаблон администрирования, ориентированный на Laravel, который призван облегчить управление вашим приложением.

Он предлагает:

  • Отзывчивый дизайн
  • HTML5 и CSS3
  • Дизайн, основанный на Bootstrap

Он также предлагает полный фронтенд и бекенд, так что вы можете получить представление об использовании вашего приложения в течение определенного периода времени. Он предлагает данные о просмотрах, продажах, подписчиках, зарегистрированных пользователях и т.д.

3. ArtVenue

ArtVenue упрощает создание собственного веб-сайта для обмена изображениями.

Пакет включает следующие функции:

  • Геотеги для фото
  • Детали EXIF для каждого изображения
  • Уникальные профили для каждого участника веб-сайта
  • Социальные функции, такие как возможность подписаться и отменить подписку
  • Возможность загрузить исходное изображение
  • Поддержка подписки на заданный канал через RSS
  • ... и другие

Для разработчиков, которые хотят построить решение для фотографов (или фотографов, которые занимаются веб-разработкой), это может быть отличным решением, если вы хотите взять на себя полную ответственность за свою работу, не размещая ее на принадлежащем и управляемым кем-то другим сайте .

4. Генератор Sximo CMS (для Laravel 5.1 LTS)

Эта версия Sximo использует многие функции, предлагаемые в Laravel 5.1.

Подобно версии, совместимой с более ранними версиями Laravel, она построена на:

  • MySQL
  • Bootstrap
  • PHP
  • JQuery
  • CSS
  • ... и др

Он также предлагает возможность сортировки, поиска и загрузки записей, а также возможность экспорта данных в формате Excel. Кроме того, он предлагает редактор MySQL, макет формы, редактор параметров и многое другое.

Если вы хотите создать собственную CMS, но хотите сделать это на Laravel 5.1, то эта версия Sximo может быть именно то, что вам нужно.

5. Мобильный сайт Builder Pro

Этот пакет позволяет создавать размещенные мобильные сайты на Laravel. Возможно, самая крутая вещь в пакете - это то, что он не останавливается на базовом веб-сайте.

Вместо этого он предлагает:

  • социальные медиа
  • Электронная коммерция
  • изображения
  • видео
  • ... и др

Поэтому, если вы работаете в мобильном бизнесе и хотите, чтобы ваше решение поддерживалось Laravel, обязательно проверьте этот пакет.

6. Kotakin для самостоятельного обмена файлами

Kotakin превращает ваше веб-приложение на основе Laravel в решение для обмена файлами, которое позволяет вам легко обмениваться файлами и папками с другими.

Если вы знакомы с такими сервисами, как Dropbox и/или Box, то вы знакомы с типом возможностей, предлагаемым здесь. Этот пакет предоставит вашей программе следующие возможности:

  • API
  • Многоязычность
  • Управление пользователями
  • Встроенный просмотрщик мультимедиа
  • Редактор шаблонов
  • Загрузка файла через Ajax
  • Адаптивный дизайн
  • Возможность поиска
  • Миниатюры изображений
  • ... и многое другое

Если вы хотите разместить свою собственную службу общего доступа к файлам без необходимости использовать сторонний ресурс, проверьте этот пакет.

7. Classic Invoicer

Допустим, вы несете ответственность за управление бэк-офисными службами компании, часть которых включает выставление счетов. Бизнес пытается запустить все свое программное обеспечение самостоятельно, но в настоящее время не имеет системы выставления счетов.

Обратите внимание на Classic Invoicer. Этот пакет позволяет вам запустить систему выставления счетов поверх Laravel. Он предлагает следующие функции:

  • Он предлагает гибкий дизайн, чтобы вы могли использовать приложение на своем компьютере и мобильном устройстве.
  • Вы можете хранить информацию о клиенте для каждого счета-фактуры.
  • Существует возможность обрабатывать прогнозы в приложении.
  • Он управляет различными типами платежей.
  • ... и др.

Хотите сохранить собственные счета-фактуры в пользовательском приложении? Не пропустите этот пакет.

8. Buzzy для новостей, списков, опросов и видео

На оборотной стороне бизнес-пакетов находится Buzzy, который представляет собой пакет, ориентированный на запуск социального сайта, ориентированного на новости, вирусные списки, опросы и видео.

Buzzy предлагает следующие возможности:

  • 100% отзывчивый дизайн
  • Настраиваемые макеты
  • Панель администрирования внутреннего плана
  • Профили пользователей
  • Интеграция с различными каналами социальных сетей
  • Встроенный редактор сообщений
  • Многоязычная поддержка

Если вы хотите создать собственный социальный новостной сайт, который предлагает все, что есть на других крупных сайтах, то это может быть именно то, что вам нужно.

9. BeDrive для совместного использования файлов и облачного хранилища

Как и Kotakin BeDrive стремится быть похожим на Dropbox и Google Drive, предоставляя аналогичный, впечатляющий набор функций, все из которых предназначены для работы на Laravel.

Используя BeDrive, у вас есть возможность:

  • Предлагать файловый хостинг
  • Предоставить панель для ваших пользователей
  • Принимать платежи
  • Управлять приложением с панели администрирования
  • Просмотр аналитики, связанной с приложением
  • Просмотр предпросмотра файлов
  • Делиться файлами через социальные сети
  • И многое, многое другое

Основные требования приложения включают, по меньшей мере, PHP 5.4 вместе с расширением MCrypt PHP, а также MySQL или аналогичный сервер баз данных, такой как Postgres, SQLite или SQL Database Server.

10. Клиент и сервер API RESTful

Этот конкретный пакет является надстройкой для Sximo, которая рассматривается выше, и превращает ее в полный веб-сервис, доступ к которому можно получить через различные API точки входа.

Это означает, что у вас будет возможность генерировать ключи API для клиентских приложений, встроенные функции CRUD, а также основные функции GET, PUT, POST, DELETE и базовую проверку подлинности.

11. Улыбнитесь СМИ

В соответствии с желанием создавать сайты, которые помогают продвигать контент для перехода на вирусы, Smile Media предоставляет пакет, который поможет вам сделать это, используя:

  • Адаптивный дизайн
  • Различные форматы носителей
  • Поддержка Amazon S3
  • Аналитика
  • Настройка
  • Такие инструменты, как PHP, Sass, CSS3, HTML5 и другие

Smile Media представляет собой надежную платформу, с помощью которой вы и ваши пользователи можете делиться медиа в своих социальных сетях.

12. Employer Zone

Одним из аспектов ведения бизнеса является отслеживание всех вещей, связанных с человеческими ресурсами.

Существует множество служб, которые могут помочь вам в этом, но если вы хотите сделать это в контексте вашего собственного приложения и/или вашего собственного сервера, тогда Employer Zone может оказаться для вас ценной альтернативой.

Пакет предлагает:

  • Возможность хранить данные о персонале
  • Управление учетными записями праздников, учет посещаемости и праздничные записи
  • Управление заявками на работу
  • Управление заработной платой и управление расходами
  • Поддержка Twilio для построения интеграции SMS
  • и многое другое

В конечном итоге пакет предлагает богатый набор функций, каждый из которых может помочь малым и средним компаниям эффективно и эффективно управлять своими человеческими ресурсами.

13. ShotPro Community

Такие сервисы, как Dribbble, проделали потрясающую работу по созданию сообществ творческих типов, которые хотят делиться друг с другом и взаимодействовать друг с другом. Что, если бы вы создали собственный сайт, специально нацеленный на то же самое?

Войдите в ShotPro.

14. Loyalty Cards for Mobile Site Builder Pro

Если вы используете сайт, основанный на пакете Mobile Site Builder Pro, то Loyalty Cards могут быть ценным дополнением для вас.

Позвольте клиентам сэкономить на скидках с помощью Loyalty Cards.

Это простое расширение, но оно позволяет клиентам создавать учетные записи и сохранять купоны, поскольку они становятся доступными для их учетной записи, которые затем могут быть использованы в магазине на базе Mobile Site Builder Pro.

15. zFiles для общего доступа к файлам

zFiles - другой пакет, который стремится предоставить еще одну альтернативу для совместного использования файлов, как и другие, которые мы видели ранее. Однако выбор никогда не был плохим!

  • Загрузка несколько файлов.
  • Установка квот использования диска.
  • Защита файлов паролем.
  • Встроенная рекламная система
  • Бесплатные обновления, темы, поддержка и многое другое.

Кроме того, и прямо с сайта:

ZFiles - это усовершенствованный PHP Multiple File Uploader, который позволяет посетителям и пользователям загружать сразу несколько файлов одновременно. Также можете делиться файлами с другими пользователями или создавать общедоступную ссылку для всех. zFiles были созданы для всех, кто должен постоянно отправлять или получать файлы.

Так что, если ни одно из других решений, которые уже были упомянуты, вам не подходит, то, возможно, zFiles - это то что нужно.

16. Niche Job Ads

Говоря о доставке программного обеспечения на дом, возможно, вы хотите перечислить работу (или рабочие места) для вашей компании в Интернете, но предпочли бы избежать этого на одном из основных сайтов.

Почему бы не разместить его на своем собственном домене с помощью приложения, такого как Niche Job Ads? КNiche Job Abs стремится сделать это настолько простым и безболезненным, насколько это возможно, чтобы создать доску объявлений и запустить ее на вашем приложении Laravel.

Демоверсию проекта вы можете просмотреть в действии со их демонстрационной страницы продукта. Обязательно используйте имя пользователя «demo@demo.com» и пароль «demopass».

17. Online Store CMS

Совсем не редко можно встретить людей, продающих товары через Интернет. Дело в том, что мы часто делаем это с помощью сторонних магазинов, которые в конечном счете берут комиссию за предоставление нам возможности перечислять наши товары через их рынок.

Эта модель работает хорошо: они приносят аудиторию, вы приносите товары. Но что, если вы хотите пойти еще дальше, и создать интернет-магазин специально для ваших товаров на условиях вашего сервера?

Тогда Online Store CMS может быть тем пакетом Laravel, который подходит именно вам. Он предлагает:

  • Меню и управление категориями
  • Современный интерфейс
  • Интеграция с Stripe и PayPal
  • Встроенное построение страницы
  • Простой способ управлять продуктами и заказами
  • ... и многое другое

Соедините этот пакет с некоторыми другими вещами, которые мы рассмотрели в этой статье, и вы могли бы запустить весь свой бизнес поверх нескольких пакетов Laravel.

18. jCMS

Системы управления контентом являются одним из основных продуктов веб-программного обеспечения. Хотя есть много вариантов, каждый из них предлагает уникальный подход к решению проблемы.

Некоторые предлагают больше возможностей, чем другие, некоторые более самоуверенные, чем другие, и некоторые из них сложно настроить. Но если вы знакомы с Laravel и ищете популярную, удобную в использовании CMS, построенную на основе фреймворка, то обратите свое внимание на jCMS.

Он предлагает возможность создавать блог, сайт электронной коммерции, галерею или любой другой тип управления контентом - все из их конфигурации приложения и все поверх основания из Laravel.

19. Stock Awesome Inventory System and Stock Control

Одной из задач ведения любого бизнеса, который держит инвентарь, является управление записями упомянутого инвентаря. Stock Awesome направлена на то, чтобы упростить этот процесс с помощью веб-программного обеспечения, предназначенного для работы на Laravel.

Из описания пакета:

Попрощайтесь со своей книгой запасов в офисе. Stock Awesome - это веб-приложение, которое позволяет вам отслеживать расходные материалы офиса и складские рассылки. Он позволяет пользователю легко управлять своими товарами и инвентарем, легко управлять уровнем запасов и стоимостью предметов в магазине, а также отправленных, следя за тем, чтобы поддерживать правильный уровень запасов, чтобы удовлетворить спрос, сохраняя при этом затраты на содержание запасов к минимуму.

Неплохо, правда? Кроме того, он имеет встроенную функцию поиска, позволяющую находить и извлекать элементы и их местоположение. Также имеются усовершенствованные возможности управления для хранения вкладок по количеству инвентаря и поддержки оповещений, когда запасы становятся низкими.

20. Signal Autos

Вот милая идея для веб-приложения: онлайновый автосалон, встроенный в пакет Laravel.

Signal Autos позволяет вам управлять всеми вашими списками автомобилей через веб-интерфейс в вашем собственном приложении. Но не только это, он также включает в себя:

  • Поддержка REST API Laravel 5
  • Родное приложение для Android
  • Возможность автоматической синхронизации содержимого из мобильного приложения и веб-приложения
  • Документация API для будущего развития
  • Многоязычная поддержка
  • ... и др

Если вы занимаетесь продажей автомобилей и хотите сделать это через собственное приложение и через свое собственное программное обеспечение для Android, Signal Autos может стать для вас надежным вариантом.

Заключение

В последние годы Laravel вырос из перспективной PHP-инфраструктуры во что-то более, что используется во всей отрасли в качестве прочной основы для создания веб-приложений на основе PHP. Его подход MVC к созданию веб-приложений, выделенного диспетчера зависимостей и системы упаковки делает его надежным выбором для веб-проектов.

Именно такие ресурсы, как перечисленные выше, помогают разработчикам создавать зрелые и надежные решения для своих клиентов. Поскольку Laravel продолжает развиваться и появляются дополнительные пакеты на Marketplace, мы обязательно их рассмотрим.
