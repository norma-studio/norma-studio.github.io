В этом уроке я покажу вам, как создать абстрактную картину, объединив несколько исходных изображений с абстрактными элементами в программе Adobe Photoshop. Кроме стандартной техники совмещения с помощью корректирующих слоёв, маски и кистей, вы также узнаете, как создать абстрактный объект в программе Adobe Illustrator, как применить текстуру и усилить глубину резкости, а также многое другое.

1. Добавляем Пейзаж и Текстуру

Шаг 1

Создайте новый документ 1500рхх 1000рх в программе Photoshop. Установите настройки, которые указаны на скриншоте ниже.

Шаг 2

Откройте исходное изображение с пейзажем. С помощью инструмента Перемещение (Move Tool (V), переместите данное изображение на наш рабочий документ.

Шаг 3

Далее, идём Фильтр –Размытие-Размытие по Гауссу (Filter > Blur > Gaussian Blur). Установите радиус размытия 6 px.

Шаг 4

На небе присутствует нежелательное облако. Т.к. мы планируем расположить все основные элементы в центре, то данная деталь может испортить весь эффект.

Создайте новый слой, а затем выберите инструмент Штамп (Clone Tool (S). Используйте данный инструмент, чтобы исправить выделенный участок.

Шаг 5

Откройте исходную гранжевую текстуру, переместите данную текстуру на наш рабочий документ, расположив поверх слоя с пейзажем. Поверните текстуру на 90 градусов по часов стрелке, для этого идём Редактирование - Трансформация - Поворот на 90 градусов по часовой стрелке (Edit > Transform > Rotate 90 CW). Далее, отразите текстуру по вертикали, для этого идём Редактирование - Трансформация - Отразить по вертикали (Edit > Transform > Flip Vertical). С помощью инструмента Свободная трансформация (Free Transform Tool (Ctrl+T), примените масштабирование к текстуре в соответствии с размерами рабочего документа:

Поменяйте режим наложения для слоя с гранжевой текстурой на Мягкий свет (Soft Light), непрозрачность слоя 100%.

Шаг 6

Чтобы уменьшить жёлтую насыщенность сцены, идём Слой- Новый корректирующий слой- Цветовой тон / Насыщенность (Layer > New Adjustment Layer > Hue/Saturation). Уменьшите значение Насыщенности (Saturation) для Жёлтых (Yellows) тонов до -78

Шаг 7

Добавьте корректирующий слой Кривые (Curves), чтобы затемнить задний фон с пейзажем. Основной свет, падающий на изображение модели, будет поступать с переднего плана, поэтому, задний фон должен быть темнее.

2. Добавляем Модель

Шаг 1

Откройте исходное изображение с моделью. Отделите изображение модели от заднего фона. Переместите выделенное изображение модели на наш рабочий документ, расположив в центре композиции.

Шаг 2

Нажмите второй значок в нижней части панели слоёв, чтобы добавить слой-маску к слою с моделью. С помощью инструмента Лассо (Lasso Tool (L), создайте искривлённое выделение на голове модели.

Перейдите на маску слоя, далее, переключите цвет переднего плана с чёрного на белый и нажмите клавишу (Delete).

Шаг 3

Создайте новый слой ниже слоя с моделью. С помощью инструмента Лассо (Lasso Tool (L) создайте активное выделение за головой модели. Постарайтесь создать рваное выделение, как в предыдущем шаге.

Залейте выделенную область тёмно-коричневым цветом (#281e1e):

Шаг 4

Перейдите на слой с моделью, а затем ещё раз с помощью инструмента Лассо (Lasso Tool), выделите два участка: один вокруг видимого глаза, а второй в области руки и спины.

Далее, идём Слой- Новый корректирующий слой - Кривые (Layer > New Adjustment Layer > Curves). При создании данного коррктирующего слоя, не забудьте поставить галочку в окошке Использовать предыдущий слой для создания обтравочной маски (Use Previous Layer to Create Clipping Mask). Потяните кривую вверх, чтобы усилить яркость.

Теперь выделенные участки выглядят ярче, чем существующая кожа.

Шаг 5

Далее, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски, чтобы уменьшить красноту на изображении модели.

Шаг 6

Добавьте корректирующий слой Цветовой Баланс (Color Balance). Поменяйте настройки для Средних тонов (Midtones). Данный шаг совместит цвет изображения модели с задним фоном.

Шаг 7

Добавьте корректирующий слой Кривые (Curves) для осветления области контура, в частности, в области спины, щеки и тёмной части руки. Перейдите на маску корректирующего слоя Кривые (Curves) и с помощью мягкой чёрной кисти, прокрасьте фронтальную часть изображения, восстановив оригинальную яркость на этих участках.

3. Добавляем горы

Шаг 1

Откройте исходное изображение с горами. С помощью инструмента Волшебная палочка (Magic Wand Tool (W), выделите только изображение горы. Переместите выделенное изображение горы на наш рабочий документ, расположив гору на голове модели. Значительно уменьшите масштаб изображения горы.

Шаг 2

К слою с горой, добавьте слой-маску и с помощью жёсткой чёрной кисти, скройте фрагмент горы, закрывающий глаз модели. А также удалите лишние участки изображения, чтобы гора воссоздавала форму головы модели. Сохраните пустоты в голове для создания глубины резкости и тени.

Шаг 3

Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation), чтобы полностью обесцветить изображение горы. Примечание переводчика: не забудьте преобразовать данные корректирующий слой, а также следующие корректирующие слои в обтравочные маски к слою с горой.

Шаг 4

Добавьте корректирующий слой Цветовой Баланс (Color Balance). Установите следующие настройки для Средних тонов (Midtones), которые указаны на скриншоте ниже.

Шаг 5

Добавьте корректирующий слой Кривые (Curves), чтобы усилить контраст изображения горы, т.к. гора выглядит немного мутной.

4. Создаём Абстрактные Эффекты на Заднем Фоне

Шаг 1

Откройте ещё раз исходное изображение с пейзажем. С помощью инструмента Овальная область (Elliptical Marquee Tool (M), выделите фрагмент изображения в правой части изображения пейзажа:

Нажмите клавиши (Ctrl+C), чтобы скопировать выделенную область. Создайте новый слой поверх гранжевой текстуры (ниже корректирующего слоя Цветовой тон / Насыщенность (Hue/Saturation), а затем нажмите клавиши (Ctrl+V), чтобы вклеить скопированное выделение на новый слой. Расположите вклеенный фрагмент пейзажа за изображение модели.

Поменяйте режим наложения для слоя с вклеенным фрагментом пейзажа на Мягкий свет (Soft Light), непрозрачность слоя 100%:

Шаг 2

Загрузите активное выделение, для этого, удерживая клавишу (Ctrl), щёлкните по миниатюре слоя с фрагментом пейзажа. Далее, идём Выделение - Модификация - Сжать (Select > Modify > Contract). Установите значение Сжатия (Contract) на 30 px:

Нажмите значок Добавить слой-маску (Add layer mask), а затем нажмите клавиши (Ctrl+I) для инверсии маски.

Шаг 3

Продублируйте слой с получившимся эффектом, а затем нажмите клавиши (Ctrl+T), чтобы немного сместить дубликат абстрактного эффекта вправо, а также уменьшить его масштаб.

Далее, перейдите на маску дубликата слоя, переключите цвет переднего плана на белый цвет, а затем поменяйте кисть на кисть Брызги. Используйте различные кисти Брызги, чтобы восстановить круг и создать рваные неоднородные края.

Шаг 4

Создайте новый слой поверх корректирующих слоёв Цветовой тон / Насыщенность (Hue/Saturation) и Кривые (Curves) (которые были созданы в параграфе 1). Выберите инструмент Эллипс (Ellipse Tool), в панели управления данного инструмента. установите режим на Слой-фигура (Shape Layers), цвет фигуры #e3e3d0 Нарисуйте круг за изображением модели, закрыв существующие круги.

Поменяйте режим наложения для данного слоя с кругом на Мягкий свет (Soft Light), непрозрачность слоя 100%:

Шаг 5

К слою с нарисованным кругом, добавьте слой-маску и с помощью мягкой чёрной кисти с низкой непрозрачностью, сделайте круг едва уловимым со слегка видимыми краями.

5. Создаём Абстрактный Круг

Шаг 1

Откройте программу Illustrator. Создайте новый документ. Установите настройки,которые указаны на скриншоте ниже.

С помощью инструмента Эллипс (Ellipse Tool (L), нарисуйте круг, цвет круга #716E72 (не забудьте выбрать Заливу (Fill) Без цвета (None), а также установите толщину обводки (stroke) 8 pt:

Шаг 2

Удерживая клавишу (Alt), потяните за круг, чтобы продублировать его. С помощью инструмента Свободная трансформация (Free Transform Tool (E), немного уменьшите масштаб дубликата круга, расположив его внутри существующего круга. Поменяйте цвет обводки на более тёмный оттенок (я выбрал цветовой оттенок #3E3D3F).

Шаг 3

Продублируйте первый слой, а затем поместите дубликат слоя в центр предыдущего, значительно уменьшив его масштаб.

Шаг 4

Нарисуйте круг, залейте его следующим цветовым оттенком #231F20, расположите его внутри всех существующих кругов.

Шаг 5

Выделите все слои с кругами, а затем идём в меню Объект > Переход > Параметры перехода (Object > Blend > Blend Options):

Нажмите клавиши (Ctrl+Alt+B), чтобы создать плавный переход между кругами с вышеуказанными настройками. Результат должен быть, как на скриншоте ниже.

Шаг 6

Нажмите клавишу (F5), чтобы открыть вкладку Кисть (Brush). Щёлкните по выпадающему меню и выберите Открыть библиотеку кистей > Художественный > Художественные чернила (Open Brush Library > Artistic > Artistic_Ink). Выберите Сухие чернила 2 (Dry Ink 2):

Сохраните кисть в качестве EPS файла для использования в своих будущих проектах.

6. Добавляем Абстрактный Круг

Шаг 1

Возвращаемся в программу Photoshop. В окне браузера, выберите файл с абстрактным кругом, растрируйте его. Примените настройки, которые указаны на скриншоте ниже (не бойтесь уменьшить или увеличить размер круга, если есть необходимость).

Шаг 2

Поместите круг за изображением модели и немного поверните его. Поменяйте режим наложения для слоя с кругом на Мягкий свет (Soft Light), непрозрачность слоя 100%, а затем добавьте слой-маску и с помощью кисти Брызги, создайте рваные края на круге, придавая ему гранжевый вид. Примечание переводчика: цвет кисти чёрный.

Шаг 3

Добавьте корректирующий слой Цветовой Баланс (Color Balance) в качестве обтравочной маски. Измените параметр Голубого (Cyan) цвета для настройки тонового режима Средние тона (Midtones). Примечание переводчика: автор добавляет данный корректирующий слой в качестве обтравочной маски к слою с кругом.

Шаг 4

Продублируйте слой с кругом, поменяйте режим наложения для дубликата слоя на Умножение (Multiply), непрозрачность слоя 100%. Расположите данный дубликат круга в верхней части головы за горами, а затем с помощью кисти Брызги, создайте рваные края на маске этого круга. Эффект рваных краёв, вы можете создать на своё усмотрение - нет необходимости делать их на 100% похожими на мои.

Шаг 5

Продублируйте слой с кругом несколько раз, а затем поменяйте режим наложения для дубликатов слоёв на Мягкий свет (Soft Light). Расположите дубликаты с кругами за моделью, а также расположите их на лице и плече модели, а затем с помощью кисти Брызги, придайте им гранжевый вид.

Уменьшите непрозрачность круга, который расположен в правой части сцены, до 50% (данный круг выделен на скриншоте ниже). Нет необходимости располагать круги в том же порядке, как у меня - не стесняйтесь экспериментировать, проявите свою креативность!

Шаг 6

Создайте новый слой поверх слоя с моделью в качестве обтравочной маски, поменяйте режим наложения на Перекрытие (Overlay), непрозрачность слоя 100%, а также выполните заливку 50% нейтральным серым (fill it with 50% gray):

Выберите инструмент Осветлитель / Затемнитель (Dodge and Burn Tool (O) в панели управления данногоинструмента, установите Диапазон (Range) на Средние тона (Midtones), значение Экспозиции (Exposure) примерно 15-20%. С помощью указанных инструментов, проведите коррекцию вдоль ярких и рваных краёв участков кожи, созданных с помощью Curves (Кривые) и абстрактных кругов, чтобы они выглядели реалистичными. А также немного затемните область у рта. Вы можете увидеть мою обработку при режиме Нормальный (Normal) и результат при режиме Перекрытие (Overlay) на скриншоте ниже:

7. Добавляем Лес

Шаг 1

Откройте исходное изображение с лесом. С помощью инструмента Прямоугольная область (Rectangular Marquee Tool), выделите верхнюю часть изображения леса. Переместите выделенный фрагмент леса на наш рабочий документ, расположив его на изображении модели. Поверните данный фрагмент леса, как показано на скриншоте ниже.

К слою с фрагментом леса, добавьте слой-маску и с помощью кисти средней жёсткости, скройте участки текстуры леса так, чтобы эффект был виден только в области спины модели.

Шаг 2

К слою с лесом добавьте корректирующий слой Кривые (Curves), чтобы затемнить текстуру леса. Примечание переводчика: не забудьте преобразовать данный корректирующий слой в обтравочную маску к слою с текстурой леса.

Теперь, перейдите на маску корректирующего слоя Кривые (Curves) и с помощью мягкой чёрной кисти, проведите коррекцию светотеней, чтобы текстура леса стала полностью частью спины.

Шаг 3

Выделите ещё один фрагмент леса, используя оригинальное изображение с лесом. Переместите данный фрагмент леса на наш рабочий документ, расположив его в области головы.

К данному слою с фрагментом леса, добавьте слой-маску и с помощью кисти Брызги, создайте нежный гранжевый эффект, как показано на скриншоте ниже.

Шаг 4

Используя ту же самую технику, добавьте ещё эффекты леса на лицо, тело, руку, одежду модели, а также на горы. Здесь не существует правил - мы создаём абстрактные вещи, поэтому, просто экспериментируйте и не ограничивайте своё воображение, творите, пока не почувствуете, что получилось хорошо!

8. Создаём Линии

Шаг 1

Создайте новый слой поверх всех остальных слоёв. Выберите инструмент Линия (Line Tool (U). В панели управления данного инструмента, установите Толщину (Weight) линии 2 px, цвет линии #1b1101. Нарисуйте несколько наклонных линий слева направо поверх головы модели.

Шаг 2

Сгруппируйте все слои с линиями в одну группу, а затем к созданной группе, добавьте слой-маску. С помощью мягкой чёрной кисти, размойте линии в области головы модели - т.к. нет необходимости, чтобы линии в этой области выглядели слишком очевидными.

9. Добавляем Дизайн Кругов

Шаг 1

Откройте набор с абстрактными фигурами растрируйте их точно так же, как мы растрировали абстрактный круг. Выделите круги, которые вам нравятся, чтобы расположить их на теле модели, а также между линиями:

Шаг 2

К некоторым слоям с кругами, добавьте слой-маски, чтобы совместить их с телом модели. Примечание переводчика: с помощью мягкой чёрной кисти с низкой непрозрачностью прокрасьте круги, которые расположены на теле модели, создав плавное затенение. Я поменял режим наложения для слоя с кругом, который расположен на плече модели на Мягкий свет (Soft Light), непрозрачность 100%:

Шаг 3

Используйте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation), где вам необходимо изменить цвет или уменьшить насыщенность

10. Добавляем Цветы

Шаг 1

Откройте исходное изображение с цветами. Переместите данное изображение на наш рабочий документ, расположив их в верхней части спины модели.

К слою с цветами, добавьте слой-маску, чтобы совместить цветы со спиной модели.
