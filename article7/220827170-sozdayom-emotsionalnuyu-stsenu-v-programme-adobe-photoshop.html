В этом уроке я покажу вам, как создать тёмную эмоциональную сцену со спящей девушкой. Вы узнаете, как объединить различные исходные изображения, чтобы создать волшебную лесную сцену. Мы проведём коррекцию света и контраста, чтобы усилить тёмную и загадочную атмосферу.

Исходные материалы

Следующие исходные материалы были использованы в этом уроке:

  • Модель
  • Заброшенный Лес 1
  • Заброшенный Лес 2
  • Лес
  • Листья

1. Создаём Передний План

Шаг 1

Создайте новый документ1500рх х 1080рх в программе Photoshop. Установите настройки,которые указаны на скриншоте ниже.

Шаг 2

Откройте исходное изображение с Заброшенным лесом 1. С помощью инструмента Перемещение (Move Tool (V), переместите данноеизображение на наш рабочий документ. С помощью инструмента Свободная трансформация (Free Transform Tool (Ctrl+T), слегка поверните изображение, как показано на скриншоте ниже:

Шаг 3

Скройте лишний ствол дерева в левой части изображения, для этого используйте фрагмент с правой части изображения. С помощью инструмента Лассо (Lasso Tool (L), выделите фрагмент леса в правой части изображения. Установите радиус Растушёвки (feather) выделенной области на 50.

Щёлкните правой кнопкой мыши по выделенной области и в появившемся окне, выберите опцию Скопировать на новый слой (Layer via Copy). Таким образом, мы получили выделенную часть изображения на новом слое. Отразите по горизонтали (Flip it horizontally) выделенный фрагмент леса, для этого идём Редактирование - Трансформация - Отразить по горизонтали (Edit > Transform > Flip Horizontal), a затем сместите данный фрагмент леса в левую часть изображения, чтобы закрыть указанную область со стволом дерева:

Нажмите второй значок в нижней части панели слоёв, чтобы добавить слой-маску к слою с выделенным фрагментом леса. С помощью мягкой круглой кисти, цвет кисти чёрный, скройте жёсткие края, чтобы совместить фрагмент леса с оригинальным изображением. Примечание переводчика: не забудьте перейти на маску слоя, для этого, просто щёлкните по ней.

Шаг 4

К слою с выделенным фрагментом леса, добавьте корректирующий слой Кривые (Curves) в качестве обтравочной маски, чтобы лучше совместить со всей сценой. Идём Слой- Новый корректирующий слой - Кривые (Layer > New Adjustment Layer > Curves). В окне кривых, уменьшите яркость.

Шаг 5

Далее, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски, чтобы уменьшить насыщенность изображения.

Шаг 6

Ближе к средней части изображения, присутствует лишнее небольшое дерево. Чтобы удалить его, создайте новый слой и выберите инструмент Штамп (Clone Tool (S). Будьте аккуратнее при работе со штампом, отберите образец со смежного участка, а затем прокрасьте указанное дерево. Будьте внимательны при отборе источника клонирования, соблюдая свет /контраст заднего фона и всей сцены.

Шаг 7

Откройте исходное изображение с Заброшенным лесом 2. С помощью инструмента Прямоугольная область (Rectangular Marquee Tool (M), выделите участок на переднем плане на данном изображении леса.

Добавьте выделенный участок леса на наш рабочий документ, расположив данное изображение на переднем плане с помощью трансформации (Ctrl+T).

Добавить слой-маску к данному слою с выделенным фрагментом леса. С помощью мягкой круглой кисти, цвет кисти чёрный, скройте жёсткие края, чтобы совместить фрагмент леса существующей сценой. Теперь земля выглядит насыщенной и более интересной.

Шаг 8

Для соответствия цвета данного участка земли со всей сценой, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски. Увеличьте значение Насыщенности (Saturation) до +39.

Шаг 9

Добавьте корректирующий слой Цветовой Баланс (Color Balance). Усильте красные тона в режиме Средние тона (Midtones) на +19: Примечание переводчика: автор проводит коррекцию добавленного фрагмента земли, поэтому. не забудьте преобразовать данный корректирующий слой Цветовой Баланс (Color Balance) в обтравочную маску.

Шаг 10

Добавьте корректирующий слой Кривые (Curves), чтобы усилить яркость данного фрагмента земли.

2. Добавьте Лес

Шаг 1

Откройте исходное изображение с лесом. Переместите данное изображение на наш рабочий документ с помощью инструмента Перемещение (Move Tool). Расположите лес на заднем плане.

К слою с лесом, добавьте слой-маску и с помощью мягкой чёрной кисти, скройте передний план и среднюю часть изображения, чтобы совместить с существующей сценой.

Шаг 2

Теперь добавьте корректирующий слой Кривые (Curves) в качестве обтравочной маски, чтобы сделать лес светлее.

Перейдите на маску корректирующего слоя Кривые (Curves) и с помощью мягкой кисти чёрного цвета, скройте эффект коррекции по бокам, чтобы сфокусировать освещение в центре изображения.

3. Создаём Основную Атмосферу

Шаг 1

Добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) поверх всех остальных слоёв. Уменьшите Насыщенность (Saturation) до -43: Примечание переводчика: автор проводит общую коррекцию, поэтому, корректирующие слои НЕ надо преобразовывать в обтравочные маски.

Шаг 2

Добавьте корректирующий слой Кривые (Curves). Уменьшите яркость изображения.

Перейдите на маску корректирующего слоя Кривые (Curves) и с помощью мягкой кисти чёрного цвета, скройте эффект кривых в центре изображения, чтобы сохранить освещение в этой части сцены.

4. Добавляем Модель

Шаг 1

Отделите изображение модель от заднего фона. Переместите выделенное изображение модели на наш рабочий документ. Расположите модель на земле в середине сцены.

К слою с моделью, добавьте слой-маску и с помощью мягкой круглой кисти чёрного цвета, Непрозрачность (Opacity) кисти примерно 50–55%, смягчите края изображения модели, чтобы совместить её с изображением земли.

Шаг 2

Создайте новый слой ниже слоя с моделью, чтобы добавить тени к модели. С помощью мягкой круглой кисти чёрного цвета, Непрозрачность (Opacity) кисти примерно 30–35%, прокрасьте нижнюю левую часть тела. особенно в нижней части тени должны быть темнее.

Шаг 3

К слою с моделью, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски. Уменьшите Насыщенность (Saturation) до -43:

Шаг 4

Модель выглядит слишком яркой по сравнению с задним фоном, поэтому, добавьте корректирующий слой Кривые (Curves) в качестве обравочной маски. Уменьшите яркость изображения.

Перейдите на маску корректирующего слоя Кривые (Curves) и с помощью мягкой кисти чёрного цвета, скройте эффект кривых в верхней части, т.к. она освещается задним фоном.

Шаг 5

Создайте новый слой в качестве обтравочной маски к слою с моделью, поменяйтережим наложения на Перекрытие (Overlay),непрозрачность слоя 100%, а также выполните заливку 50% серым (50% gray):

С помощью инструмента Затемнитель (Burn Tool (O), в настройках данного инструментаустановите Диапазон (Range) на Средние тона (Midtones), а значение Экспозиции (Exposure)примерно на 10-15%, затемните нижние края изображения модели. Вы можете увидеть результат обработки кистью на скриншотениже при режиме наложения Нормальный (Normal) иконечный результат при режиме наложения Перекрытие(Overlay):

5. Добавляем Световые Эффекты на Задний фон

Шаг 1

Создайте новый слой поверх всех остальных слоёв и с помощью мягкой кисти, цвет кисти #d4d4d4, прокрасьте среднюю часть заднего фона.. Поменяйте режим наложения для данного слоя с эффектом освещения на Мягкий свет (Soft Light), непрозрачность слоя 100%:

Шаг 2

Чтобы создать мягкий световой блик, создайте новый слой и с помощью той же самой кисти, прокрасьте ту же самую область. Уменьшите Непрозрачность (Opacity) данного слоя до 40%.

6. Добавляем Листья

Шаг 1

Откройте исходное изображение с листьями. Выделите отдельные листья. Добавьте листья к нашей сцене. Не забудьте добавить падающие листья на модель. Продублируйте листья. С помощью трансформации (Ctrl+T), трансформируйте листья.

Шаг 2

К листьям, которые лежат на модели, примените стиль слоя Тень (Drop Shadow). Для этого, дважды щёлкните по слою с листиком, чтобы появилось окно стилей слоя. Установите цвет тени на чёрный.

Шаг 3

Добавьте ещё падающие листья. Не забудьте увеличить размер листьев, которые расположены на краях. К каждому слою с листьями, примените фильтр Размытие в движении (Motion Blur), чтобы придать динамику всей сцене. Для листьев слева и справа, примените две различные настройки.

Шаг 4

Выделите все слои с листьями, а затем нажмите клавиши (Ctrl+G), чтобы сгруппировать их в одну группу. Поменяйте режим наложения для слоя с созданной группой с режима Пропустить (Pass Through (режим группы по умолчанию) на режим Нормальный (Normal), непрозрачность слоя 100%. Внутри созданной группы, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation), чтобы уменьшить насыщенность изображения листьев.

Шаг 5

Добавьте корректирующий слой Кривые (Curves), чтобы затемнить листья, т.к. большинство листьев выглядят слишком ярко по сравнению с задним фоном в данный момент.

Перейдите на маску корректирующего слоя Кривые (Curves) и с помощью мягкой кисти чёрного цвета, скройте эффект кривых, сохранив яркость на листьях, особенно на тех участках листьев, которые обращены к свету на заднем фоне.

Шаг 6

Добавьте ещё один корректирующий слой Кривые (Curves), чтобы дополнительно затемнить листья.

Перейдите на маску корректирующего слоя Кривые (Curves) и с помощью мягкой кисти чёрного цвета, скройте затемнение на листьях, которые расположены близко к светлой части заднего фона.

Шаг 7

Для дальнейшего улучшения совмещения листьев с задним фоном, а также, чтобы добавить глубины, добавьте слой-маску к слою с группой с листьями. С помощью мягкой круглой кисти чёрного цвета, Непрозрачность (Opacity) кисти примерно 20–25%, уменьшите непрозрачность выделенных листьев, чтобы они как бы растворялись в свете.

7. Заключительная Коррекция

Шаг 1

Добавьте корректирующий слой Фото Фильтр (Photo Filter) поверх всех остальных слоёв. Установите светло-голубой цвет (я выбрал цвет #83ccff). Данный шаг предназначен для того, чтобы добавить холодное чувство к нашей сцене.

Шаг 2

Добавьте корректирующий слой Кривые (Curves), чтобы затемнить всю сцену.

Перейдите на маску корректирующего слоя Кривые (Curves) и с помощью мягкой кисти чёрного цвета, восстановите свет в центре изображения.
