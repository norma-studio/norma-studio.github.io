Создание Постера Национального Дня ОАЭ в Adobe Illustrator...

Национальный день в Объединенных Арабских Эмиратах (ОАЭ) отмечается каждый год 2 декабря невероятными феерверками, праздничными шоу, концертами, выставками и нерабочим праздником для всех!-...

Национальный день в Объединенных Арабских Эмиратах (ОАЭ) отмечается каждый год 2 декабря невероятными феерверками, праздничными шоу, концертами, выставками и нерабочим праздником для всех!

В этот день отмечается национализация ОАЭ, когда семь эмиратов объединились под один флаг и сформировали Объединенные Арабские Эмираты. Шейх Халифа ибн Зайд Аль Нахайян, президент федерации, возглавляет семь эмиратов (Абу-Даби, Дубай, Шарджа, Аджман, Умм-эль-Кайвайн, Рас-эль-Хайма и Фуджейра). Дубай является самым большим и популярным эмиратом, по этой причине я решила создать постер Национального дня, показывающий достопримечательность города и самое высокое строение в мире, Бурдж Халифа.

В этом уроке мы будем использовать новые настройки и усовершенствования Adobe Illustrator, а также более традиционные инструменты для создания праздничного графического плаката для Национального дня ОАЭ.

Праздник начинается!

1. Зарисовка и Настройка

Шаг 1

Мы начнем с наброска для нашего дизайна. Я советую начать с просмотра в интернете фотографий ОАЭ, Дубая и башни Бурдж Халифа, чтобы вы могли представить себе элементы для их создания. Обычно, для создания наброска, я пользуюсь Photoshop CS6 и моим планшетом Wacom Intuos Pro Pen.

В CS6 откройте новый файл: File > New (Command-N).

Дайте название файлу: Uae_National_Day_Poster_Sketch.

Выберите Preset: International Paper, Size: A4, Color Mode: CMYK, настройте Resolution на 72 Pixels/inch, и Background Contents: White. Когда разрешение поменяется, вы увидите, что Preset сменится на Custom.

Я использовала 72 pixels, потому что я сделала примерный набросок с контурами рисунка, а не добавила какие-либо подробно сделанные его части.

Шаг 2

Выберите Brush Tool (B), с сероватым цветом для вашей кисти, и нарисуйте вашу основную концепцию на рабочем поле.Рисование серой кистью поможет вам легче обвести набросок в Illustrator.

Я представляю себе яркий плакат в стиле ретро с архитектурной темой. На наброске я добавила нарисованное от руки послание на арабском языке “Мы все Эмираты”.

С помощью выбранной круглой кисти закрастье участок с текстом, выделив его тем самым. Залейте участок с текстом кистью с настройкой Mode на Multiply.

Как только ваш набросок будет готов File > Save (Command-S), используя формат JPEG , и переходите к Illustrator CC.

Шаг 3

В Illustrator CC создайте новый файл: File > New (Command-N). Назовите его UAE_National_Day_Poster, настройте Size на A4, и нажмите OK.

Откройте панель Layers : Windows > Layers. Назовите первый слой Sketch , куда мы и поместим наш рисунок.

File > Place (Command-Shift-P), выделите файл Uae_National_Day_Poster_Sketch.jpg, и нажмите OK. Вы заметите появившийся графический указатель с крошечным выбранным рисунком. Перетащите указатель и щелкните им на верхнем левом углу вашего рабочего стола, чтобы поместить ваш набросок на рабочий стол. Двойным щелчком на слое Sketch , откройте панель Layers Options . Выберите Lock & Dim Images to 50%. Нажмите OK.

Чтобы добавить новый слой над выделенным слоем, нажмите кнопку Create New Layerв панели Layersи создайте новый слой под названием Burj Khalifa, где мы начнем обводку первой части башни.

File > Save (Command-S), сохраните файл как UAE_National_Day_Poster.ai.

2. Обводка и Иллюстрация

Шаг 1

Начнем рисовать нашу башню. Выберите Rectangle Tool (M) и черного цвета Stroke, и настройте Fill на None, нарисуйте прямоугольник для измерения размеров одной из секций башни. С помощью Direct Selection Tool (A), перетащите верхний левый угол вниз, чтобы наклонить верхнюю часть, придерживайте клавишу Shift, чтобы передвинуть опорную точку строго по вертикали.

Выберите Pen Tool (P) и плавно нарисуйте кривую для создания верхней поверхности здания. Откройте панель Pathfinder : Window > Pathfinder. С помощью Selection Tool (V) выделите обе формы и из панели Pathfinder нажмите кнопку Divide.

Он делит все перекрывающиеся формы на неперекрывающиеся. Он также объединяет вместе формы. Выделите форму, Ungroup (Command-Shift G) и удалите лишние формы.

С помощью Line Segment Tool (\), нарисуйте линию от одного конца прямоугольника до другого, соединив концы. Затем вам нужно будет искривить линию немного вверх, чтобы создать впечатление изогнутой формы на башне. Выберите Anchor Point Tool (Shift-C), который находится под Pen Tool, и подтяните линию немного вверх.

Сделайте дубликат изогнутой линии, чтобы создать остальные этажи башни. Выделите изогнутую нами линию, придерживайте клавиши Option и Shift и перетащите ее вниз для копирования. Теперь у вас есть первая копия. Все, что вам нужно сделать, это сделать все остальные этажи башни, нажимая Command-D для дублирования линии. Продолжайте нажимать Command-D, пока башня будет заполнена.

Опять возьмите Line Segment Tool (\) и нарисуйте вертикальные линии от верха до низа прямоугольника. Придерживайте клавиши Option и Shift и потяните выделенную вами линию вниз для ее копирования и заполните всю башню. Некоторые линии будут короче высоты прямоугольника. Выберите Direct Selection Tool (A), чтобы вручную продлить концы. Придерживайте клавишу Shift для расставления линий по вертикали. File > Save (Command-S).

Шаг 2

Давайте начнем окраску нашей башни.

Выделите прямоугольник, есть очень простой способ скопировать и вставить объект на место. Для этого есть клавиши быстрого доступа Command-C для Копирования и Command-F для того, чтобы Вставить Вперед и Command-B для Всавить Назад. Убедитесь, что вы Вставили Назад. Вы не увидите процесс, но если перетащите выделение в сторону, то увидите вставленную копию позади. Как только сделаете дубликат прямоугольника, сделайте то же самое и с верхним полукругом.

Откройте Window > Gradient, панель Stroke и панель Swatches

В панели Gradient выберите Type: Linear, Angle: -90°, и придайте градиент от светло-красного до темно-красного, Stroke (X) на None. Затем перетащите образец градиента из панели Gradient в панель Swatches для его сохранения на случай, если нам нужно будет вновь им воспользоваться.

Выделите оба полукруга, Object > Arrange > Bring to Front (Command-Shift-]), так, чтобы мы закрыли линии. Придайте нижнему полукругу розовато-красный цвет и сохраните черную обводку верхнего полукруга.

Выделение одной из черных линий обводки с помощью Select > Same > Stroke Color выделит все формы с черной обводкой. Отмените выделение границ, придерживая клавишу Shift. Group (Command-G) внутренние линии вместе и придайте им оранжево-красную обводку с толщиной 0.3 pt.

Windows > Transparency, и дайте обводкам Screen: Blending Mode.

Выделите границы прямоугольника, Group (Command-G) и дайте им белую обводку, Rounded Caps & Corners толщиной 1 pt.

Во время просмотров рисунков Бурджа Халифа вы заметите, что некоторые этажи выделены, нам надо повторить этот эффект. Select All (Command-A), и Lock Selection (Command-2). Возьмите PenTool (P) и нарисуйте слегка изогнутый прямоугольник, заполняющий три или четыре этажа. Залейте красным цветом и Blending Mode: Screen. Object > Unlock all (Command-Option-2), и расположите выделения за белыми границами.

Выделите все элементы башни и Group (Command-G) их вместе. Теперь наша башня состоит из одной части. Все, что вам нужно сделать сейчас, чтобы завершить воспроизведение башни, это выполнить те же действия, которые мы сделали для первой ее части.

Я хотела придать башне цвета флага ОАЭ: красный, зеленый, белый и черный. Но черный слишком агрессивный, белый не будет хорошо смотрется,поэтому я поработала с красным, зеленым и бело-голубым градиентами.

Заметка: Каждый раз, когда будете создавать секцию, расположите ее позади другой секции.

Когда достигните верхушки башни, сделайте толщину обводки меньше, и просто нарисуйте вертикальные прямоугольники, воспользовавшись Rectangle Tool (M).

Как только вы закончите, Select All (Command-A) и Group (Command-G) все элементы башни вместе. File > Save (Command-S).

Шаг 3

Закройте слой Burj Khalifa, создайте новый слой и назовите его Text.

Выберите Type Tool (T), с размером шрифта 43 pt, и напечатайте текст на английском, We are all Emirates с любым шрифтом. Я работала с шрифтом Nova, который вы можете скачать бесплатно.

Я заметила, что между некоторыми буквами надо добавить пространство, для этого я использовала Touch Type Tool (Shift-T) и выделила в предложении буквы I, S и L , чтобы добавить пространство между ними.

Для ввода на арабском вам нужно полное издание Adobe-ME (Middle Eastern) для арабского, которое вы можете найти на сайте Adobe. Для арабского шрифта вы можете скачать нужный вам тип бесплатно на Fonts2U.

Возьмите Type Tool (T) и напечатайте текст на арабском, настройте толщину шрифта на 50 pt. Убедитесь в том, что Paragraph Direction настроен на Right-to-Left Direction, так как арабский читается справа налево.

Мы должны наклонить текст, выберите Shear Tool, который вы можете найти под Scale Tool . Для наклона вдоль вертикальной оси текста, перетащите курсор в окне документа вниз. Придерживайте клавишу Shift, чтобы сохранить первоначальную ширину объекта.

Вам нужно расширить текст для превращения их в формы, потому что мы придадим им эффекты на более позднем этапе. Object > Expand. Откроется панель Expand Options; убедитесь, что Object и Fill выбраны, нажмите OK. Переместите текст в любое на ваше усмотрение место.

Заприте и спрячьте слой Text, создайте новый слой над Sketch. Назовите его Background.

Шаг 4

На фоновом слое нам нужно обвести оставшиеся элементы. Выберите Pen Tool (P) и начните обводить все оставшиеся элементы.

Для моста нарисуйте длинный прямоугольник Rectangle Tool (M), придерживайте клавишу Shift и Alt. Выделив оба прямоугольника, нажмите Free Transform Tool (E). Появится виджет трансформирования с опциями ограничения, свободного трансформирования, искажения перспективы и свободного искажения. Выберите Perspective Distort (Искажение Перспективы).

Вы заметите, что Perspective Distort ограничено, так что придерживать клавишу Shift во время передвижения опорных точек не надо. Чтобы выровнить все так, как вы хотите, нажмите Free Distort и выровните соответственно.

Возьмите Rectangle Tool (M) и нарисуйте рельс на каждом конце параллельных линий. Object > Blend > Blend Options, и появится панель Blend Options. Вы можете смешать объекты для создания и распространения форм поровну между двумя объектами, для этого мы настроем Specified Steps на 6 и нажмем OK. Потом Blend > Make (Command-Option-B) и рельсы появятся.

Object > Blend > Expand объект смешения, чтобы сделать формы. Выделите рельсы и параллельные линии и в панелиPathfinder нажмите Shape Modes: Unite, чтобы создать соединенную форму. Затем завершите обводку остального наброска, переключите видимость на слое Text и удалите слой Sketch.

Откройте слой Burj Khalifa. Выделите рельсы и мост и перетащите их на слой Burj Khalifa, чтобы покрыть нижнюю часть башни. File > Save (Command-S).

Для лучшего представления плаката нам нужно покрасить все остальное.

3. Окрашивание

Шаг 1

В Дубае круглый год солнечно, поэтому мы должны работать с яркой палитрой цветов для фона, в то же самое время нам нужно согласовть ее с палитрой цветов флага. Для неба я выбрала желтоватый градиент, а остальные формы сделал либо белого и желтого цвета, либо оттенков флага ОАЭ.

Во время окрашивания я заметила, что некоторые фигуры были неправильного оттенка. Например, верхушки башни, с розово-красным цветом, были изменены на белые заливки. Также верхушка башни не соответствовала по оттенку, поэтому я сделала ее в голубой тон неба. Дымчатые следы самолета я сделала выцветившими. Переместите и поменяйте масштаб облака за следом самолета. Вы должны преобразовать белый мост, слегка расширив его. Переместите башню, слегка выровнив ее.

Иллюстрация начинает оживать.

Заприте слои Background и Burj Khalifa и отоприте слой Text.

Когда мы расширили арабский текст, все элементы были сгруппированы вместе. Так что Ungroup (Command-Shift-G) , а затем выберите буквы каждого слова и создайте соединенную форму в панели Pathfinder : Unite. В панели Pathfinder Panel Options вы найдете в меню опциюMakeCompound Shape. Выберите эту опцию и нажмите кнопку Expand в панели Pathfinder , так что форма объединена.

Для текста на английском Объедините все предложение вместе.

Теперь вы начинаете замечать, что слишком много всего происходит на одной стороне плаката, текст невозможно прочесть, так что отоприте слой Burj Khalifa и передвиньте башню влево, затем слегка уменьшите размер текста. Таким образом мы придадим ему свободное пространство. Отоприте слой Background и передвиньте облака и другие элементы, чтобы убрать беспорядок. Затем опять заприте эти два слоя и оставьте слой Text открытым.

Английский текст выглядит зажатым арабским, так что я решила перетащить его под арабский и перенести арабский текстближе. Выделив арабский текст, Object > Path > Offset на 2.2 мм, Round, и OK, чтобы дать границы.

В ранели Pathfinder нажмите Unite, а затем удалите все отверстия, которые могут появиться в новой форме. Вы также заметите некоторые пробелы, которые я обвела.

Соедините пробелы, добавив новые формы с помощью Pen Tool (P). Выделите только что созданные формы вместе с созданной границей и Объедините в Pathfinder. В Pathfinder Panel Options > Make Compound Shape и нажмите Expand. Залейте новую форму белым цветом и Перенесите Назад (Command-[).

Шаг 2

Придадим тексту эффекты.

С помощью Ellipse Tool (L) нарисуйте два овала, чтобы имитировать кривую вспышку света. Скопируйте (Command-C), и Command-F Втавьте Вперед. Из панели Pathfinder выберите Shape Mode: Intersect , чтобы создать соединенную фигуру из двух пересекающихся форм. Удалите лишний овал. Дайте новой форме градиент от черного до серого с Angle: -90°.

Придайте тот же эффект нижнему тексту.

Шаг 3

Заприте слой Text и отоприте слои Background и Burj Khalifa.

Вам, возможно, потребуется изменить неокторые элементы на слое Background, чтобы они гармонично смотрелись. Я сузила границы башни, срезала немного форму моста, придала облакам выцветивший градиент, отрегулировала цвет и форму солнца и изменила самолет. После того, как вы в целом будете довольны изображением, вы можете перейти на добавление блеска рисунку.

Заприте слой Background и переместитесь на слой Burj Khalifa.

Шаг 4

Придерживайте и нажмите кнопку Rectangle Tool и выберите Star Tool.

Щелкните на вашем рабочем поле и откроется окно Star Option.

Заполните Radius 1: 0.5 mm, Radius 2: 7 mm, и Points: 8. Нажмите OK.

Заполните звезду Радиальным градиентом от белого до черного и примените Screen Blending Mode из панели Transparency .

Нарисуйте еще блеска с помощью Ellipse Tool (L). Создайте белый круг, Effect > Distort &Transform > Pucker and Bloat. Придайте ему Pucker -155% и Screen Blending Mode.

Шаг 5

Нарисуйте еще несколько кругов разных размеров, придайте им градиент и Screen режим наложения.

Откройте Windows > Symbols.

Перетащите звезды и круги в панель Symbols , чтобы создать новый символ. Сохраните настройки в Symbol Options, назовите символ Sparkle и нажмите OK.

Выберите Symbol Sprayer Tool (Shift-S) и, выделив символ Sparkle, распространите копии символа на форме башни. Старайтесь остаться в рамках формы и не переусердствуйте. Symbol Sprayer Tools являются очень хорошими инструментами для тестирования. Вы можете использовать символические инструменты для изменения образцов, рассеивания, увеличения, уменьшения, чередования или окрашивания экземпляров, чтобы сделать их более реалистичными.

Шаг 6

Мы почти закончили.

Отоприте все слои, Select All (Command-A) и Group (Command-G) рисунок и текст вместе. Все переместится в слой Text.

Удалите пустые слои.

С помощью Rectangle Tool (M) нарисуйте прямоугольник в центре рабочего поля, чтобы замаскировать рабочее поле.

Вам придется переместить или поменять масштаб сгруппированного рисунка, чтобы захватить все внутри прямоугольника.

Выделите прямоугольник и сгруппируйте рисунок, Object > Clipping Mask > Make (Command-7). Теперь он выглядит так, как будто взят в рамку.

Готово!

Праздничный плакат Национального дня готов! Будем праздновать.

File > Print (Command-P), и можете повесить его в вашем районе.

Если вы знаете инструменты и клавиши быстрого доступа, то Illustrator CC поможет вам работать быстрее и эффективнее. Надеюсь, вам понравился этот урок, и вы уже создаете ваш праздничный национальный дизайн. Проявим патриотизм!!
