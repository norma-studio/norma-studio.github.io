Полное руководство по инструменту Type (Шрифт)...

1. Основы

Type - это инструмент Photoshop для создания векторных контуров, математических фигур и символов шрифта. Он расположен на стандартной панели инструментов в виде крошечной буквы T. Быстрая клавиша для этой команды - буква T, и если вы нажмете, удерживая, на этот инструмент (или нажмете Shift + T несколько раз), вы увидите четыре опции: Horizontal Type, Vertical Type, Horizontal Type Mask и Vertical Type Mask.

  • Horizontal Type Tool: позволяет создавать горизонтальный стандартный текст (слева направо и сверху вниз).
  • Vertical Type Tool: позволяет создавать вертикальный текст (сверху вниз и справа налево), что позволяет писать на восточных языках, например, на японском или китайском, или если вы хотите поэкспериментировать с дизайном шрифта.
  • Horizontal Type Mask Tool, Vertical Type Mask Tool: создает быструю маску с использованием фигуры в качестве выделения. Вскоре мы подробнее рассмотрим эти опции.

Как только вы выберете нужный инструмент, вы заметите, что курсор мыши изменяется на стандартный курсор шрифта, что-то вроде I, это означает, что можно начинать вводить текст.

2. Создание слоя со шрифтом

Текстовый слой можно создать двумя способами, Point Type и Paragraph Type:

  • Point Type: создает слой со шрифтом в одну строку, новая строка появится, если вы нажмете Return или Enter на клавиатуре.

    Чтобы создать слой типа Point Type, выберите нужный инструмент Type (Горизонтальный или Вертикальный текст) и щелкните один раз курсором в нужном месте. Затем просто начните писать, а по окончании нажмите либо маленькую кнопку Commit на панели параметров, либо Enter на цифровой клавиатуре или просто нажмите Command + Enter.

  • Paragraph Type: создает текстовый слой с текстом в рамке. Подойдет и для печати, и для веб-дизайна. Чтобы добавить слой абзаца, сначала необходимо создать прямоугольник. Щелкните и перетащить курсор по диагонали, пока не получите прямоугольник нужного размера.

Неважно, есть ли фоновое изображение или какой-либо другой объект, инструмент Type создаст новый слой для нового текста. Кроме того, вы можете легко переключаться между Point Type и Paragraph Type, перейдя в Layer > Type > Convert to Paragraph Text / Convert to Point Text.

3. Изменение размера и трансформация

Очевидно, можно изменять размер и форму текстового слоя с помощью Move Tool (V), выбрав слой и отобразив элементы трансформации. Можно растянуть, увеличить или сильно исказить форму шрифта.

Чтобы лучше использовать Paragraph Type, сделайте следующее: Выберите инструмент Type и нажмите на тексте абзаца, затем перейдите к одному из элементов трансформации, нажмите и перетащите, чтобы изменить размер поля, и текст автоматически изменит свою форму. Нажмите и перетащите, чтобы сохранить пропорции или поддерживать постоянное приращение вращения, удерживайте Command и перетащите, чтобы изменить размер текста, удерживайте Command и перетащите центральную точку, чтобы перекосить окно с текстом, и удерживайте Option и перетащите для изменения размера окна из его центра.

4. Параметры шрифта

Основное форматирование

Пришло время двигаться вперед. После ввода текста вы, наверное, захотите изменить шрифт, цвет и многое другое, и это очень просто. Сначала надо вызвать окно Character, перейдя в Window > Character. В окне Character есть несколько опций для форматирования текста, есть несколько примеров форматирования: Font Family, Style, Size и Color (нужно дважды щелкнуть по цветовому образцу, чтобы увидеть набор цветов). После ввода шрифта можно щелкнуть по миниатюре текста на панели Layers и изменить весь формат текстового слоя, или выбрав инструмент Type, щелкнуть по текстовому слою, который вы хотите отредактировать, выделить его и изменить его формат.

Кернинг и трекинг

Расстояние между буквами легко настроить в Photoshop. Выберите текстовый слой или выделите текст и посмотрите значения Kern и Track в окне Character.

Чтобы настроить кернинг (пробел между определенными парами символов), выберите Metrics Kerning или Optical Kerning. Metrics Kerning автоматически настроит расстояние между конкретными парами символов. Optical Kerning основывается на форме шрифта. Значения можно изменять, введя нужное число (положительное или отрицательное) в поле Kerning или выбрав один из пресетов.

С трекингом все очень похоже, просто установите числовое значение (положительное или отрицательное) для увеличения / уменьшения пробела между буквами. Кроме того, вы можете комбинировать Kern и Track, чтобы получить нужный результат.

Вертикальное и горизонтальное масштабирование

Изменяйте размер текстового слоя по горизонтали или по вертикали, изменяя значения в окне Character. Мой совет: я почти никогда не использую эти значения, потому что, изменяя размер шрифта, вы искажаете саму фигуру, и иногда получается абы-что. Посмотрите на изображение ниже: красивая буква Century Gothic - идеальный круг до масштабирования, а после превращается в овал. Кроме того, обратите внимание, что меняется и форма: линия в верхней и нижней части буквы «O» при вертикальном масштабировании становится шире.

Сдвиг базовой линии

Это пригодится, когда нужно поместить текстовый слой где-нибудь, например, вокруг изображения. Измените это значение, чтобы переместить базовую линию выделенного текста выше или ниже базовой линии остальной части текста.

Псевдополужирный шрифт и псевдокурсив

У Photoshop есть возможность автоматически создавать вариации Faux Bold и Faux Italic для любого шрифта, что иногда очень полезно, но вы должны проявлять особую осторожность, чтобы как можно меньше исказить его форму. Ниже приведены примеры Faux Bold на шрифте Arial, который не сильно искажает тип шрифта, но применение Faux Italic на Myriad Pro фактически искажает оригинальный курсивный шрифт (например, посмотрите на символ «a»).

Варианты шрифтов и текстовое оформление

Вы можете легко настроить шрифт, используя все заглавные символы или преобразуя его в маленькие заглавные буквы. Для этого просто выберите текстовый слой или выделите текст и нажмите соответствующую кнопку в окне символов. Кроме того, вы можете подчеркнуть или зачеркнуть свой текст.

Сглаживание, где и когда

Сглаживание создает текст с гладким краем, при котором границы текста смешиваются с фоновым слоем или изображением. Есть четыре варианта сглаживания: Sharp, Crisp, Strong и Smooth, и опция None. Его часто применяют для удобства чтения, в частности, для шрифта Serif. Существуют некоторые исключения, когда нужно выбирать None: для образца шрифта в веб-дизайне, при использовании пиксельного шрифта и т.д.

Интерлиньяж

В любом из текстовых слоев в полях Point или Paragraph можно настроить интерлиньяж (вертикальное пространство) между строками. Оставьте автоматическое значение (auto) или введите собственное, чтобы самостоятельно отрегулировать это пространство.

5. Свойства абзаца

Основное выравнивание

Поскольку слой текста абзаца может содержать несколько строк, их форматирование очень важно, и оно регулируется окном Paragraph. Откройте его, перейдя в Window > Paragraph. Создайте текстовый слой абзаца, введите что-нибудь, скопируйте текст и нажмите на миниатюру слоя, чтобы активировать параметры всего текста. Либо выделите текст с помощью инструмента Type и примените форматирование абзаца только к выделенным строкам текстового слоя.

Основное свойство текста абзаца - Выравнивание. Его можно легко настроить (по левой стороне, по центру, по правой стороне или по ширине листа), щелкнув на соответствующие значки в окне Paragraph. Кроме того, есть еще три параметра выравнивания по ширине листа для последней строки текстового слоя или выделенного текста (по левой стороне, по центру, по правой стороне).

Чтобы настроить выравнивание по ширине окна, перейдите в меню Paragraph и нажмите Justification, там вы сможете настроить интервалы между словами и буквами, масштабирование глифов и интерлиньяж.

Перенос строки

Настройки для переноса строк влияют на горизонтальные расстояния в абзаце, делая его шире или растягивая, в зависимости от количества слов в каждой строке. Чтобы переносить слова с помощью дефиса, просто выберите нужный язык в окне Character и активируйте перенос в окне Paragraph. Чтобы отключить его, просто снимите флажок в поле Hyphenation в окне Paragraph.

В большинстве случаев с автоматическим переносом не будет проблем, но иногда его нужно настроить. Чтобы включить расширенные варианты переносов, нажмите на меню Paragraph и выберите Hyphenation. Вы увидите следующие варианты:

  • Words Longer Than # Letters: задайте минимальное количество символов для слов, которые можно переносить; то есть если вы установите значение 5, слово «фото» не будет переноситься, но если вы установите значение «3», вы получите «фо-то». Значение по умолчанию: «2».
  • After First # Letters and Before Last # Letters: указывает минимальное количество символов в начале или в конце слова, которое может быть разбито дефисом; т.е. если вы установите любое из этих значений как «1», вы можете получить нежелательный перенос, например, «п-ривет» или «действи-е». По умолчанию оба значения устанавливаются как «2».
  • Hyphen Limit: устанавливает максимальное количество строк, на которых последовательно можно сделать перенос.
  • Hyphenation Zone: устанавливает расстояние от правого края абзаца.
  • Hyphenate Capitalized Words: предотвращает перенос слов, написанных заглавными буквами.

Отступ и пробел между абзацами

Это довольно просто. Просто выберите текстовый слой или выделите текст, и впишите значения для отступа справа и слева. Вы можете сделать отступ для текста слева или справа или отступать от первой строки абзаца, как показано на втором изображении ниже.

Кроме того, вы можете легко добавить пробел между параграфами (до или после) в окне Paragraph: просто введите нужное значение в поле Space Before или Space After.

Деформация текста

Одна из самых мощных функций инструмента Type - это способность деформировать любой текстовый слой в соответствии с конкретными потребностями. Чтобы деформировать текстовый слой, дважды щелкните по его миниатюре и нажмите кнопку Create Warped Text в меню Параметры. Вам будет предложено выбрать вертикальную или горизонтальную ориентацию с помощью трех ползунков для увеличения или уменьшения значений Bend, Horizontal и Vertical Distortion. Ниже приведен пример того, как преобразовать текст, используя Arc Warp и различные значения Bend и Distortion.

Конечно, можно также деформировать и слой текста абзаца, см. изображение ниже. Вот как получить знаменитый эффект «Звездных войн», просто увеличив значение Vertical Distortion и используя небольшую маску слоя градиента.

Все это очень весело, попробуйте различные варианты деформации. Однако есть несколько ограничений: вы не можете деформировать слой текста, если ранее применили стили Faux Bold или Faux Italic на панели Character.

Шрифт по контуру

Можно создать слой текста по контуру, созданному с помощью инструмента Pen или Vector Shape. Чтобы это сделать, сначала нарисуйте контур с помощью Pen Tool, затем выберите инструмент Type и поместите курсор в любом месте контура. Вы увидите, что форма курсора изменяет базовую линию шрифта по S-образной линии. Щелкните по контуру и введите текст.

Текст по контуру фигуры

Точно так же можно добавить текст на векторную фигуру. Во-первых, убедитесь, что вы ее выбрали с помощью Path Selection Tool. Как только будет виден контур, выберите инструмент Type, щелкните в любом месте контура и напишите что-нибудь. Вы можете отредактировать любые параметры в меню Character, например, немного увеличить сдвиг базовой линии, чтобы создать пространство между текстом и фигурой.

Редактировать шрифт на контуре

Иногда нужно изменить ориентацию и положение шрифта на контуре. Для этого выберите Path Selection Tool, поместите курсор над текстом, пока не появится маленькая черная стрелка на курсоре шрифта, затем перетащите курсор ниже базовой линии, чтобы перевернуть текст. Используйте тот же метод, чтобы перевернуть текст и поместить его внутри фигуры. Очевидно, вы можете добавить любой стиль слоя к тексту, и его позже можно будет редактировать.

Преобразовать шрифт в фигуру

Иногда приходится изменять форму букв, например, для дизайна логотипа, и с Photoshop это легко. Просто создайте текстовый слой или выделите его и перейдите в Layer > Type > Convert to Shape. Этот инструмент преобразует шрифт в маску векторного слоя, которую можно редактировать, как любую другую векторную форму. Вы также можете создать Work path, перейдя в Layer > Type > Convert Work Path.

Инструмент Type Mask

Наконец, делая все описанное выше, можно использовать не формы текста, а выделение. Для этого нажмите и удерживайте инструмент Type, пока не появятся дополнительные параметры. Там выберите Horizontal (или Vertical) Type Mask. С помощью этого инструмента можно создать быстрое выделение в виде фигуры, что пригодится в режиме Layer Mask или Quick Mask.
