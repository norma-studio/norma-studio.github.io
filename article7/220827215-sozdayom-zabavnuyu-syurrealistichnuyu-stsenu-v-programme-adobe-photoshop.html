Интро

Вначале мы объединим несколько исходных изображений, чтобы создать естественную подземную сцену. Затем мы добавим остальные элементы сцены, такие как череп, гнездо, яйцо и птицу. Далее, мы совместим все эти элементы с помощью корректирующих слоёв, коррекции маски и работы кистью. ...

В этом уроке я покажу вам, как создать смешную сюрреалистическую сцену, используя различные техники создания фотоманипуляций.

Вначале мы объединим несколько исходных изображений, чтобы создать естественную подземную сцену. Затем мы добавим остальные элементы сцены, такие как череп, гнездо, яйцо и птицу. Далее, мы совместим все эти элементы с помощью корректирующих слоёв, коррекции маски и работы кистью.

Мы также добавим элементы природы (дерево и листья), а затем используя ту же самую технику, совместим их со всей сценой. В заключение, мы добавим несколько корректирующих слоёв. В этом уроке вы узнаете про работу со слоями и группами, а также узнаете, как сфокусироваться на коррекции деталей.

1. Создаём Основу Сцены

Шаг 1

Создайте новый документ 1500 рх х 1500 рх в программе Photoshop. Установите настройки, которые указаны на скриншоте ниже.

Шаг 2

Откройте исходное изображение с небом в программе Photoshop. С помощью инструмента Прямоугольная область (Rectangular Marquee Tool (M), выделите область неба.

С помощью инструмента Перемещение (Move Tool), переместите выделенный фрагмент неба на наш рабочий документ.

Шаг 3

Чтобы добавить глубины к сцене, примените фильтр Размытие по Гауссу (Gaussian Blur), для этого идём Фильтр –Размытие-Размытие по Гауссу (Filter > Blur > Gaussian Blur). Установите радиус размытия 6 px.

Шаг 4

Добавьте корректирующий слой Цветовой Баланс (Color Balance), чтобы немного изменить цветовой тон неба.

Шаг 5

Откройте исходное изображение с текстурой Земли 1 (выберите второе изображение). С помощью инструмента Прямоугольная область (Rectangular Marquee Tool (M), выделите фрагмент земли, а затем переместите выделенное изображение земли на наш рабочий документ, расположив ниже слоя с небом.

Шаг 6

Добавьте корректирующий слой Кривые (Curves) в качестве обтравочной маски к слою с землёй, чтобы затемнить землю

Шаг 7

Чтобы обесцветить землю, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтарвочной маски. Уменьшите значение Насыщенности (Saturation) до -82:

Шаг 8

Создайте новый слой в качестве обтравочной маски к слою с землёй, поменяйте режим наложения на Перекрытие (Overlay), непрозрачность слоя 100%, а также выполните заливку 50% нейтральным серым (fill it with 50% gray):

Выберите инструмент Затемнитель (Burn Tool (O) в панели управления данного инструмента, установите Диапазон (Range) на Средние тона (Midtones), значение Экспозиции (Exposure) примерно 10-15%. Слегка затемните участки земли на краях изображения. Вы можете увидеть мою обработку при режиме Нормальный (Normal) и результат при режиме Перекрытие (Overlay) на скриншоте ниже:

Шаг 9

Откройте исходное изображение с текстурой Земли 2 (выберите четвёртое изображение). С помощью инструмента Прямоугольная область (Rectangular Marquee Tool (M), выделите верхнюю часть земли.

Переместите выделенное изображение земли на наш рабочий документ, расположив поверх слоя с землёй 1.

Нажмите второй значок в нижней части панели слоёв, чтобы добавить слой-маску к данному слою с землёй. С помощью мягкой чёрной кисти, скройте жёсткие края, чтобы совместить данную часть земли с остальной сценой.

Шаг 10

Добавьте корректирующий слой Кривые (Curves) в качестве обтравочной маски к новому слою с землёй, чтобы затемнить землю.

Далее, перейдите на маску данного корректирующего слоя и с помощью мягкой чёрной кисти, скройте эффект затемнения в верхней части земли, т.к. эта часть освещается светом с неба.

Шаг 11

Далее, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation). Уменьшите значение Насыщенности (Saturation) до -75.

Шаг 12

Откройте исходное изображение с травой. С помощью инструмента Прямоугольная область (Rectangular Marquee Tool (M), выделите изображение с травой.

Переместите выделенное изображение с травой на наш рабочий документ. Активируйте инструмент Свободная трансформация (Free Transform Tool (Ctrl+T), а затем выберите инструмент Искажение (Distort), чтобы изменить перспективу. Примечание переводчика: для выбора режима Искажения, идём Редактирование > Трансформирование > Искажение (Edit > Transform > Distort).

К слою с травой, добавьте слой-маску и с помощью мягкой чёрной кисти, скройте жёсткие края, чтобы создать плавный переход и совместить траву с землёй и небом.

Шаг 13

Находясь на слое с травой, идём Фильтр - Размытие - Размытие по Гауссу (Filter - Blur - Gaussian Blur). Установите радиус размытия 3 px. Примечание переводчика: перед тем, как применить фильтр, преобразуйте слой с травой в смарт-объект, для этого идём Фильтр - Преобразовать для смарт-фильтров (Filter - Convert for Smart Filters).

Перейдите на маску фильтра и с помощью мягкой чёрной кисти, скройте эффект размытия поверх основной части травы, сохранив эффект размытия только вдали.

Данный шаг направлен на усиление глубины сцены.

Шаг 14

На поверхности травы присутствует несколько лишних тёмных участков. Создайте для травы новый слой в качестве обтравочной маски (Ctrl+Alt+G), а затем с помощью инструментом Штамп (Clone Stamp Tool) (S), скройте эти участки.

Шаг 15

Добавьте корректирующий слой Цветовой Баланс (Color Balance) в качестве обтравочной маски к слою с травой. Увеличьте настройки для Средних тонов (Midtones) для Красных (Red), Зелёных (Green) и Синих (Blue) оттенков.

Шаг 16

Я планирую добавить основной источник света в левую часть сцены, поэтому свет будет поступать слева направо. Добавьте корректирующий слой Кривые (Curves) в качестве обтравочной маски. Потяните кривую вверх для усиления яркости.

Далее, перейдите на маску данного корректирующего слоя и с помощью мягкой чёрной кисти, скройте эффект осветления в нижней части изображения травы, особенно в нижней правой части.

Шаг 17

Добавьте ещё один корректирующий слой Кривые (Curves), чтобы затемнить нижнюю правую часть изображения травы.

Далее, перейдите на маску данного корректирующего слоя и с помощью мягкой чёрной кисти, скройте эффект затемнения в верхней левой части изображения травы.

2. Добавляем Череп

Шаг 1

Откройте исходное изображение с черепом. С помощью инструмента Прямолинейное лассо (Polygonal Lasso Tool (L), выделите только изображение черепа.

Переместите выделенное изображение черепа на наш рабочий документ, расположив череп в центре нашей сцены между травой и землёй. Поверните изображение с помощью трансформации (Ctrl+T).

К слою с черепом. добавьте слой-маску и с помощью мягкой чёрной кисти, скройте среднюю часть черепа, чтобы создать ощущение, как будто он застрял между травой и землёй.

Шаг 2

К слою с черепом, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски. Уменьшите значение Насыщенности (Saturation) до -52.

Шаг 3

Оригинальное изображение черепа освещается спереди, а также снизу сбоку. Чтобы исправить светотени на изображении черепа в соответствии с задним фоном, мы используем несколько корректирующих слоёв в качестве обтравочных масок к слою с черепом.

Вначале добавьте корректирующий слой Кривые (Curves), чтобы затемнить нижнюю часть черепа (которая расположена ниже поверхности земли).

Теперь перейдите на маску данного корректирующего слоя и с помощью мягкой чёрной кисти, скройте эффект затемнения в верхней части черепа, сохранив эту часть светлой.

Шаг 4

Добавьте ещё один корректирующий слой Кривые (Curves) для осветления верхней части черепа.

Далее, перейдите на маску данного корректирующего слоя и с помощью мягкой чёрной кисти, скройте коррекцию кривых во фронтальной части черепа.

Шаг 5

Создайте новый слой, поменяйте режим наложения на Перекрытие (Overlay), непрозрачность слоя 100%, а также выполните заливку 50% нейтральным серым (fill it with 50% gray): Выберите инструмент Осветлитель / Затемнитель (Dodge and Burn Tool (O) в панели управления данногоинструмента, установите Диапазон (Range) на Средние тона (Midtones), значение Экспозиции (Exposure) примерно 15-20%. Усильте свет в верхней части черепа, а также усильте затенение и контраст глаз. Вы можете увидеть мою обработку при режиме Нормальный (Normal) и результат при режиме Перекрытие (Overlay) на скриншоте ниже:

Шаг 6

Создайте новый слой ниже слоя с черепом. С помощью мягкой чёрной кисти непрозрачностью примерно 40-45%, добавьте тень от черепа на траве (тень должна узкой и вытянутой, т.к. свет падает строго слева на направо).

3. Добавляем Корни

Шаг 1

Откройте исходное изображение с корнями. Переместите данное изображение на наш рабочий документ с помощью инструмента Перемещение (Move Tool), немного повернув изображение (Ctrl+T).

Далее, к слою с изображением корней, добавьте слой-маску и с помощью мягкой чёрной кисти, скройте жёсткие края изображения, чтобы совместить корни с землёй.

Шаг 2

С помощью инструмента Лассо (Lasso Tool), выделите фрагмент корней, используя оригинальное изображение, а затем переместите выделенную часть корней на наш рабочий документ, расположив в левой части нашей сцены под землёй. К данному слою с корнями, добавьте слой-маску, а затем с помощью чёрной кисти, совместите эти корни с корнями, которые мы добавили ранее.

Шаг 3

Продублируйте данный слой с корнями, а затем сместите дубликат слоя в нижнюю среднюю часть подземной сцены. С помощью слой-маски, проведите совмещение этих корней с остальными корнями.

Шаг 4

Выделите все слои с корнями, а затем нажмите клавиши (Ctrl+G), чтобы сгруппировать их в одну группу. Поменяйте режим наложения для слоя с группой с режима Пропустить (Pass Through) (режим группы по умолчанию) на режим Нормальный (Normal), непрозрачность слоя 100%.

Добавьте корректирующий слой Кривые (Curves) поверх всех слоёв с корнями (внутри созданной группы), чтобы уменьшить яркость изображений корней.

Шаг 5

Далее, добавьте корректирующий слой Уровни (Levels), чтобы затемнить отдельные участки корней, в частности область ниже поверхности земли.

Далее, перейдите на маску данного корректирующего слоя и с помощью мягкой чёрной кисти, скройте коррекцию в нижней части, так как она должна быть менее тёмной, чем верхняя.

4. Добавляем Гнездо

Шаг 1

Откройте исходное изображение с гнездом. С помощью инструмента Перемещение (Move Tool), переместите гнездо на наш рабочий документ, расположив его на черепе.

Шаг 2

Создайте новый слой ниже слоя с гнездом. С помощью мягкой чёрной кисти непрозрачностью примерно 40-45%, добавьте тени под гнездом.

Шаг 3

Создайте новый слой (вновь ниже слоя с гнездом) и с помощью мягкой чёрной кисти непрозрачностью 30%, добавьте тень от гнезда на черепе, а также тень на траве.

Шаг 4

К слою с гнездом, добавьте корректирующий слой Кривые (Curves) в качестве обтравочной маски для осветления гнезда, т.к. оно выглядит темнее, чем задний фон в данный момент.

Теперь перейдите на маску данного корректирующего слоя и с помощью мягкой чёрной кисти, скройте коррекцию кривых в правой части гнезда (справа от наблюдателя), чтобы сохранить затенение в этой части изображения гнезда.

Шаг 5

К слою с гнездом, добавьте корректирующий слой Цветовой тон / Насыщенность (Hue/Saturation) в качестве обтравочной маски, чтобы уменьшить насыщенность изображения гнезда. Уменьшите значение Насыщенности (Saturation) до -28.

5. Добавляем Яйцо и Птицу

Шаг 1

Откройте исходное изображение с яйцом. Выделите изображение яйца с помощью инструмента Волшебная палочка (Magic Wand Tool (W).

Переместите выделенное яйцо на наш рабочий документ, расположив его на уровне верхней части гнезда. Далее, к слою с яйцом, добавьте слой-маску и с помощью жёсткой чёрной кисти, скройте нижнюю часть яйца, создав ощущение, как будто оно лежит внутри гнезда (необходимо обратить внимание на мелкие зазоры в гнезде между веточками).

Шаг 2

К слою с яйцом, добавьте корректирующий слой Кривые (Curves) в качестве обтравочной маски, чтобы затемнить яйцо Далее, перейдите на маску данного корректирующего слоя и с помощью мягкой чёрной кисти, скройте коррекцию кривых в верхней части яйца, сохранив эту часть светлой.

Шаг 3

Откройте исходное изображение с птицей. С помощью инструмента Прямолинейное лассо (Polygonal Lasso Tool) или любым другим инструментом выделения, с которым вы предпочитаете работать, отделите изображение птицы от заднего фона. Переместите выделенное изображение с птицей на наш рабочий документ, расположив её на краю гнезда.

Шаг 4

Создайте новый слой ниже слоя с птицей. С помощью мягкой чёрной кисти непрозрачностью 30-35%, добавьте тень от птицы на гнездо, а также на траву. Единственную вещь, вам нужно запомнить - это то, что птица находится на самом верху гнезда (дальше от земли, чем череп), поэтому. тень должна быть светлее.

Шаг 5

К слою с птицей, добавьте корректирующий слой Кривые (Curves) в качестве обтравочной маски, чтобы ослабить светлые блики на изображении птицы. Теперь перейдите на маску данного корректирующего слоя и с помощью мягкой чёрной кисти, скройте коррекцию кривых в верхней и левой части изображения, т.к. эти участки должны быть светлее, чем остальные участки изображения птицы.

Шаг 6

Создайте новый слой в качестве обтравочной маски, поменяйте режим наложения на Перекрытие (Overlay), непрозрачность слоя 100%, а также выполните заливку 50% нейтральным серым (fill it with 50% gray): С помощью инструмента Осветлитель (Dodge Tool), осветлите хвост и левую лапку птицы, а с помощью инструмента Затемнитель (Burn Tool), затемните правую лапку птицы.

6. Добавляем Дерево

Шаг 1

Откройте исходное изображение с деревом. С помощью инструмента Перемещение (Move Tool), переместите данное изображение на наш рабочий документ, расположив в верхней правой части сцены.

К слою с деревом, примените фильтр Размытие по Гауссу (Gaussian Blur), для этого идём Фильтр –Размытие- Размытие по Гауссу (Filter > Blur > Gaussian Blur). Установите радиус размытия 5 px.

Шаг 2

Чтобы немного изменить цвет дерева, добавьте корректирующий слой Цветовой Баланс (Color Balance) в качестве обтравочной маски. Измените настройки для Средних тонов (Midtones) для Красных (Red) оттенков.

Шаг 3

Добавьте корректирующий слой Кривые (Curves) для осветления дерева, т.к. оно не достаточно светлое в данный момент.

Далее, перейдите на маску данного корректирующегослоя и с помощью мягкой чёрной кисти, проведите коррекцию светотеней между листьями. Вы можете увидеть коррекцию на слой-маске, а также результат на изображении.

7. Добавляем Листья

Шаг 1

Откройте исходное изображение с листиком. С помощью инструмента Перемещение (Move Tool), переместите листик на наш рабочий документ. Немного поверните изображение листика (Ctrl+T). Расположите листик под изображением дерева (или в любое другое подходящее место на ваше усмотрение).

Шаг 2

Продублируйте слой с листиком несколько раз, расположив дубликаты листьев вокруг черепа. К каждому листику, примените фильтр Размытие по Гауссу (Gaussian Blur), для этого идём Фильтр –Размытие- Размытие по Гауссу (Filter > Blur > Gaussian Blur). Установите радиус размытия 3 px.

Шаг 3

Дважды продублируйте один из слоёв с листьями. Увеличьте масштаб дубликатов слоёв (Ctrl+T). Расположите один дубликат листика у правой кромки сцены, а второй дубликат листика на переднем плане. К данным дубликатам с листьями, примените фильтр Размытие по Гауссу (Gaussian Blur). Установите радиус размытия 12px.

Шаг 4

Сгруппируйте все слои с листьями в одну группу точно так же, как мы уже проделывали с корнями. Чтобы изменить цвет листьев, добавьте корректирующий слой Цветовой Баланс (Color Balance) в качестве обтравочной маски. Измените настройки для Средних тонов (Midtones) для Красных (Red) и Жёлтых (Yellow) оттенков.

Шаг 5

Чтобы добавить немного света на листья, добавьте корректирующий слой Кривые (Curves) внутри созданной группы. Потяните кривую вверх для усиления яркости.

Перейдите на маску корректирующего слоя Кривые (Curves) и с помощью мягкой чёрной кисти, варьируя непрозрачность кисти между 20-100%, скройте коррекцию кривых в правой части отдельных листьев в соответствии с освещение заднего фона.

8. Заключительная Коррекция Цвета

Шаг 1

Пришло время для цветовой коррекции всей цветы. Добавьте корректирующий слой Карта градиента (Gradient Map). Установите цвета градиента #7d6a51 и #a2b54a Поменяйте режим наложения для данного корректирующего слоя на Мягкий свет (Soft Light), непрозрачность 50%.

Шаг 2

Добавьте корректирующий слой Цветовой Баланс (Color Balance). Измените настройки для Средних тонов (Midtones) для Красных (Red) и Жёлтых (Yellow) оттенков.

Шаг 3

Добавьте корректирующий слой Кривые (Curves),чтобы изменить цвет и контраст всей сцены.

Шаг 4

Добавьте корректирующий слой Вибрация (Vibrance), чтобы усилить заключительный эффект.
