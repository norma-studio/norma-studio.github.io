Как создать Текст с Эффектом Радужной Змеиной Кожи в Adobe Photoshop...

Интро

В этом коротком уроке мы собираемся создать простой текстовый эффект с помощью Стиля Слоя, похожий на текстовые эффекты, которые можно купить в-Envato Market. Мы будем использовать набор текстур, узор и пару эффектов слоев, чтобы создать текстовый эффект змеиной кожи. Давайте начнем!...

В этом коротком уроке мы собираемся создать простой текстовый эффект с помощью Стиля Слоя, похожий на текстовые эффекты, которые можно купить в Envato Market. Мы будем использовать набор текстур, узор и пару эффектов слоев, чтобы создать текстовый эффект змеиной кожи. Давайте начнем!

Исходники для Урока

При создании урока мы использовали следующие исходники:

  • Шрифт Ballpark Weiner
  • Texture 42 от xnienke
  • Texture 630 от Sirius-sdz
  • Подборка фонов Premium Blur Backgrounds bundle от raspanky
  • Узоры Rubber grip от Sinisha
  • Бесплатный набор градиентов Free Pack 6000 photoshop gradients от Supertuts007 (Загрузите файлы CSP True Sky Gradients.grd и FOILS.grd)

1. Создаем Градиентный Фон

Шаг 1

Создайте новый документ 800 x 600px и дублируйте слой Фон / Background.

Шаг 2

Дважды кликните по слою Фон копия / Background copy, чтобы применить эффект Наложение Градиента / Gradient Overlay со следующими настройками:

  • Поставьте галку в Дизеринг / Dither
  • Масштаб / Scale: 150%
  • Примените градиентную заливку, как на картинке ниже

Так мы добавим фоновый градиент.

2. Добавьте Текстуру

Шаг 1

Поместите изображение Texture 42 поверх слоя Фон копия / Background copy, затем измените его Режим Наложения / Blend Mode на Мягкий Свет / Soft Light, а Непрозрачность / Opacity на 75%.

Шаг 2

Добавьте изображение Texture 630 новым слоем и измените Режим Наложения / Blend Mode на Мягкий Свет / Soft Light.

Шаг 3

Добавьте изображение 6 (2) из набора Premium Blur Backgrounds bundle на новый слой и измените Режим Наложения / Blend Mode на Умножение / Multiply.

Шаг 4

Добавьте изображение 9 из набора Premium Blur Backgrounds bundle на новый слой и измените его Режим Наложения / Blend Mode на Цветность / Color, а Непрозрачность / Opacity на 20%.

3. Создаем и Стилизуем Текст

Шаг 1

Создайте текст, используя шрифт Ballpark Weiner, и поместите слой с текстом под слоем с изображением 9. Цвет / Color #7bb6cf, а Размер / Size 150 pt.

Дважды кликните по слою с текстом, чтобы применить к нему следующий стиль слоя:

Шаг 2

Добавьте Тиснение / Bevel and Emboss со следующими настройками:

  • Размер / Size: 10
  • Снимите галку Глобальное Освещение / Use Global Light.
  • Угол / Angle: 11
  • Высота / Altitude: 64
  • Контур Глянца / Gloss Contour: Перевернутый Конус / Cone - Inverted
  • Поставьте галку Сглаживание / Anti-aliased
  • Режим Подсветки / Highlight Mode: Яркий Свет / Vivid Light
  • Непрозрачность / Opacity: 75%
  • Режим Тени / Shadow Mode:
  • Цвет / Color: #474747
  • Непрозрачность / Opacity: 75%

Шаг 3

Добавьте Контур / Contour со следующими установками:

  • Контур / Contour: По Гауссу - Инверсия / Guassian - Inverse
  • Поставьте галку Сглаживание / Anti-aliased
  • Диапазон / Range: 65%

Шаг 4

Добавьте Текстуру / Texture с этими установками:

  • Узор / Pattern: Резина / Rubber grip

Шаг 5

Добавьте Внутреннюю Тень / Inner Shadow с этими установками:

  • Цвет / Color: #2a84a9
  • Расстояние / Distance: 0
  • Размер / Size: 5

Шаг 6

Добавьте Внутреннее Свечение / Inner Glow с этими установками:

  • Режим Наложения / Blend Mode: Линейный Свет / Linear Light
  • Непрозрачность / Opacity: 44%
  • Шум / Noise: 50%
  • Цвет свечения / Color: Используйте градиентную заливку drum 10
  • Источник / Source: Из Центр / Center
  • Размер / Size: 21
  • Контур / Contour: Мелкая Выемка / Cone - Shallow

Шаг 7

Добавьте Глянец / Satin со следующими установками:

  • Режим Наложения / Blend Mode: Линейный Свет / Linear Light
  • Цвет / Color: #fff1c0
  • Непрозрачность / Opacity: 25%
  • Угол / Angle: 7
  • Расстояние / Distance: 108
  • Размер / Size: 35
  • Контур / Contour: Тройное Кольцо / Ring - Triple
  • Поставьте галку Сглаживание / Anti-aliased
  • Поставьте галку Инвертировать / Invert

Шаг 8

Добавьте Наложение Градиента / Gradient Overlay со следующими настройками:

  • Режим Наложения / Blend Mode: Перекрытие / Overlay
  • Непрозрачность / Opacity: 42%
  • Примените градиентную заливку, как на картинке ниже

Шаг 9

Добавьте Наложение Узора / Pattern Overlay со следующими настройками:

  • Режим Наложения / Blend Mode: / Blend Mode: Мягкий Свет / Soft Light
  • Шаблон / Pattern: Узоры - Туманность / Nebula

Шаг 10

Добавьте Тень / Drop Shadow со следующими установками:

  • Цвет / Color: #1b1635
  • Непрозрачность / Opacity: 100%
  • Расстояние / Distance: 7
  • Размер / Size: 10
