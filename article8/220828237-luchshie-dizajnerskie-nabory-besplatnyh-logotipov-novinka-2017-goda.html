Редактируем Набор Логотипов в Adobe Illustrator

Не знаете с чего начать? Тогда следите за кратким описанием, которое приведено ниже, чтобы узнать с чего начать редактирование с помощью дизайнерской векторной программы.

Шаг 1

Вначале, выберите набор логотипов, который вам понравился. Давайте выберем потрясающий набор элегантных монограммных логотипов.

Далее, скачайте, а затем распакуйте исходный набор. Мы предоставляем файлы с векторным дизайном в формате EPS, данный формат можно открыть во множестве векторных программ.

Откройте исходный файл с логотипом, который вы выбрали, в программе Adobe Illustrator. Я выбрала файл с буквой ‘M’, данный логотип представлен различной комбинацией из двух букв, связанных с буквой ‘M’. Чтобы логотип соответствовал моему имени Melody Nieves, я выбрала комбинацию ‘MN’. С помощью инструмента Выделение Объектов (Group Selection Tool), чтобы выделить нужную комбинацию, а затем Вырежьте (Cut) и Вклейте (Paste) на новый слой. Скройте другие комбинации.

Шаг 2

Проявите своё воображение, экспериментируя с элементами дизайна, чтобы создать идеальный логотип. Увеличьте масштаб основных фигур с помощью инструмента Выделение (Selection Tool (V). Или добавьте и измените текст с помощью инструмента Текст (Type Tool (T).

Продолжайте редактировать свой логотип, настраивая цвета и другие настройки. Далее, выберите цвет заливки, который вы хотите применить, например, яркий однотонный цвет или красивый градиент.

Шаг 3

Как только вы завершили, Сохраните свой файл с соответствующим расширением (.ai, .eps, .jpg, .pdf, etc.), чтобы сразу же использовать свой созданный логотип. Это так просто!
