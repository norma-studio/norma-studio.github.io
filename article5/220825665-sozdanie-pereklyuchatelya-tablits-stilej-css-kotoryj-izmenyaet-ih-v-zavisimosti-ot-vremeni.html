В настоящее время переключатели стилей стали популярной возможностью веб-сайтов. Таблицы стилей CSS позволяют веб-дизайнеру изменять вид и ощущения от веб-сайта без особых усилий. Некоторые сайты используют тип переключателей стилей "Day and Night" (* «День и Ночь». Здесь и далее примеч. пер.), при котором тема сайта изменяется автоматически в зависимости от времени суток.

В этом руководстве вам будет показано, как создавать переключатель таблиц стилей CSS, который изменяет их в зависимости от времени, при помощи PHP, за счет которого вы можете немедленно менять множество таблиц стилей по наступлении определенного времени суток. Также добавлено немного кода jQuery UI забавы ради!

Почему будем использовать PHP вместо JavaScript?

Единственный недостаток при использовании PHP для реализации обсуждаемого тут переключателя таблиц стилей CSS, – это то, что время определяется либо веб-сервером, либо «часовым поясом», заданным в скрипте.

Когда я создавал свой персональный сайт-портфолио, то создал обсуждаемый здесь переключатель таблиц стилей при помощи JavaScript и jQuery. Функция скрипта, в которой задается стиль в зависимости от времени, работала корректно. Однако, всегда присутствовала короткая «вспышка» версии страниц с использованием таблицы стилей по умолчанию после загрузки страниц.

В веб-страницах с большим количеством кода JavaScript могут возникать конфликтные ситуации между различными скриптами JavaScript. Вместо него я решил использовать PHP для реализации нашего переключателя. Поскольку PHP используется на стороне сервера, то отсутствовали конфликтные ситуации JavaScript и скрипт работал как положено.

При помощи JavaScript вы можете сделать так, чтобы функции скрипта, в которых определяется стиль в зависимости от времени, работали на основе времени, установленном на компьютере пользователя.

Анализ разметки HTML-документа

Использование директивы include PHP для подключения скрипта для создания ссылок rel="stylesheet"

Скрипт для переключателя таблиц стилей CSS будет содержаться во включаемом файле (* файл с исходным текстом, который может перед трансляцией с помощью директивы include включаться (встраиваться) препроцессором в текст основной программы) PHP под названием stylesheets.php.

За счет директивы include подключается и обрабатывается указанный файл. Подключение включаемого файла stylesheets.php происходит при помощи этого кода:

 <!-- CSS style sheets --> <?php include("style sheets.php"); ?>

Поскольку мы используем скрипт PHP для изменения таблиц стилей CSS в установленное время, то ссылки rel="stylesheet" будут сгенерированы скриптом PHP.

Размещаем ссылки на библиотеки jQuery и jQuery UI

Нам необходимо убедиться, что на странице имеются ссылки на библиотеки jQuery и jQuery UI.

Разметка для главного изображения

Для того чтобы главное изображение также изменялось в установленное время при помощи скрипта PHP, оно должно быть добавлено на страницу в виде фонового изображения элемента в таблице CSS.

Так что вместо добавления изображения на страницу при помощи подобного кода:

оно было добавлено на страницу в виде фонового изображения <div> в таблице CSS с классом "main_image" следующим образом:

 <!-- Main Image Switches at Set Times --> <div class="main_image"></div> <p class="main_description"> Main Image changes at the times indicated. </p>

Анализ кода PHP для переключателя таблицы стилей CSS

Устанавливаем часовой пояс по умолчанию: 

 date_default_timezone_set("Australia/Melbourne"); // Set default time zone

С помощью функции date_default_timezone_set устанавливается часовой пояс по умолчанию, который используется всеми функциями в скрипте, работающими с датой/временем.

Если в скрипте не содержится date_default_timezone_set, то для всех функций в скрипте, работающих с датой/врменем, будет использоваться время веб-сервера, на котором располагается сайт.

Обращайтесь, пожалуйста, к List of Supported Timezones (* Список поддерживаемых часовых поясов) для нахождения соответствующего вашему местонахождению часового пояса.

В новый версиях PHP среди функциональных возможностей date_default_timezone_set имеется и "Daylight Savings Times" (* декретное время – поясное время, переведенное для территории России на один час вперед). В некоторых более поздних версиях PHP "Daylight Savings Times" не учитывается.

Устанавливаем время в 24-часовом формате записи 

 $time = date("H"); // Set the hour in 24 hour format

date("H") устанавливает локальное время/дату в 24-часовом формате, при котором впереди располагаются нули. Перейдите по этой ссылке для просмотра информации о 24-часовом формате записи времени.
