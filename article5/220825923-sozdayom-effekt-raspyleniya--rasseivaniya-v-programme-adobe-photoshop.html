В этом уроке я покажу вам, как создать эффект распыления / рассеивания на любой фотографии. В конце урока у вас будет экшен, который будет автоматически воссоздавать весь процесс за счёт одного щелчка мыши.

Я постараюсь кратко описать все действия, чтобы любой, кто первый раз открыл программу Photoshop, мог выполнить данный урок.

Экшен, который мы создадим, является частью Экшена Распыления / Рассеивания. Если вы хотите приобрести его, то вы можете перейти на сайт GraphicRiver и ознакомиться с Экшеном Распыления / Рассеивания.

Полная версия экшена имеет следующие дополнительные возможности:

  • Четыре направления рассеивания: Вправо, Влево, Вверх и Вниз.
  • Возможность изменять цвет свечения рассеивающихся частиц.
  • Возможность воссоздания оригинального заднего фона, который будет виден через рассеивание основного объекта.
  • Десять цветовых решений и дополнительная цветовая коррекция.
  • Дополнительные элементы рассеивания также включены.

В этом уроке мы создадим эффект рассеивания в направлении вправо, а также применим цвет по умолчанию.

Что вам понадобится

Для создания данного эффекта, вам понадобится следующие исходные материалы:

  • Модель
  • Кисти Пыль / Угольки (скачайте приложение к уроку)

1. Создаём Документ

Шаг 1

Вначале, откройте исходную фотографию, с которой вы хотите работать. Чтобы открыть фотографию, идём Файл –Открыть (File > Open), а затем в окне браузера, выберите свою исходную фотографию. Нажмите кнопку Открыть (Open). Теперь, перед тем, как мы приступим, необходимо, чтобы вы уточнили пару моментов:

  1. Ваша исходная фотография должна быть в цветовом режиме RGB, 8 бит/канал (8 Bits/Channel). Чтобы проверить это, идём Изображение – Режим (Image > Mode).
  2. Ваша исходная фотография должна быть слоем заднего фона. Если она не является таковой, тогда идём Слой- Новый – Задний фон из слоя (Layer > New > Background from Layer). Примечание переводчика: слой с исходной фотографией должен быть с замочком и представлять задний фон.
  3. В панели Слои (Layers), щёлкните по маленькому значку в верхнем правом углу и в появившемся меню, выберите опцию Параметры панели (Panel Options). Далее, в появившемся окне, поставьте галочку в окошке Добавить слово "копия" к скопированным слоям и группам (Add "copy" to Copied Layers and Groups).

Шаг 2

Для получения наилучшего результата, размеры вашей исходной фотографии должны быть в пределах 3500–4000 px по ширине / высоте. Если ваше исходное изображение меньше, тогда увеличьте его, для этого идём Изображение – Размеры изображения (Image > Image Size). Вы всегда можете уменьшить изображение до оригинальных размеров после того, как примените эффект.

Шаг 3

Мы планируем создать эффект рассеивания, который будет направлен вправо, поэтому, нам необходимо расширить холст вправо, чтобы у нас было достаточно места для размещения эффекта. Итак, идём Изображение-Размер холста (Image> Canvas Size), Расположите Точку привязки (Anchor) слева. Укажите значение Ширины (Width): 4500 рх.

2. Загружаем Кисти

Шаг 1

Идём Редактирование - Наборы - Управление наборами (Edit > Presets > Preset Manager). В появившемся окне Управление наборами (Preset Manager) в выпадающем меню Тип набора (Preset Type), выберите опцию Кисти (Brushes), а затем нажмите кнопку Загрузить (Load).

В появившемся окне навигации, выберите файл Кисти Пыль / Угольки.atn (Ashes n Embers Brushes.atn), которые можно скачать через приложение к уроку. Далее, нажмите кнопку Готово (Done).

Шаг 2

Нажмите клавишу "B", чтобы активировать инструмент Кисть (Brush Tool). Убедитесь, чтобы оба значения Непрозрачность (Opacity) и Нажим (Flow) кисти были установлены на 100%.

3. Создаём Область для Эффекта

Шаг 1

Идём Слой- Новый - Слой (Layer > New > Layer), чтобы добавить новый слой поверх слоя Задний фон (Background). Назовите новый слой Объект (subject).

Теперь создайте выделение вокруг вашего основного объекта. Это можно выполнить различными способами. Например, вы можете создать выделение с помощью инструмента Перо (Pen Tool (P), Волшебной палочки (Magic Wand Tool (W), Лассо (Lasso Tool (L) или с помощью других инструментов, а затем идём Редактирование - Выполнить заливку (Edit > Fill), чтобы залить выделенную область любым цветовым оттенком. Или вы можете выбрать инструмент Кисть (Brush Tool (B) и с помощью круглой жёсткой кисти, прокрасьте выделенную область.

Шаг 2

Ещё раз идём Слой- Новый - Слой (Layer > New > Layer), чтобы добавить новый слой поверх слоя Объект (subject). Назовите этот слой Область (area).

Теперь с помощью круглой жёсткой кисти, цвет кисти любой, прокрасьте область, где бы вы хотели применить эффект.

4. Приступаем к Записи Экшена

Шаг 1

Теперь давайте приступим к подготовке основы для нашего экшена. Мы поместим его в отдельный набор для облегчения работы.

Идём Окно - Операции (Window > Actions) или нажмите клавишу "F9", чтобы открыть панель Операции (Actions). Далее, нажмите значок Создать новый набор (Create new set), назовите этот набор Пепел и Угольки (Ashes n Embers), А затем нажмите кнопку "OK". Мы поместим этот экшен внутри этого набора.

Шаг 2

Нажмите значок Создать новую операцию (Create new action). Назовите эту операцию Экшен Пыль / Угольки (Ashes n Embers Action).

Нажмите значок Запись (Record), чтобы приступить к записи всех последующих шагов в программе Photoshop.

С этого момента, всё, что вы сделаете, будет записано, поэтому, присматривайте за панелью Операции (Actions). Если вы совершили ошибку, нажмите значок Остановить Запись (Stop Playing/Recording), а затем удалите ненужный шаг из операции Пыль / Угольки (Ashes n Embers. Далее, нажмите значок Начать Запись (Begin Recording), чтобы продолжить запись экшена.

5. Создаём Задний Фон

Шаг 1

Перейдите на слой Задний фон (Background layer), а затем отключите видимость слоёв Объект (subject) и Область (area), для этого щёлкните по глазку рядом с миниатюрой слоя.

Далее, нажмите клавишу "D", чтобы выставить цвета по умолчанию.

Шаг 2

Идём Слой- Создать дубликат слоя (Layer > Duplicate Layer). Назовите дубликат слоя Основа (Base), а затем нажмите кнопку "OK". Нажмите клавиши (Ctrl+G), чтобы поместить этот слой в группу. Далее, идём Слой - Переименовать группу (Layer > Rename Group). Назовите эту группу Пепел и Угольки (Ashes n Embers). Мы поместим все наши дальнейшие слои в эту группу, чтобы всё было организовано в палитре слоёв.

Шаг 3

Перейдите на слой Основа (Base), а затем идём Слой- Новый слой-заливка - Цвет (Layer > New Fill Layer > Solid Color). Назовите этот слой Основной цвет BG (BG Base Color), укажите цветовой код #bebebe, нажмите кнопку "OK".

Шаг 4

Перейдите на слой Объект (subject), щёлкните правой кнопкой мыши по миниатюре слоя и в появившемся окне. выберите опцию Выделить пикселы (Select Pixels). Далее, перейдите на слой Основа (Base), а затем нажмите клавиши (Ctrl+J), чтобы скопировать выделение на новый слой.

Идём Слой - Переименовать слой (Layer > Rename Layer). Назовите этот слой Основной объект (Base Subject), переместите его наверх, расположив поверх слоя Основной цвет BG (BG Base Color).

Шаг 5

Теперь перейдите на слой Область (area), щёлкните правой кнопкой мыши по миниатюре слоя и в появившемся окне. выберите опцию Выделить пикселы (Select Pixels). Далее, перейдите на слой Основной Объект (Base Subject ), а затем нажмите клавиши (Ctrl+J), чтобы скопировать выделение на новый слой.

Идём Слой - Переименовать слой (Layer > Rename Layer). Назовите этот слой Исчезающий след (temp_trail).

6. Создаём Слои с Направлением Рассеивания

Шаг 1

Находясь на слое Исчезающий след (temp_trail), идём Фильтр - Размытие- Размытие в движении (Filter > Blur >Motion Blur). Установите Угол (Angle) на 0°, а Смещение (Distance) на 20 рх.

Шаг 2

Нажмите клавиши (Ctrl+J), чтобы продублировать слой.

Находясь на дубликате слоя Исчезающий след (temp_trail copy), нажмите клавиши (Ctrl+T) для активации Свободной трансформации (Free Transform). Сохраните Положение центра преобразования (Reference point location) слева, а также введите значение Ширины (Width): 105%. Нажмите клавишу (Enter), чтобы подтвердить изменения.

Далее, идём Фильтр - Размытие- Размытие в движении (Filter > Blur >Motion Blur). Установите Угол (Angle) на 0°, а Смещение (Distance) на 50 рх.

Шаг 3

Нажмите клавиши (Ctrl+J), чтобы продублировать слой.

Находясь на дубликате 2 слоя Исчезающий след (temp_trail copy 2), нажмите клавиши (Ctrl+T) для активации Свободной трансформации (Free Transform). Сохраните Положение центра преобразования (Reference point location) слева, а также введите значение Ширины (Width): 110%. Нажмите клавишу (Enter), чтобы подтвердить изменения.

Далее, идём Фильтр - Размытие- Размытие в движении (Filter > Blur >Motion Blur). Установите Угол (Angle) на 0°, а Смещение (Distance) на 130 рх.

Шаг 4

Нажмите клавиши (Ctrl+J), чтобы продублировать слой.

Находясь на дубликате 3 слоя Исчезающий след (temp_trail copy 3), нажмите клавиши (Ctrl+T) для активации Свободной трансформации (Free Transform). Сохраните Положение центра преобразования (Reference point location) слева, а также введите значение Ширины (Width): 120%. Нажмите клавишу (Enter), чтобы подтвердить изменения.

Далее, идём Фильтр - Размытие- Размытие в движении (Filter > Blur >Motion Blur). Установите Угол (Angle) на 0°, а Смещение (Distance) на 220 рх.

Шаг 5

Нажмите клавиши (Ctrl+J), чтобы продублировать слой.

Находясь на дубликате 4 слоя Исчезающий след (temp_trail copy 4), нажмите клавиши (Ctrl+T) для активации Свободной трансформации (Free Transform). Сохраните Положение центра преобразования (Reference point location) слева, а также введите значение Ширины (Width): 120%. Нажмите клавишу (Enter), чтобы подтвердить изменения.

Далее, идём Фильтр - Размытие- Размытие в движении (Filter > Blur >Motion Blur). Установите Угол (Angle) на 0°, а Смещение (Distance) на 280 рх.

Шаг 6

Нажмите клавиши (Ctrl+J), чтобы продублировать слой.

Находясь на дубликате 5 слоя Исчезающий след (temp_trail copy 5), нажмите клавиши (Ctrl+T) для активации Свободной трансформации (Free Transform). Сохраните Положение центра преобразования (Reference point location) слева, а также введите значение Ширины (Width): 110%. Нажмите клавишу (Enter), чтобы подтвердить изменения.

Далее, идём Фильтр - Размытие- Размытие в движении (Filter > Blur >Motion Blur). Установите Угол (Angle) на 0°, а Смещение (Distance) на 280 рх.

Шаг 7

Выделите слой Исчезающий след (temp_trail), а затем удерживая клавишу (Shift), выделите дубликат 5 слоя Исчезающий след (temp_trail copy 5). Нажмите клавиши (Ctrl+G), чтобы сгруппировать все выделенные слои в одну группу.

Идём Слой- Переименовать группу (Layer > Rename Group). Назовите эту группу Направление следа рассеивания (dispersion_trail_guide)

На данный момент, результат должен быть, как на скриншоте ниже.

Шаг 8

Нажмите клавиши (Ctrl+J), чтобы продублировать группу. Идём Слой- Переименовать группу (Layer > Rename Group). Назовите дубликат группы Направление рассеивания фрагментов (dispersion_patch_guide). Нажмите клавиши (Ctrl+E), чтобы выполнить сведение группы.

Идём Слой- Слой-маска - Показать всё (Layer > Layer Mask > Reveal All), чтобы добавить слой-маску к слою Направление рассеивания фрагментов (dispersion_patch_guide).

Нажмите клавишу "D", чтобы выставить цвета по умолчанию, а затем идём Фильтр- Рендеринг - Облака (Filter > Render > Clouds).

Далее. идём Изображение - Коррекция - Порог (Image > Adjustments > Threshold). Установите Уровень порога (Threshold Level): 128.

Идём Слой- Слой-маска- Применить (Layer > Layer Mask > Apply).

Отключите видимость слоя с группой Направление следа рассеивания (dispersion_trail_guide), для этого, просто щёлкните по глазку рядом с миниатюрой слоя.

Теперь, результат должен быть, как на скриншоте ниже.

Отключите видимость слоя Направление фрагментов рассеивания (dispersion_patch_guide), для этого, просто щёлкните по глазку рядом с миниатюрой слоя.

7. Создаём Пепел

Шаг 1

Перейдите на слой Область (area), щёлкните правой кнопкой мыши оп миниатюре слоя и в появившемся окне, выберите опцию Выделить пикселы (Select Pixels).

Далее, перейдите на слой Основной Объект (Base Subject ), а затем нажмите клавиши (Ctrl+J), чтобы скопировать выделение на новый слой.Идём Слой - Переименовать слой (Layer > Rename Layer). Назовите этот слой Исчезающий частицы (temp_particles).

Идём Фильтр - Размытие- Размытие по Гауссу (Filter > Blur > Gaussian Blur). Установите Радиус (Radius) размытия на 3 рх.

Шаг 2

Щёлкните правой кнопкой мыши по слою Исчезающий частицы (temp_particles) и в появившемся меню, выберите опцию Параметры наложения (Blending Options).

Выберите стиль слоя Наложение цвета (Color Overlay), установите цвет наложения чёрный #000000, режим наложения (Blend Mode): Нормальный (Normal), Непрозрачность (Opacity): 90%. Нажмите кнопку "OK".

Шаг 3

Щёлкните правой кнопкой мыши по миниатюре слоя Исчезающий частицы (temp_particles) и в появившемся меню, выберите опцию Выделить пикселы (Select Pixels).

Нажмите клавишу "M", чтобы активировать инструмент Прямоугольная область (Rectangular Marquee Tool). Далее, щёлкните правой кнопкой мыши по изображению и в появившемся окне, выберите опцию Образовать рабочий контур (Make Work Path). В появившемся окне, укажите Допуск (Tolerance): 1.0 pх, а затем нажмите кнопку "OK".

Шаг 4

Идём Слой- Слой-маска - Скрыть всё (Layer > Layer Mask > Hide All), чтобы добавить чёрную слой-маску к слою Исчезающие частицы (temp_particles) и спрятать его содержимое.

Нажмите клавишу "B", чтобы активировать инструмент Кисть (Brush Tool). В окне настроек кисти, выберите кисть br-ashes-particles-1. Убедитесь, чтобы оба значения Непрозрачности (Opacity) и Нажим (Flow) кисти были установлены на 100%. Примечание переводчика: кисть br-ashes-particles-1, вы можете скачать в приложении к уроку.

Шаг 5

Нажмите клавишу "P", чтобы выбрать инструмент Перо (Pen Tool). Убедитесь, чтобы слой-маска слоя Исчезающие частицы (temp_particles) была активна.

Далее, щёлкните правой кнопкой мыши по холсту и в появившемся меню, выберите опцию Выполнить обводку (Stroke Path) и в появившемся окне, не забудьте убрать галочку в окошке Имитировать нажим (Simulate Pressure). Нажмите кнопку "OK".

Выбрав инструмент Перо (Pen Tool), ещё раз щёлкните правой кнопкой мыши по холсту и в появившемся меню, выберите опцию Удалить контур (Delete Path).

Шаг 6

Идём Изображение - Коррекция - Порог (Image > Adjustments > Threshold). Установите Уровень порога (Threshold Level): 50. Нажмите кнопку "ОК".

Далее, идём Слой- Слой-маска- Применить (Layer > Layer Mask > Apply).

Шаг 7

Находясь на слое Исчезающие частицы (temp_particles), нажмите клавиши (Ctrl+T) для активации Свободной трансформации (Free Transform). Сохраните Положение центра преобразования (Reference point location) слева, а также введите значение Ширины (Width): 105%. Нажмите клавишу (Enter), чтобы подтвердить изменения.

Шаг 8

Дважды нажмите клавиши (Ctrl+J), чтобы продублировать слой Исчезающие частицы (temp_particles) и создать ещё два дубликата слоя.

Удерживая клавишу (Shift), выделите слой Исчезающие частицы (temp_particles). Нажмите клавиши (Ctrl+E), чтобы объединить все слои. Примечание переводчика: автор объединил все три слоя, оригинал и два дубликата.

Шаг 9

Нажмите клавиши (Ctrl+J), чтобы продублировать дубликат 2 слоя Исчезающие частицы (temp_particles copy 2). Примечание переводчика: автор продублировал объединённый слой.

Находясь на дубликате 3 слоя Исчезающие частицы (temp_particles copy 3), нажмите клавиши (Ctrl+T) для активации Свободной трансформации (Free Transform). Сохраните Положение центра преобразования (Reference point location) слева, а также введите значение Ширины (Width): 110%. Нажмите клавишу (Enter), чтобы подтвердить изменения.

Идём Фильтр- Искажение - Волна (Filter > Distort > Wave). Примените следующие настройки. которые указаны ниже:

  • Число генераторов (Number of Generators): 5
  • Длина волны (Wavelength): Мин (Min): 10 Макс (Max): 750
  • Амплитуда (Amplitude): Мин (Min):1 Макс (Max):15
