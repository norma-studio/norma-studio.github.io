Преимущества для размещения вашей собственной аналитики

Я был давним пользователем freemium, поставщика аналитики облачной статистики Statcounter. Мне нравится видеть конкретную активность пользователя и трафик IP-адресов. Хотя Google Analytics не предоставляет эту информацию, Statcounter не имеет многих функций, которые предоставляет Google Analytics бесплатно.

Piwik - это зрелый, хорошо документированный, полнофункциональный продукт с открытым исходным кодом, написанный на PHP; он даже предлагает бесплатные мобильные приложения для iOS и Android.

У команды есть планы по предоставлению инструмента для переноса данных Google Analytics в Piwik в 2015 году.

Установка Piwik на вашем собственном сервере

Piwik работает на любой современной конфигурации сервера LAMP. Если вы хотите, вы можете начать с моей общей установки Ubuntu LAMP здесь.

Затем следуйте инструкциям по установке ниже и / или просмотрите установочный видеоролик:

Во-первых, давайте загрузим код Piwik и создадим веб-каталог для вашей информационной панели:

mkdir /var/www/piwikcd /var/www/piwiksudo wget http://builds.piwik.org/piwik.zipsudo unzip piwik.zip sudo rm piwik.zipsudo rm How\ to\ install\ Piwik.html

Измените права и разрешения, соответствующие вашему веб-серверу:


chown -R www-data:www-data /var/www/piwiksudo mkdir /var/www/piwik/tmpsudo mkdir /var/www/piwik/tmp/assets/sudo mkdir /var/www/piwik/tmp/cache/sudo mkdir /var/www/piwik/tmp/logs/sudo mkdir /var/www/piwik/tmp/tcpdf/sudo mkdir /var/www/piwik/tmp/templates_c/sudo chmod  -R 0755 /var/www/piwik/tmpsudo chmod  -R 0755 /var/www/piwik/tmp/assets/sudo chmod  -R 0755 /var/www/piwik/tmp/cache/sudo chmod  -R 0755 /var/www/piwik/tmp/logs/sudo chmod  -R 0755 /var/www/piwik/tmp/tcpdf/sudo chmod  -R 0755 /var/www/piwik/tmp/templates_c/

Создайте файл конфигурации сайта Apache и вставьте в него определение виртуального хоста. Затем включите сайт.

cd /etc/apache2/sites-available/sudo nano stats.confsudo a2ensite stats.conf

Вставьте и настройте определение виртуального хоста для вашего сайта Piwik, как показано ниже:

<VirtualHost *:80>   ServerName piwik.yourdomain.com   DocumentRoot /var/www/piwik    DirectoryIndex index.php   <Directory /var/www/piwik/>      AllowOverride All      Order Deny,Allow      Allow from all   </Directory></VirtualHost>
