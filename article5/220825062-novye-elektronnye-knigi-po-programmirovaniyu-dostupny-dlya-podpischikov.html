Чему Вы Научитесь

За последнюю пару месяцев мы создали 16 новых электронных книг, доступных для загрузки подписчикам Envato Tuts+. Вот перечень данных книг и краткое описание, что вы можете узнать, после их прочтения.

1. TypeScript Дизайн Паттерны

В программировании часто возникают различные трудности. Для разрешения подобных проблем, существует набор типичных решений, также известных, как дизайн паттерны. Дизайн паттерны отличный способ улучшить эффективность ваших программ и стать более продуктивным.

Эта книга коллекция самых важных паттернов по улучшению производительности приложений и вашей продуктивности. Книга начинается с объяснения текущих трудностей при дизайне и разработке приложений, и как решить эти трудности применяя дизайн паттерны и лучшие практики.

2. Swift: Разработка iOS Приложений

The Swift: Разработка iOS приложений электронная книга сделает из вас эффективного iOS и macOS разработчика, познакомит вас с последними трендами. После основ вы узнаете более продвинутые темы, эта книга описывает подробно все детали.

Книга состоит из четырёх модулей. Каждый из модулей является мини-книгой сам по себе, и как только вы закончите каждый из них, вы получите необходимые навыки для перехода к материалу следующего модуля.

3. Python: Путешествие От Новичка к Эксперту

Python динамичный и мощный язык, нашедший применение в различных сферах программирования. Он обладает легким в использовании, простым синтаксисом, а также мощными библиотеками, которые включают сотни модулей для решения различных задач, благодаря этому он так популярен среди программистов.

Эта электронная книга научит вас необходимым основам, затем покажет вам замечательные инструменты и техники, которые помогут вам улучшить навык программирования.

4. Изучаем GraphQL и Relay

На сегоднящний день появился новый способ реализовать API - это спецификация с открытым исходным кодом, созданная Facebook под названием GraphQL. Она Разработана для решения множества проблем связанных с работой с REST, GraphQL идёт вместе с RelayJS, React библиотекой для выполнения запросов к серверу, реализованному по спецификации GraphQL. С этой книгой вы быстро поймёте навыки, необходимые для разработки готовых приложений с GraphQL и RelayJS.

5. JavaScript: Функциональное Программирование для JavaScript Разработчиков

Функциональное программирование способ написания более чистого кода, благодаря мутации, объединению и использованию функций. JavaScript предоставляет всё необходимое для реализации данного подхода. Открыв для себя функциональную сущность JavaScript, вы будете в состоянии реализовать более мощные и надёжные веб-приложения, которые гораздо легче поддерживать.

Эта книга даст вам понимание, как функциональный подход вместе с другими популярными техниками, может сделать JavaScript более эффективным.

6. Ionic 2 Blueprints

Ionic 2, последняя версия Ionic Mobile SDK, построен на основе последних технологий таких как Angular 2, TypeScript, SASS, и так далее. Идея позади Ionic 2 - сделать процесс разработки ещё более увлекательным.

Эта книга даст вам возможность создать интересные и увлекательные приложения с Ionic2. Вы научитесь использовать различные компоненты Ionic, добавлять внешние сервисы, а также как делать профессиональные приложения с Ionic 2.

К концу этой книги, вы сможете с гордостью назвать себя профессиональным Ionic разработчиком, который в состоянии создавать и поддерживать различные Ionic приложения, и у вас будет более глубокое понимание использования Ionic на практике.

7. Go: Создание Веб-приложений

Go язык с открытым исходным кодом, который позволяет с легкостью создать простые, надёжные и эффективные программы. Это строго-типизированный язык с синтаксисом основанным на C, добавьте сюда сборку мусора, динамические возможности типов, встроенные типы, такие как variable-length arrays и key-value maps, и большой выбор встроенных библиотек.

Книга начинается, как путеводитель необходимый для создания нового веб-приложения. Будь-то безопасность веб-приложений, соединение с базой данных, добавления аутентификации основанной на токенах, или использование шаблонов, с этой книгой вы будете надёжно осведомлены, касательно данных тем.

8. Django: Веб-разработка с Python

Наука о данных очень популярная на сегодняшний день, и нуждается в разработчиках с разным набором талантов как никогда раньше. Основы построения приложений с минималистичным, мощным и легким в изучении фреймворком Django, будет отличным навыком для начала карьеры предпринимателя, или веб-разработчика.

Django - веб-фреймворк созданный с мыслями о балансе между быстро-создаваемыми и производительными веб-приложениями. С этой книгой вы станете эффективным веб-разработчиком и поймёте ключевые понятия Django фреймворка. К концу каждого из четырёх модулей, вы будете в состоянии применить Django фреймворк для разработки полнофункциональных веб-приложений, с минимальной затратой усилий.
