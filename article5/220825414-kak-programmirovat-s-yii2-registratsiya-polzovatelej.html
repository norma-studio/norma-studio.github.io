Регистрация пользователя для вашего веб-приложения

Практически любое веб-приложение требует определенной регистрации пользователя. Yii предоставляет несколько способов регистрации пользователей. Это одна из вещей, которые мне нравятся в использовании Yii над ванильным PHP, у меня может быть полнофункциональный фреймворк для веб-приложений, готовый строить классные вещи.

Существует несколько причин для изобретения колеса и написания для аутентификации пользователей, а также многих других требований и функций с нуля, например, отправка электронной почты и аутентификация для проверки регистрации, восстановления пароля, сторонней социальной аутентификации и т. д.

Шаблон расширенного приложения Yii2 обеспечивает встроенную регистрацию пользователя; мы используем этот подход в моей серии Building Your Startup. Однако в этой серии мы основываемся на базовом шаблоне приложения Yii2. Шаблон базового приложения по умолчанию включает жестко закодированный вход пользователя, а что не очень полезно.

Другой подход к регистрации пользователей - использование сторонних расширений. В этом уроке я расскажу вам об использовании расширения Yii2-User Дмитрия Ерофеева. Документация для Yii2-User доступна здесь. Erofeev пишет также и другие плагины для Yii2.

Установка пользователя Yii2

Давайте начнем установку расширения Yii2-User. Мы будем следовать инструкциям по установке.

Установка Yii2-User с Composer

Во-первых, нам нужно добавить Yii2-User в требуемые расширения Composer. Отредактируйте файл /composer.json, чтобы включить в него Yii2-User:

    "require": {        "php": ">=5.4.0",        "yiisoft/yii2": "*",        "yiisoft/yii2-bootstrap": "*",        "yiisoft/yii2-swiftmailer": "*",        "dektrium/yii2-user": "*"    },

Затем, когда мы обновляем Composer, вы увидите что-то вроде этого:

Обновление файла конфигурации

Затем нам нужно указать Yii использовать компонент Yii2-User. В /config/web.php мы заменим компонент User по умолчанию ...

        "user" => [            "identityClass" => "app\models\User",            "enableAutoLogin" => true,        ],

... компонентом Yii2-User:

        "user" => [            "class" => "dektrium\user\Module",            "enableUnconfirmedLogin" => true,            "confirmWithin" => 21600,            "cost" => 12,            "admins" => ["admin"]        ],

Активация SwiftMailer

Поскольку Yii2-User использует электронную почту для отправки регистрационных подтверждений и забытых паролей, пришло время активировать нашу конфигурацию SwiftMailer. В config/web.php замените конфигурацию Mailer по умолчанию здесь ...

        "mailer" => [            "class" => "yii\swiftmailer\Mailer",            // send all mails to a file by default. You have to set            // "useFileTransport" to false and configure a transport            // for the mailer to send real emails.            "useFileTransport" => true,        ],

... этим - вам нужно будет включить собственные учетные данные SMTP:

        "mailer" => [                "class" => "yii\swiftmailer\Mailer",                "viewPath" => "@app/mailer",                "useFileTransport" => false,                "transport" => [                    "class" => "Swift_SmtpTransport",                    "host" => "your-host-domain e.g. smtp.gmail.com",                    "username" => "your-email-or-username",                    "password" => "your-password",                    "port" => "587",                    "encryption" => "tls",                                ],            ],
